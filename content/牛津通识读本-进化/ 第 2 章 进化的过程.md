# _第_2_章_进化的过程 Summary

## 生物学的两大核心问题

**"怎么样(how)"问题**
- 分子过程层面运作机制
- 最简单生物(细菌): 成千上万种蛋白质 → 精密复杂机器
- 高等生物发育: 单细胞 → 精密调控分裂 → 分化 → 有序结构
- P: 生物最复杂特性(意识) = 化学+物理过程 (可被科学分析)

**"为什么(why)"问题**
- 进化前: 归因于造物主
- "适应"概念: 18世纪神学家 → 设计表象 ⟹ 超自然设计师
- 休谟: 证明神创论逻辑缺陷 (但缺替代理论)
- 进化论: 自然过程解释 → 多样性+适应性 (¬需超自然力量)

## 突变：进化的原材料

**突变特征**
- Notion: 突变 := 遗传物质稳定变化 → 亲代传递子代
- 频率: ~1/100,000 每代
- 影响范围:
  - 无表型影响 (仅分子层面)
  - 微小改变 (ex: 眼色, 抗药性)
  - 显著改变 (ex: 果蝇头部长腿)

**突变性质**
- 大部分突变: 无影响 ∨ 降低生存/繁殖能力
- 突变随机出现 (¬受自然选择影响)

## 自然选择：进化的主要驱动力

**达尔文-华莱士理论要素**
1. 大量后代 > 存活数量 → 生存竞争
2. 个体变异 → 影响生存/繁殖能力
3. 变异可遗传 → 子代特征≠种群整体

**选择过程**
- 环境改变 → 微小有利变异 → 生存机会↑
- 遗传优势 → 频率上升 → 适应性进化
- P: "天平中最细小颗粒决定生死" × 千代 = 巨大影响

## 遗传漂变：随机进化机制

**选择中性变异**
- Notion: 选择中性 := 对生存/繁殖无影响的变异
- 子代 = 亲代基因随机样本
- ∵ 种群大小有限 ∴ 随机差异必然存在

**遗传漂变特征**
- 即使最大种群(细菌)也有限 → 漂变总是存在
- 与自然选择共同作用 → 种群组成改变

## 物种形成与多样性

**物种定义**
- 核心标准: 杂交繁殖障碍
- 同种: 可杂交 (ex: 人类不同种群)
- 异种: 不可杂交 (ex: 人类 ↔ 黑猩猩)

**物种形成过程**
1. 杂交繁殖障碍进化
2. 种群独立发展
3. 突变+选择+漂变 → 分化
4. ∴ 生物多样性

**灭绝法则**
P: 灭绝 = 几乎所有物种宿命

## 进化机制总结

**三大机制协同**
突变(原材料) + 自然选择(定向) + 遗传漂变(随机) → 进化改变

**时间尺度**
- 短期: 数年 (ex: 抗生素抗性)
- 长期: 数百万年 (ex: 爬行类→哺乳类)
- P: 重大变化 = 微小改变长期积累

**理论优势**
进化论 vs 神创论:
- 进化论: 数学模拟 + 精确理论 + 自圆其说
- 神创论: 逻辑缺陷 + 难以自圆其说
