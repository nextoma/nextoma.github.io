# _第_5_章_适应与自然选择 Summary

## 适应的本质与例证

**经典适应实例**
- 眼睛多样性: 水底眼 ≠ 空气眼 (紫外线+偏振光透视)
- 鸟类中空骨骼: 内部支杆 = 飞机机翼设计
- 动物关节: 表面顺滑移动
- 蝴蝶长口器 ↔ 花朵深蜜腺 (协同进化)
- 伪装色: 银色鱼类(水中) vs 隐蔽色(陆地)

**细胞层面适应**
- 细胞分裂: 蛋白分子微型发动机驱动
- DNA校对: 降低有害突变概率
- 神经信号: 带电离子流 → 电信号传递
- 老化速率: 适应生存环境

**前达尔文解释**
- 18世纪: "创世论" → 造物主设计
- 问题: 适应需要解释 (¬自然产物)
- P: 达尔文理论 → 自然过程解释复杂性

## 人工选择与可遗传变异

**人工选择证据**
- 卷心菜品种: 花椰菜(巨大花朵) + 球芽甘蓝(特殊叶子)
- 狗品种: 共同祖先=狼 (1.4-13.5万年前分离)
  - DNA序列: 狗⊂狼
  - 差异: 外形+性格+智力+疾病+寿命
  - ex: 吉娃娃寿命 = 2×大丹狗寿命

**可遗传变异类型**
1. **离散性状**: 单基因控制 (ex: 眼色, 血型)
2. **连续变异**: 多基因+环境 (ex: 身高, 体重)
3. **多态性**: 常见变异 (ex: ABO血型)
4. **分子多态性**: DNA序列差异
   - 人类: <0.1% DNA差异
   - 玉米: >2% DNA差异

**变异分布**
- 种群间差异 < 种群内差异 (人类)
- P: 基因流动 → 种群同质化

## 自然选择机制

**选择原理**
```
可遗传差异 + 影响生存/繁殖 → 频率改变 → 适应进化
```

**适应性(fitness)概念**
- Notion: 适应性 := 生存+繁殖总能力
- 包含: 速度+警觉+育幼+吸引传粉者
- 环境特异性: 树栖蜥蜴(短腿>长腿)

**选择速率**
- 年尺度: 缓慢 (多代积累)
- 地质尺度: 迅速 (<相邻地层时间)
- ex: 玉米驯化~1万年 → 现代巨型玉米

## 抗性进化实例

**抗生素抗性**
- 1940s首次使用 → 迅速出现抗性
- 机制: 突变 → 抗性个体存活 → 种群建立
- 问题: 抗性可在细菌间传递 (跨物种)
- 对策: 联合用药 (降低双重抗性概率)

**杀鼠灵抗性**
- 机制: 维生素K代谢改变
- 代价: 生长/存活概率↓
- 进化: 新型无副作用抗性出现

**疟疾抗性**
- 镰刀形贫血: AS型(抗疟) vs SS型(致病)
- 葡萄糖-6-磷酸脱氢酶变体: 抗疟+食物敏感
- 达菲阴性血型: 抗疟无明显代价
- P: 同一选择压力 → 多种响应方式

## 适应的限制

**进化无前瞻性**
- 爱尔兰大角鹿: 巨角 → 灭绝
- 北美旅鸽: 繁殖慢 → 捕猎灭绝
- 熊猫: 特化食性 → 濒危

**不完美适应**
1. **军备竞赛**: 宿主↔寄生虫 (无绝对优势)
2. **历史限制**: 脊椎动物视网膜(神经在前)
3. **权衡代价**: 一方面提升 → 另一方面减弱
4. **修补性质**: 翅膀=改良前肢 (失去原功能)

**趋同进化**
- 相似环境 → 相似适应
- ex: 江豚简化眼睛 (多地独立进化)
- DNA序列: 揭示真实亲缘关系

## 选择的实证研究

**达尔文雀研究(30年)**
- 鸟喙大小: 可遗传变异
- 干旱 → 大种子 → 大喙优势
- 环境恢复 → 选择逆转
- 结果: 30年持续波动

**花-传粉者协同进化**
- 兰花蜜腺管 ↔ 天蛾口器
- 最优长度: 恰好够不到 (强迫携带花粉)
- 极端例: 马达加斯加(30cm vs 25cm)

## 稳定化选择

**净化选择作用**
- 对抗有害突变 → 维持最适状态
- ex: 血友病基因 (低频维持)

**最适值选择**
- 人类婴儿: 中等体重存活率最高
- 暴风后: 中等大小鸟类/昆虫存活
- 结果: 精密拟态 + 活化石现象

## 性选择

**竞争机制**
- 雄性竞争: 斗争+求偶 → 交配权
- 雌性选择: 挑剔 → 繁殖差异

**性选择特征**
- 雄孔雀尾: 飞行能力↓ 交配成功↑
- 燕子长尾: 超最适飞行长度
- 热带蛙鸣叫: 吸引配偶+暴露位置

**适应悖论**
P: 生物学"适应" ≠ 日常"适应"
- 雄孔雀: 飞行不适应, 繁殖高适应
- 性选择 → 生存代价换取繁殖优势

## 核心洞察

**选择的双重性**
- 改变: 环境变化 → 新适应
- 维持: 稳定环境 → 保持现状

**进化工程师比喻**
P: 自然选择 = 修补而非重新设计
∴ 生物 = 历史妥协的产物

**人类影响普遍性**
∵ 人类遍布全球
∴ 极少物种不受人类选择压力影响
