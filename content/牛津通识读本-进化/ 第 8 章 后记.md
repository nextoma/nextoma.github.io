## 人类进化的意义

**遗传与环境**
- 基因 = 可能范围设定
- 环境 = 实际表达决定
- 学习受限: 大脑结构 ← 遗传构成
- ex: 狗不能说话, 人不能嗅兔子

**种族问题的科学观**
- P: 群体内差异 >> 群体间差异
- ∴ 种族同质性/优越性 = 无稽之谈
- 科学作用: 提供信息 ≠ 直接决策

**人类特性的进化基础**
- 语言/思维/情感 ← 千万年选择
- 社会性行为: 非完全自私
- 公平感 = 进化遗产
- 但: ¬所有行为遗传决定

## 进化中的进步悖论

**有条件的进步**
- 简单→复杂: 普遍趋势
- 但: ¬必然性
  - 细菌: 依然最成功
  - 退化例子: 穴居动物失明
  - 寄生虫: 结构简化

**复杂性的双向性**
- 进化: 功能改善 → 复杂性↑
- 退化: 功能丧失 → 复杂性↓
- P: 选择无前瞻性

## 进化的冷酷与美丽

**冷酷面**
- 捕食者技巧 ∧ ¬顾猎物感受
- 寄生虫精妙 ∧ 宿主痛苦
- 衰老必然性
- 低生育率 → 灭绝风险

**美丽面**
- 30亿年 → 惊人物种多样性
- "自然之战" → 生命奇迹
- 人类独特: 推理能力

## 人类的责任

**进化赋予的能力**
- 独特推理能力 ← 进化产物
- 运用远见 → 减轻"自然之战"

**道德使命**
- 敬畏进化成果
- 保护生物多样性
- 防止贪婪/愚蠢毁灭
- 为后代留存

**最终警告**
P: 不保护 → 人类也将灭绝

## 核心洞察总结

**科学与社会**
- 科学 = 理解工具
- 进化论 = 位置认知
- 人类 = 自然一部分

- 无目的性: 只积累当前有利

- 持续性: 永不停止

**人类的特殊性**
- 产物: 冷酷进化力量
