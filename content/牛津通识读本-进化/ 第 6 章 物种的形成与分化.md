# _第_6_章_物种的形成与分化 Summary

## 物种概念与生殖隔离

**物种划分标准**
- P: 生殖隔离 = 最佳物种界定标准
- 同地物种: 保持区别 (¬杂交)
- 假想: 无隔离 → 连续形态谱系

**生殖隔离机制**
1. **生态隔离**: 生境差异, 繁殖时间不同
2. **行为隔离**: 求偶信号不匹配 (气味/声音/舞蹈)
3. **生理隔离**: 精子无法受精, 花粉被拒绝
4. **杂种不活**: 杂交后代早期死亡
5. **杂种不育**: 存活但无法繁殖 (ex: 骡子)

## 地理变异与种群分化

**地理变异模式**
- Notion: 地理变异 := 种群因地理位置产生的适应性差异
- ex: 人类种族差异 (肤色/身高)
- 北方哺乳动物: 体型↑ 耳朵↓ (热量保存)

**分化机制**
```
地理隔离 → 独立进化 (突变+选择+漂变) → 基因分化
```
- 迁移 = 均质化力量 (对抗分化)
- 强选择 > 迁移 → 局部适应 (ex: 矿山金属耐受植物)

**血型地理分布**
- B型血: 印度>30% vs 美洲原住民罕见
- O型血: 威尔士/苏格兰 > 英格兰南部

## 物种形成过程

**过渡阶段实例: 美洲拟暗果蝇**
- 波哥大种群: ~20万年前分离
- DNA序列: 轻微差异
- 杂交结果:
  - 雌性杂种: 可育
  - 雄性杂种: 不育 (母本为非波哥大时)
- P: 开始形成隔离但未完全成种

**猴面花案例**
- 彩艳龙头: 蜜蜂授粉 (黄色花)
- 红龙头: 蜂鸟授粉 (红色花)
- 杂种健康可育 BUT 野外无杂交
- 原因: 传粉者行为隔离

## 生殖隔离的进化机制

**副产物假说**
P: 隔离 = 独立进化的副产物 (¬直接选择)
- 类比: 英国插头 ≠ 欧洲插座
- 不同基因系列 → 杂种机能失调
- ~15个基因参与雄性杂种不育 (拟暗果蝇)

**进化速率差异**
- 拟暗果蝇: 20万年 → 部分隔离
- 维多利亚湖慈鲷: 1.46万年 → 500+种
  - 平均1000年/新种
  - 主要差异: 行为+颜色 (DNA差异小)
- 一般物种: 几万年/新种

## 形态与隔离的关系

**形态≠隔离程度**
- 拟果蝇vs毛里求斯果蝇: 形态相似但完全隔离
- 欧洲伏翼蝙蝠: 外观相同但两个物种
- 反例: 同种但形态差异大的种群可杂交

**进化速率不规则**
- 环境稳定 → 形态不变 (活化石)
- 环境剧变 → 快速改变
- 人工选择: 80代 → 刚毛数3倍 (果蝇)

**无性生物物种定义**
- 细菌: 无杂交概念
- 分类标准: 相似度 (特征+DNA序列)

## 分子进化与分子钟

**DNA序列差异分析**
- 置换改变: 改变氨基酸 (少见, 常有害)
- 沉默改变: 不改变氨基酸 (常见, 中性)

**人类与其他物种比较**
```
黑猩猩: 1.24% 差异
大猩猩: 1.62% 差异
DNA字母突变率: ~1次/10亿年
```

**分子钟原理**
- P: 中性突变累积 ∝ 时间
- 速率 = 突变率 (独立于种群大小)
- 应用: 估算分化时间
  - 人-黑猩猩: 600-700万年前

## 遗传漂变机制

**过程描述**
1. 中性突变出现 (1/种群)
2. 随机频率波动
3. 最终: 固定(100%) ∨ 消失(0%)

**漂变速率**
- 小种群: 快速漂变
- 大种群: 缓慢漂变
- 但分化速率 = 突变率 (种群大小效应抵消)

**两类影响**
1. 维持种群多样性
2. 推动种群分化 (即使无选择)

## 生命之树重建

**分子系统发育**
- 缓慢进化基因 → 古老分化
- 快速进化基因 → 近期分化
- 多基因累积 → 提高准确性

**进化树时间轴**
```
10亿年前: 主要生物群分化
真细菌 ↔ 古细菌 ↔ 真核生物
动物 ↔ 植物 ↔ 真菌
```

## 核心洞察

**物种形成要素**
1. 地理/生态隔离 → 独立进化
2. 突变+选择+漂变 → 基因分化
3. 副产物: 生殖不兼容性
4. 完全隔离 → 新物种

**进化的不规则性**
- 形态变化 ≠ 隔离程度
- 进化速率 = f(环境变化)
- 分子钟 > 形态钟 (规律性)

**统一性证据**
P: 生命之树可重建
∴ 所有生物共同起源
