# _第_7_章_一些难题 Summary

## 复杂适应的进化

**渐进修改原理**
- P: 复杂结构 = 连续小改进积累
- 类比: 工程师造机器 → 不断修改完善
- 爬山比喻: 大雾中每步向上 → 最终登顶

**蛋白质进化非随机**
- 错误类比: 龙卷风组装飞机
- 实际过程:
  1. 短链氨基酸 → 简单功能
  2. 连续改进 → 复杂蛋白
  3. 活性位点 = 小部分氨基酸
  4. 其他部分 = 支架结构
- P: 小改变 → 大功能变化

**代谢途径逆向进化**
```
最终产物 ← 前体1 ← 前体2 ← ...
```
- 环境物质稀缺 → 合成酶进化
- 逐步延长路径 → 完整代谢链

## 中间形态证据

**化石证据**
- 寒武纪大爆发: 缺少祖先化石
- 软体生物 → 不易成化石
- 新发现: 1.25亿年前哺乳动物 (早4000万年)

**现存过渡形态**
飞行进化序列:
```
普通哺乳 → 鼯鼠(滑翔) → 飞狐猴(大膜) → 蝙蝠(真飞行)
```
- 滑翔适应: 独立进化多次
- 蜥蜴/蛇/青蛙也有滑翔种

**眼睛进化序列**
1. 光敏蛋白(视紫红质) → 感光
2. 简单眼点 → 明暗感知
3. 眼杯 → 方向性
4. 晶状体 → 聚焦成像
- P: 无晶状体视网膜 ≠ 无用
- 所有动物: 共用视紫红质

## 衰老的进化悖论

**衰老普遍性**
- 成年后死亡率↑ = 衰老
- 小动物(老鼠): 衰老快
- 大动物(人类): 衰老慢
- 单细胞: 不衰老 (持续修复)

**衰老进化理论**
1. **外因死亡影响**
   - 事故/疾病/捕食 → 老年个体少
   - ∴ 选择偏重早期性状

2. **两种机制**
   - 有害突变积累: 晚期突变清除弱
   - 拮抗多效性: 早期利益 > 晚期代价
   - ex: 高激素 → 生育↑ 但癌症风险↑

**寿命与生态**
- 飞行动物: 长寿 (躲避捕食)
- ex: 鹦鹉寿命 > 人类
- 蝙蝠寿命 >> 同体重老鼠

**人类衰老进化**
- 黑猩猩: <50年, 11岁生育
- 人类: >80年, 延迟成熟
- 原因: 智力+合作 → 外因死亡↓
- 工业化影响:
  - 亨廷顿氏病: 选择压力↑
  - 预测: 衰老速率↓ (需几世纪)

## 社会性昆虫不育之谜

**问题核心**
Q: 为何进化出不生育个体?
- 工蚁/工蜂: 不育但劳作
- 兵蚁: 特化防御
- 裸鼢鼠: 哺乳动物也有

**亲缘选择理论**
- P: 帮助近亲 = 传递共同基因
- 兄弟姐妹: 50%基因相同
- 霍尔丹: "为2兄弟或8表亲献身"
- 群体利益 > 个体繁殖

**阶级分化机制**
1. 环境信号 → 发育路径
2. 基因控制反应能力
3. 群体成功 → 基因传播

**多细胞进化类比**
- 细胞"牺牲"繁殖 → 整体利益
- 细胞分化 = 昆虫阶级分化
- 癌症 = 细胞背叛整体

## 两大未解之谜

**生命起源**
- 目标: 自我复制分子
- 进展:
  - ✓ 合成基本成分 (氨基酸/糖/脂肪)
  - ✗ 组装复杂分子 (RNA/DNA)
  - ✗ 实现自我复制
  - ✗ 原始基因编码
- P: 进展有限但持续

**意识起源**
- 定义困难: 程度vs本质
- 动物意识争议:
  - 宠物: 情绪反应+操纵行为
  - 鹦鹉/黑猩猩: 基本语言能力
- 生物学基础:
  - 意识 = 神经细胞活动
  - 基因控制大脑发育
  - 语法基因已识别

## 核心洞察

**复杂性来源**
P: 复杂 = 简单×时间×选择
- 每步改进微小
- 累积效应巨大
- 无需预见性

**进化的限制与突破**
- 限制: 历史包袱+修补性质
- 突破: 新功能=旧结构新用途
- 悖论: 退化(衰老)也是适应

**未解之谜的共性**
- 独特事件 → 缺少比较
- 远古历史 → 缺少化石
- 但: 原则上可用进化解释
