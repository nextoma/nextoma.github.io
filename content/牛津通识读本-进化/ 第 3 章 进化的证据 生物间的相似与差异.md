# _第_3_章_进化的证据_生物间的相似与差异 Summary

## 生物相似性的层次

**相似性谱系**
- 外部特征 → 显微结构 → 生化功能 → 遗传物质
- 人类↔细菌: 相似性依然可探测
- P: 共同祖先 ⟹ 进化联系 → 相似性自然解释

**分类系统层级**
```
物种 → 属 → 科 → 目 → 纲 → 门 → 界
```
- 特征共享: 层级↑ → 共有特征↓
- ex: 节肢动物 → 昆虫 → 蝇类 → 家蝇
- Notion: 物种 := 可杂交繁殖的相似个体集合

## 结构特化与同源性

**前肢特化实例**
- 蝙蝠翅膀 ↔ 鸟类翅膀: 特化前肢 (内部骨骼相似)
- 鲸鳍: 外观像鱼鳍, 内部结构=哺乳动物足
- 哺乳动物听小骨 ← 爬行类下颌骨演化

**胚胎发育证据**
- 哺乳动物胚胎: 出现鱼类鳃裂
- P: 鱼类祖先 → 发育结构保留 (引导血管形成)
- 胚胎相似性 > 成体相似性

**痕迹器官**
- 人类阑尾: 猩猩中巨大, 人类中退化
- 蟒蛇后肢: 退化骨盆+股骨 (表达四肢发育基因)
- P: 进化 = 创造+改进+削减

## 细胞层面的统一性

**细胞基本结构**
- 真核生物: 细胞核+细胞质+细胞器
  - 线粒体: 能量产生 (源自细菌)
  - 叶绿体: 光合作用 (源自细菌)
- 原核生物: 无细胞核, 结构简单

**蛋白质功能分类**
1. 酶: 催化反应 (ex: 消化酶)
2. 运输: 血红蛋白运氧
3. 结构: 角蛋白 (皮肤/毛发)
4. 信号: 激素传递信息
5. 受体: 细胞表面交流

**代谢途径普遍性**
- 能量产生: 糖/脂肪 → ATP (所有细胞相同)
- 黑色素合成: 哺乳类=蝴蝶=酵母
- ex: 乳酸堆积 (人类=蜥蜴)

## 遗传物质的统一性

**DNA结构与功能**
- 双螺旋结构: A-T, G-C配对
- 基因 := DNA片段 → 编码蛋白质
- 三联体密码: 3个核苷酸 → 1个氨基酸
- P: 基因编码全球通用 ⟹ 共同祖先

**基因表达过程**
```
DNA → RNA(转录) → 蛋白质(翻译)
```
- 所有细胞本质相同
- 非编码DNA: 调控基因开关

**基因组比较**
- 大肠杆菌: 86% DNA编码蛋白
- 果蝇: 20% DNA编码蛋白
- 人类: <2% DNA编码蛋白
- 人类基因数: ~35,000个

## 有性繁殖的普遍性

**染色体行为**
- 2n(二倍体) → n(单倍体) → 2n循环
- 减数分裂: 染色体配对 → 交换 → 分离
- P: 有性繁殖早于多细胞生物进化

**无性繁殖特例**
- ex: 普通蒲公英
- 保留有性繁殖痕迹 (产生花粉)
- 源自有性繁殖祖先

## 突变的多重效应

**突变类型与后果**
1. 氨基酸改变 → 蛋白功能障碍
2. 基因控制异常 → 产量不足
3. 退化基因: 65%人类嗅觉受体基因

**疾病相关突变**
- 癌症: 细胞分裂控制基因突变
- 囊性纤维症: 北欧最常见遗传病
- P: 有害突变 → 自然选择淘汰

**正常多样性**
- ABO血型系统
- 葡萄糖-6-磷酸脱氢酶: 34种变体 (抗疟疾)
- 红发: 促黑激素受体突变

## 分子进化证据

**DNA序列相似性**
- 亲缘关系↑ → 序列相似性↑
- 分子钟: 差异 ∝ 分化时间
- ex: 人-猕猴 > 人-绒毛猴 (染色体重排差异)

**跨物种基因功能**
- 人类基因可"治愈"酵母突变
- ex: 人类RhGA蛋白 → 酵母铵盐摄取
- P: 基础功能高度保守

## 核心洞察

**进化的双重性**
- 多样性: 物种分化 → 差异
- 统一性: 共同祖先 → 相似
- P: "青出于蓝而胜于蓝"

**证据汇聚**
∀层次(形态/胚胎/细胞/分子) → 共同祖先证据
∴ 进化论 = 自然解释 vs 神创论难题
