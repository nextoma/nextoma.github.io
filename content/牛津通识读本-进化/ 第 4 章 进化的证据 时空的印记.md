# _第_4_章_进化的证据_时空的印记 Summary

## 地质学基础

**均变论原则**
- P: 物理规律恒定 → 地质变化=物理定律作用结果
- ex: 潮汐摩擦 → 地球转速减慢 (引力大小¬变)
- 沉积岩形成: 侵蚀 → 沉积 → 岩石
- 火山/地震: 局限特定区域 (大陆边缘+大洋中心)

**地层学方法**
- 原则: 旧岩层 < 新岩层 (位置)
- A<B, B<C → A-B-C序列
- 化石特征 → 年代环境信息
- 19世纪发现: 地球≥数千万年 (vs 圣经年表)

**放射性测年**
- 放射性元素衰变 → 子元素 (速率恒定)
- 子元素比例 → 岩石年龄
- 结果: 地球~46亿年

## 化石记录时间轴

**生命演化主要节点**
```
35亿年前: 细菌/单细胞生物
8亿年前: 简单细胞群
7-5.5亿年前: 软体多细胞动物
5.5-5亿年前: 寒武纪大爆发 (硬骨骼动物)
4.4亿年前: 淡水生物+陆生植物
4-3.6亿年前(泥盆纪): 陆地脊椎动物
3.6-2.8亿年前(石炭纪): 树状植物→煤炭
2.5亿年前: 二叠纪大灭绝
2-1.4亿年前(侏罗纪): 恐龙+始祖鸟
1.4亿-6500万年前(白垩纪): 有花植物
6500万年前: 恐龙灭绝(小行星撞击)
700-200万年前: 人类特征出现
```

**化石形成偏差**
- 水生环境 > 陆生环境
- 硬组织 > 软组织
- 浅海生物记录最好, 飞行生物最差
- ex: 腔棘鱼 (6500万年前→1939年重现)

## 过渡物种证据

**始祖鸟**
- 1.2亿年前, 侏罗纪
- 鸟类特征: 羽毛+翅膀
- 爬行类特征: 牙齿+长尾
- 骨架=恐龙, 但会飞
- P: 恐龙→鸟类过渡

**人类进化**
- 达尔文预测: 起源非洲 (基于解剖相似性)
- 化石证据: 距今越近→越似现代人
- 智人: 几十万年前
- 离开非洲: ~150万年前

**有孔虫连续演化**
- 海底沉积物: 长期连续记录
- 身体大小渐变 (除两个间断点)
- 400万年演化历程可追踪

## 化石记录的进化意义

**符合进化预期**
- 简单→复杂: 细菌→真核→多细胞
- 海洋→陆地: ∵早期缺氧→无臭氧层
- 陆地动物→飞行动物 (逻辑必然)
- 周期性灭绝+辐射 (环境巨变)

**霍尔丹检验**
P: "寒武纪前的兔子"将推翻进化论
结果: 迄今未发现此类化石

## 空间分布证据

**海岛生物特征**
1. 物种数量少 (定居困难)
2. 类型偏差 (爬行类+鸟类多, 哺乳类少)
3. 特有种多 (独立进化)
4. 与最近大陆相似 (共同祖先)

**加拉帕戈斯群岛**
- 陆鸟20种, 14种雀类 (比例异常)
- 34%植物特有种
- 达尔文雀: 鸟喙极度多样化
  - 尖嘴地雀: 吸血
  - 形树雀: 用工具捕虫
- DNA证据: 230万年前共同祖先

**夏威夷群岛**
- 果蝇种类>世界其他地方总和
- 体型/翅膀/食性差异巨大
- P: 无竞争→适应辐射

## 生物地理学原理

**岛屿定殖模式**
```
穿越海洋 → 少数物种到达 → 无竞争环境
→ 快速分化 → 特有种形成
```

**达尔文洞察**
问题: 为何岛屿物种似美洲但环境不同?
答案: 共同祖先 + 独立进化

## 核心证据汇聚

**时间证据**
- 化石序列: 简单→复杂
- 过渡物种: 填补间断
- 连续演化: 可追踪谱系

**空间证据**
- 岛屿生物: 定殖+辐射
- 地理隔离 → 独立进化
- DNA验证亲缘关系

**结论**
P: 时空印记 = 进化最强证据
∴ 150年研究充分证明达尔文理论
