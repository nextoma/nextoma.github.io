# 第5章：黑洞中是什么？

## 概述

本章是霍金对其最著名研究成果的详尽阐述。他从约翰·米歇尔1783年提出"暗星"概念讲起，追溯黑洞理论的发展历程。核心内容包括：事件视界、奇点、黑洞无毛定理、面积定理、霍金辐射的发现、信息悖论及其可能的解决方案。霍金解释了黑洞如何"不那么黑"——它们会通过量子效应发射辐射并最终蒸发，这引发了关于信息丢失的深刻悖论。最新研究表明，"超平移毛"可能保存了信息，解决了困扰物理学40年的难题。

## 核心概念

- **逃逸速度** - 摆脱引力束缚所需的最小速度
- **事件视界** - 连光都无法逃脱的边界
- **黑洞无毛定理** - 只有质量、电荷、角动量三个参数
- **奇点** - 密度无限大、时空曲率无限的点
- **宇宙监督猜想** - 奇点都隐藏在视界内
- **面积定理** - 事件视界面积总是增加
- **霍金辐射** - 黑洞通过量子效应发射粒子
- **信息悖论** - 信息是否在黑洞中永久丢失
- **超平移毛** - 可能编码黑洞内部信息的守恒量
- **虚粒子对** - 真空涨落产生的粒子-反粒子对

## 关键发展历程

### 1. 早期概念（1783-1939）

- **约翰·米歇尔（1783）**：暗星概念，逃逸速度超光速
- **爱因斯坦（1915）**：广义相对论，引力弯曲时空
- **爱因斯坦（1939）**：错误地认为恒星不能坍缩
- 大多数科学家抵制引力坍缩思想

### 2. 坍缩理论确立（1930-1965）

- **钱德拉塞卡（1930）**：白矮星最大质量~1.4倍太阳
- **兰道**：中子星也有最大质量
- **奥本海默（1939）**：证明大质量恒星必然坍缩到奇点
- 二战中断研究
- **类星体发现（1963）**：重新激起兴趣

### 3. 现代黑洞理论（1965-1975）

- **罗杰·彭罗斯（1965）**：奇点定理，不对称坍缩仍产生奇点
- **约翰·惠勒（1967）**："黑洞"命名
- **黑洞无毛定理**：只有三个参数
- **霍金面积定理**：视界面积总增加
- **黑洞热力学**：面积类似熵

### 4. 霍金辐射（1974）

- 霍金发现黑洞发射粒子
- 温度与质量成反比
- 小黑洞更热，蒸发更快
- 引发信息悖论

### 5. 信息悖论（1974-至今）

- 信息似乎在黑洞中丢失
- 威胁科学决定论
- 40年未解之谜
- **最新进展（2016）**：超平移毛可能保存信息

## 黑洞的性质

### 基本特征

1. **质量**：决定引力强度
2. **电荷**：通常接近零
3. **角动量（自旋）**：大多数黑洞旋转
4. **事件视界**：无法返回的边界
5. **奇点**：中心无限密度点

### "意大利面条化"效应

- 小黑洞（几倍太阳质量）：
  - 到达视界前被撕裂
  - 脚部引力远大于头部
  - 纵向拉伸，横向压扁

- 超大质量黑洞（百万倍太阳）：
  - 轻松越过视界
  - 整个身体引力差不大
  - 银河系中心：400万倍太阳质量

### 时间效应

- 外部观察者：你似乎永远在视界外徘徊
- 你的形象越来越暗淡、越来越红
- 实际上从视线中消失
- "就外部世界而言，你将永远逝去"

## 面积定理与热力学

### 面积定理（霍金，露西出生后不久）

- 视界面积总是增加
- 两个黑洞合并：新面积 ≥ 原面积之和
- **LIGO验证（2015.9.14）**：探测到引力波，确认面积定理

### 与热力学的类比

| 热力学               | 黑洞          |
| -------------------- | ------------- |
| 熵                   | 视界面积      |
| 温度                 | 表面引力      |
| 第二定律（熵增）     | 面积增加      |
| 第一定律（能量守恒） | 质量-面积关系 |

### 贝肯斯坦建议（1972）

- 视界面积 = 黑洞熵
- 配置数量有限（不是无限）
- 量子不确定性限制最小尺寸
- 信息丢失的度量

### 悖论

- 如果有熵，必有温度
- 如果有温度，必能辐射
- 但黑洞按定义不能发出任何东西
- 矛盾！

## 霍金辐射机制

### 发现过程（1974）

- 研究黑洞附近量子效应
- 发现黑洞以恒定速率发射粒子
- 起初不相信，努力想摆脱
- 粒子谱恰好是热的
- 不得不接受

### 辐射特性

- **温度**：与表面引力成正比，与质量成反比
- **谱**：完美热辐射（黑体辐射）
- **速率**：小黑洞快，大黑洞慢
- **最终命运**：黑洞完全蒸发消失

### 物理机制

**图景1：虚粒子对**

1. 量子真空充满虚粒子对
2. 一个落入黑洞
3. 另一个失去伙伴，可能逃逸
4. 看起来像黑洞发射粒子

**图景2：时间倒转**

1. 落入的反粒子 = 正在时间中倒退的粒子
2. 被引力场散射
3. 在时间中前进
4. 逃逸到无穷远

### 实际例子

**太阳质量黑洞**：

- 泄漏极慢，无法探测
- 温度约10⁻⁷K（接近绝对零度）

**山岳大小迷你黑洞**：

- 功率：1000万兆瓦（足够世界供电）
- 发射X射线和伽马射线
- 可能在极早期宇宙形成
- 一直在寻找，未发现

**微型黑洞（LHC）**：

- 可能在额外维度中形成
- 引力在额外维度更强
- 会辐射粒子，易于识别
- 霍金期待获诺贝尔奖

## 信息悖论

### 问题的严重性

**信息似乎丢失**：

1. 物质落入黑洞
2. 黑洞蒸发，发射随机辐射
3. 关于落入物的信息（除质量和自旋外）丢失
4. 可能发射任何东西（概率极低但可能）：
   - 工作电视机
   - 莎士比亚全集
   - 更可能是热辐射

**威胁科学决定论**：

- 200多年来相信决定论
- 如果信息丢失，无法预测未来
- 原则问题，不仅是实践问题
- 虚拟黑洞可能无处不在

**更糟的是**：

- 无法确定过去
- 历史和记忆可能是幻觉
- "没有过去，我们就失去了自己的本我"

### 可能的解决方案

**传统观点**：

- 许多科学家认为信息不应丢失
- 但多年无人提出保持信息的机制
- 理论物理学最大未解问题之一

**新发现：超平移对称（2016）**

霍金、佩里、斯特罗明格的工作：

1. **远处时空的对称性**：
   - 平坦时空：平移和旋转对称
   - 远离黑洞处：接近平坦
   - 发现：无穷多个"超平移对称"
   - 每个对称对应一个守恒量

2. **超平移荷**：
   - 黑洞不只有3个参数（质量、电荷、角动量）
   - 还携带无穷多个超平移荷
   - "黑洞有非常大量的超平移的毛"

3. **超旋转荷**：
   - 另一类守恒量
   - 与超旋转对称相关
   - 理解尚不透彻

4. **信息存储**：
   - 超平移和超旋转荷可能编码信息
   - 信息没有丢失
   - 存储在视界上
   - **最新计算（哈科等）**：可能解释全部熵

### 当前状态

- 问题仍未完全解决
- 但正趋向解决
- 霍金"对此很乐观"
- "请关注此领域的进展"

## 实验证据

### 1. 引力波探测（LIGO，2015）

- 探测到黑洞合并
- 验证面积定理
- 测量质量和角动量
- 确定视界面积

### 2. 卡西米尔效应

- 两块金属板之间虚粒子较少
- 能量密度为负
- 实验测量到微小压力
- 证明虚粒子真实存在

### 3. 兰姆移位

- 虚粒子对氢原子能级的影响
- 可测量的小偏移
- 间接证实虚粒子

### 4. 寻找迷你黑洞

- 至今未找到（遗憾）
- 如果找到，霍金获诺贝尔奖

### 5. LHC实验

- "我终究可以获得诺贝尔奖"

## 金句

1. **关于黑洞的奇异性**：

   > "黑洞比科幻作家梦想的任何东西都更奇怪，但它们是坚实的科学事实。"

2. **黑洞无毛**：

   > "从外面看，你不能知道黑洞里面是什么。无论你投入什么，或者无论它如何形成，黑洞看起来都是一样的。"（约翰·惠勒："黑洞无毛"）

3. **意大利面条化**：

   > "如果你的脚首先跌入黑洞，引力拉你的脚比拉你的头更厉害，因为脚离黑洞更近。结果你的纵向被伸展，而横向被压扁。"

4. **尤里卡时刻**：

   > "在我的女儿露西出生后不久，我有了一个尤里卡时刻。我发现了面积定理。"

5. **霍金辐射的发现**：

   > "令我十分惊讶的是，我发现黑洞似乎以恒定的速率发射粒子...我相当努力想摆脱这种令人尴尬的效应。但是我越深入思考，它越拒绝消失。"

6. **信息与本我**：

   > "正是过去告诉我们，我们是谁。没有它，我们就失去了自己的本我。"

7. **太空旅行者的建议**：

   > "如果你想探索一个黑洞的内部，一定要选择一个大黑洞。"

8. **最终问题**：
   > "对于太空旅行者，跌入黑洞是否为坏消息？绝对是坏消息。"
