## 先验唯心论基础

### 先验方法特征

**"先验"定义**
- 论证"超越"经验探究限度 → 建立经验先天条件
- ≠ 经验论证 | 关注认识模式 > 客体本身
- Notion: 先验客体 := 超越经验,无法观察/因果关联

**理论核心**
- P: 知性法则 = 先天真理 | 认识能力 ↔ 认识对象的和谐
- 思想形式 ↔ 实在的先天本质完全一致
- 关键问题: 思想决定世界? vs 世界决定思想?

### 自我意识前提

**"我们"的含义**
- ≠ 心理学研究 | ≠ 抽象作者声音
- := 任何能使用"我"指代自己的存在物
- P: 整个哲学出发点 := 自我意识

**两部《批判》核心问题**
- 第一批判: 有自我意识的存在者必须思考什么?
- 第二批判: 他必须做什么?
- 结论: 使怀疑论可能的条件 → 证明怀疑论虚假

## 主观演绎: 知性与感性

### 知识的双重来源

**感性 vs 知性**
- 感性: 直觉能力 | 感觉状态/限定
- 知性: 概念能力 | 能动的判断活动

**两派错误**
- 经验论: 按感觉模式解释概念 @休谟"褪色印记"
- 唯理论: 把感知视为混乱的概念思维 @莱布尼茨
- P: "莱布尼茨将现象理智化,洛克将概念感性化"

### 先验综合

**判断的要求**
- 感性 + 知性结合 → 判断
- "没有感知,无客体提供;没有知性,无对象可思考"
- "没有内容的思想空洞,没有概念的直觉盲目"

**综合的本质**
- 概念 + 直觉的综合 → 真正经验
- 两阶段: 纯粹综合(直觉组合) → 判断(概念赋形)
- 先验综合 ≠ 心理事实 | 在经验中预设 ≠ 派生

### 先天概念/范畴

**范畴的必然性**
- 经验论错误: 混淆经验与感觉
- 经验已包含概念 ∵ 先验综合
- 知性含有先天概念 → 判断基本形式

**十二范畴**
- 重点: 实体(莱布尼茨源头) + 因果(休谟攻击)
- 实体: 独立存在 + 承载特性
- 通过抽象达到范畴 → 判断所有形式

## 空间与时间: 直觉形式

### 为何不是概念

**独特地位**
- 空间/时间 ≠ 概念 := 直觉形式
- 原因: 概念普遍/多实例 vs 空间时间唯一
- P: 只有一个空间,一个时间 | 各部分构成整体

**功能区分**
- 时间: "内感觉"形式 | 所有心智状态
- 空间: "外感觉"形式 | 独立世界的表象
- 每一感觉必带时间/空间组织印记

### 数学的先天性

**两种先天真理**
- 数学: 不证自明 | 直觉的直接性/明确性
- 形而上学: 有争议 | 从思维派生,需论证

**康德解释**
- 数学 := 先天直觉科学
- 几何先天性 → 源于主体 ≠ 客体
- 空间作为直觉形式 → 空间先天知识可能

## 客观演绎: 统觉与客观性

### 统觉的先验统一

**核心概念**
- Notion: 统觉 := 主体能说"这是我的"的经验 @莱布尼茨
- 统一性: "我思"伴随所有知觉 @笛卡尔
- 先验性: 非经验获得 | 经验的先决条件

**意识统一的必然性**
- 直接意识: 经验属于我 | 怀疑不可能
- ≠ 经验推论 | := 经验前提
- 最低描述: 存在意识统一

### 先验演绎论证

**核心论点**
- 统觉统一 → 需要客观世界
- 主体性条件: 其所似 = 似其所是
- 但需客观真理知识 → 独立存在的事物世界

**三个思想路径**
1. 识别经验: 需识别对象为连续 → 独立于感知
2. 时间识别: 主体持续需实体性 → 因果关系
3. 时间排序: 区分此时/彼刻 → 需独立客体/规律

**当代发展**
- @维特根斯坦"私人语言"论证
- 相同前提/结论 | 依赖意义学说 ≠ 时间形而上学
- 主体 → 主体共同体 → 公共世界

## 原则与科学基础

### 范畴的原则

**原则性质**
- := 范畴"客观应用规则"
- 双重性: 思考必然性 + 世界可理解条件
- 限制: 仅适用"可能经验的对象" ≠ 物自体

### 经验的类比

**与牛顿力学呼应**
- 三范畴: 实体/因果/相互作用 ↔ 牛顿三定律
- 守恒定律: 变化需不变实体 → 科学统一性
- P: 解释变化 ≠ 解释存在

**因果必然性**
- "原因与结果关系 := 经验判断客观有效性条件"
- 客观性 → 持续性 → 因果关系
- 因果性 → 行动 → 力 → 实体

**形而上学意义**
- 了解真理 = 了解必然性
- vs 经验论: 自然无必然性
- vs 现代理论: 现实理解 < 必然/可能理解

## 结论: 中间道路

**对莱布尼茨**
- 充足理由律: 无法证明 @"相关人士公认"
- 但因果律(先验等价物)可证明
- 限于表象世界/时间事件

**对休谟**
- 先天原理可能 ≠ 经验论否认
- 但观点确为知识组成部分
- 世界须向观点呈现表象

**先验唯心论定位**
- 经验主义 ↔ 理性主义的中间道路
- 承认先天知识 ∧ 否认非观察形而上学
- 自我中心论方法: 从视角内部回答哲学问题
