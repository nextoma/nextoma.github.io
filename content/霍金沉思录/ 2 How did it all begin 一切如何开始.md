# *2_How_did_it_all_begin*一切如何开始 总结

## 核心问题：宇宙如何开始？其本质是什么？

### 哲学语境

**哈姆雷特悖论**："被束缚在果壳中，却是无限空间之王"

- 物理限制 ↔ 无限的思想探索
- **中心问题**：宇宙是无限的吗？有开端吗？会终结吗？

### 历史观点

**创世神话**

- 波桑戈人：本巴神呕吐出太阳、月亮、星星 → 假定人类起源较近
- 厄谢尔主教：创世于公元前4004年10月22日下午6点
- 亚里士多德：宇宙永恒（∵永恒>被创造）→ 洪水重置文明

**康德二律背反**：

- 正题：若有开端 → 为何在开端前等待无限时间？
- 反题：若永恒 → 为何需要无限时间到达现在？
- 两者都假设时间=绝对

### 爱因斯坦革命（1915）

**原理：广义相对论**

- 时空≠绝对背景
- 时空=动态的，由物质/能量塑造
- 时间只在宇宙内定义
- **关键洞察**："开端之前"无意义（如"南极以南"）

### 观测证据

**夜空黑暗问题**（奥伯斯佯谬）

- 若宇宙永恒且静态 → 每条视线都击中恒星 → 天空亮如太阳
- 夜空黑暗 → 宇宙未以当前状态无限存在
- 恒星必定在有限时间前"点亮"

**哈勃的发现**（1920年代）：

1. 星云=遥远星系（数百万/数十亿光年远）
2. **宇宙膨胀**：星系退行，v∝距离
   - 含义：过去→星系更近→100-150亿年前：同一点

### 失败的替代理论

**稳恒态理论**（邦迪、戈尔德、霍伊尔 1948）：

- 持续创造新物质 → 老星系分离时形成新星系
- 宇宙始终看起来一样
- **被证伪**：射电源观测显示过去密度更高

**反弹宇宙**（栗弗席兹&哈拉特尼科夫）：

- 收缩相 → 物质错过 → 无奇点反弹
- 声称不需要开端
- **被证伪**：彭罗斯-霍金奇点定理

### 奇点定理

**彭罗斯-霍金结果**：

- 若广义相对论正确+合理条件 → 宇宙必有开端
- 开端=大爆炸奇点（无限密度点）
- 广义相对论在奇点崩溃 → 无法预测宇宙如何开始

### 量子宇宙

**不确定性原理**（海森伯 1927）：

- 不能同时精确知道位置和速度
- 宇宙=宇宙赌场，上帝确实掷骰子
- 宇宙很小时 → 掷骰子次数少 → 不确定性关键

**费曼多重历史**：

- 宇宙有许多可能历史，各有概率
- 必须结合广义相对论+量子力学得完整理论
- 需要宇宙边缘的边界条件

### 无边界设想

**哈特尔-霍金思想**：

- 宇宙在虚时间中无边界
- 避免需要边界条件
- 虚时间中多重历史 → 各决定实时间历史
- **选择**：人存原理 → 我们观察允许智慧生命的历史

### 人存原理应用

**为何3维空间？**

- 2维：消化道会将动物一分为二
- 4维+：无稳定轨道（引力∝1/r³而非1/r²）
- 仅3维允许稳定原子、行星、生命

**M理论**：10空间维+1时间维

- 7维卷曲很小
- 仅观察到3个大的平坦维度

### 暴胀与量子涨落

**暴胀期**：

- 早期宇宙膨胀至少10²⁷倍
- 解释宇宙微波背景（CMB）的均匀性

**量子涨落**（霍金 1982）：

- 不确定性原理 → 暴胀期间密度变化
- 星系形成的种子
- 我们是早期量子涨落的产物
- **证据**：COBE（1993）、WMAP（2003）、普朗克卫星证实预言

### 宇宙的未来

**临界密度决定命运**：

1. **ρ > ρ临界**：引力获胜 → 大挤压（约200亿年后坍缩）
2. **ρ < ρ临界**：永恒膨胀 → 寒冷、空虚的宇宙
3. 两种情况：还有数十亿年

### M理论与多重宇宙

- 预言从无创生许多宇宙
- 各有不同历史和物理定律
- 大多不适合生命
- 我们观察到允许智慧观察者的那个

### 关键洞察

1. **时间始于大爆炸** → 无"之前"（∄时间供因果/创造者）
2. **宇宙=量子事件** → 自发创生可能
3. **我们选择自己的历史** → 人存原理约束观察
4. **上帝掷骰子** → 量子不确定性是根本的
5. **被束缚却无限** → 有限宇宙包含无限可能

**最终问答**："大爆炸之前发生了什么？"

- 根据无边界设想，问题无意义
- 如同问"南极以南是什么？"
- 时间概念仅存在于我们的宇宙中
