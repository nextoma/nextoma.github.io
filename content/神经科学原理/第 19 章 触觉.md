# 第*20*章\_痛觉 Summary

## 核心定义与原理

**疼痛定义** (IASP)

- 疼痛 := 与实际或潜在组织损伤相关的不愉快感觉和情感体验
- P: 疼痛 = 感觉成分 + 情感成分 + 认知成分
- P: 疼痛 ≠ 伤害性感受 (主观体验 vs 神经过程)
- 生物学意义: 保护 + 促进愈合 + 学习避免

**疼痛vs伤害性感受**

- 伤害性感受 := 伤害性刺激的神经编码
- 疼痛 := 伤害性信息的意识感知
- ∃ 伤害性感受 ∧ ¬疼痛 (麻醉/脊髓损伤)
- ∃ 疼痛 ∧ ¬伤害性感受 (幻肢痛/中枢痛)

## 20.1 伤害性受体系统

### 伤害性受体类型

**机械伤害性受体**

- Aδ-HTM (高阈值机械)
- 阈值: >5mN (vs 触觉<1mN)
- 功能: 锐利物体, 强压
- 可能通道: Piezo2变体?

**热伤害性受体**

- TRPV1: >43°C激活, 辣椒素敏感
- TRPV2: >52°C极热
- TRPM3: 有害热备份系统
- 特点: 温度阈值可调(炎症↓)

**冷伤害性受体**

- TRPA1: <17°C激活, 芥末油敏感
- TRPM8: 15-25°C, 极冷时痛觉
- 冷痛悖论: 冷受体 → 灼烧感

**化学伤害性受体**

- 酸: ASIC1/3 (pH<6.5)
- ATP: P2X3 (细胞损伤释放)
- 炎症介质: [缓激肽, PGE2, NGF, TNF-α]
- 功能: 组织损伤检测

**多模式伤害性受体**

- C-PMN: 65%的C纤维
- 响应: 机械 + 热 + 化学
- 广谱检测 → 综合评估

**沉默伤害性受体**

- 正常: 无反应/高阈值
- 炎症后: 敏化激活
- 占C纤维: ~30%
- 临床意义: 炎症痛主要贡献者

### 传导纤维特性

**Aδ纤维** (快痛)

- 直径: 2-5μm, 速度: 12-30m/s
- 髓鞘: 薄
- 感觉: 锐利, 刺痛, 定位准确
- 功能: 即刻警告, 撤退反射

**C纤维** (慢痛)

- 直径: 0.4-1.2μm, 速度: 0.5-2m/s
- 髓鞘: 无
- 感觉: 钝痛, 灼烧, 弥漫
- 功能: 持续警告, 保护行为

## 20.2 脊髓痛觉处理

### 背角层级组织

**Rexed分层功能**

- I层(边缘层): 伤害特异性, 投射神经元
- II层(胶状质): 局部回路, 门控
  - IIo: C纤维末梢, 肽能
  - IIi: 非肽能, PKCγ神经元
- III-IV层: 低阈值机械, 触觉
- V层: WDR神经元(广动态范围)
- 深层: 运动前神经元

**投射神经元类型**

- NS (伤害特异): 仅响应伤害刺激
- WDR (广动态): 触觉+伤害性, 编码强度
- 分布: I层(80% NS), V层(60% WDR)

### 神经递质系统

**兴奋性传递**

- 谷氨酸: 快速传递(AMPA/NMDA)
- P物质: 缓慢延长(NK1受体)
- CGRP: 增强P物质作用
- BDNF: 中枢敏化介导

**抑制性调控**

- GABA: 快速抑制(门控)
- 甘氨酸: 补充抑制
- 强啡肽: 内源性镇痛
- 加兰丁: 伤害性调节

### 门控理论

**Melzack-Wall理论** (1965)

- P: 非伤害性输入 → 抑制伤害性传递
- 机制: Aβ纤维 → SG兴奋 → 投射神经元抑制
- 临床应用: TENS, 摩擦镇痛
- 现代修订: 多层次门控

**门控回路**

- Aβ → PKCγ神经元 → PV神经元 → 抑制
- 正常: 触觉不产生疼痛
- 病理: 去抑制 → 异常性疼痛

## 20.3 上行痛觉通路

### 脊髓丘脑束

**外侧脊髓丘脑束** (快痛)

- 起源: I/V层NS神经元
- 交叉: 前连合
- 终止: VPL/VPM (体感定位)
- 功能: 痛觉辨别成分

**前脊髓丘脑束** (慢痛)

- 起源: V-VII层WDR神经元
- 终止: 髓板内核(情感成分)
- 功能: 痛觉情感/自主反应

### 其他上行通路

**脊髓网状束**

- → 网状结构 → 觉醒/自主反应
- 弥漫投射 → 全脑激活

**脊髓中脑束**

- → PAG → 下行痛觉调制
- → 上丘 → 定向反应

**脊髓臂旁束**

- → 臂旁核 → 杏仁核
- 直接通路 → 情绪/恐惧

**脊髓下丘脑束**

- → 下丘脑 → 神经内分泌
- 应激反应激活

## 20.4 脑部痛觉处理

### 丘脑中继

**外侧丘脑** (感觉辨别)

- VPL/VPM: 精确定位
- Po: 伤害特异性
- → S1/S2: 感觉分析

**内侧丘脑** (情感动机)

- MD: 情感评估
- 髓板内核: 唤醒
- → ACC/岛叶: 情感处理

### 皮层痛觉网络

**一级体感皮层(S1)**

- 位置: 中央后回
- 功能: 痛觉定位, 强度编码
- 特点: 对侧支配, 体位映射

**二级体感皮层(S2)**

- 位置: 顶盖
- 功能: 痛觉识别, 注意
- 特点: 双侧感受野

**岛叶皮层**

- 后岛叶: 感觉强度
- 前岛叶: 情感评价
- 功能: 内感受整合

**前扣带皮层(ACC)**

- 背侧: 认知评价
- 前部: 情感成分
- 功能: 痛苦体验, 共情疼痛

**前额叶皮层**

- 功能: 痛觉调节, 预期
- 安慰剂效应起源
- 认知调控中心

### 痛觉矩阵概念

**分布式处理**

- 无单一"痛觉中枢"
- 平行处理网络
- 功能分离与整合

**突显网络激活**

- ¬特异于疼痛
- 威胁检测系统
- 注意资源分配

## 20.5 下行痛觉调制

### PAG-RVM系统

**中脑导水管周围灰质(PAG)**

- 输入: 皮层, 下丘脑, 脊髓
- 功能: 整合中枢
- 神经递质: 内啡肽, P物质

**延髓头端腹内侧部(RVM)**

- ON细胞: 易化痛觉
- OFF细胞: 抑制痛觉
- 中性细胞: 状态依赖
- 5-HT/NE下行投射

### 内源性镇痛系统

**阿片系统**

- μ受体: 主要镇痛
- δ/κ受体: 调节作用
- 内源性配体: [内啡肽, 脑啡肽, 强啡肽]
- 作用位点: PAG, RVM, 脊髓背角

**单胺系统**

- 5-HT: 双向调节(抑制/易化)
- NE: α2受体镇痛
- DA: 奖赏/动机调节

**大麻素系统**

- CB1受体: 中枢分布
- CB2受体: 外周免疫
- 内源性: AEA, 2-AG
- 功能: 炎症痛调节

## 20.6 痛觉敏化机制

### 外周敏化

**炎症汤成分**

- 离子: [H⁺, K⁺]
- 脂类: [PGE2, LTB4]
- 肽类: [缓激肽, P物质]
- 细胞因子: [TNF-α, IL-1β, IL-6]
- 神经营养因子: [NGF, GDNF]

**分子机制**

- TRPV1磷酸化 → 阈值↓
- Nav1.7/1.8表达↑ → 兴奋性↑
- PKA/PKC激活 → 通道敏化
- 基因转录 → 表型改变

**神经源性炎症**

- 轴突反射 → CGRP/SP释放
- 血管扩张 + 血浆渗出
- 肥大细胞脱颗粒
- 正反馈放大

### 中枢敏化

**脊髓机制**

- NMDA受体激活 → Ca²⁺内流
- CaMKII/PKC激活 → AMPA磷酸化
- NO/PGE2产生 → 逆行信号
- 小胶质细胞激活 → 细胞因子

**突触可塑性**

- LTP诱导: 高频C纤维输入
- 沉默突触激活
- 抑制性中间神经元凋亡
- KCC2下调 → GABA兴奋

**表型转换**

- Aβ纤维 → P物质表达
- 神经元 → 胶质激活
- 急性 → 慢性转变

## 20.7 临床疼痛分类

### 伤害性疼痛

**急性疼痛**

- 原因: 组织损伤
- 机制: 伤害受体激活
- 特点: 保护性, 自限性
- 治疗: NSAIDs, 阿片类

**炎症性疼痛**

- 原因: 组织炎症
- 机制: 外周敏化
- 特点: 痛觉过敏, 触诱发痛
- 治疗: 抗炎药, COX抑制剂

### 神经病理性疼痛

**外周神经病理性痛**

- 原因: 神经损伤/疾病
- 机制: 异位放电, 交感耦合
- 症状: 灼烧, 针刺, 电击样
- ex: 糖尿病神经病, 带状疱疹后

**中枢神经病理性痛**

- 原因: CNS损伤
- 机制: 中枢去抑制, 重组
- 症状: 自发痛, 痛觉过敏
- ex: 脊髓损伤, 卒中后

### 功能性疼痛

**纤维肌痛**

- 机制: 中枢敏化, 下行易化
- 特点: 广泛疼痛, 疲劳
- 生物标志物: ⊥

**肠易激综合征**

- 机制: 内脏超敏
- 脑-肠轴功能紊乱
- 应激相关

## 20.8 疼痛评估

### 主观评估

**疼痛强度**

- VAS (视觉模拟评分): 0-10
- NRS (数字评分): 0-10
- 面部表情量表: 儿童

**疼痛性质**

- McGill问卷: 感觉/情感/评价
- DN4: 神经病理性筛查
- PainDETECT: 机制分类

### 客观测量

**定量感觉测试(QST)**

- 机械: Von Frey, 压痛计
- 温度: 热痛/冷痛阈值
- 时间总和: 重复刺激
- 条件性疼痛调制(CPM)

**神经生理**

- 诱发电位: 激光/电刺激
- 神经传导: 纤维功能
- 皮肤活检: 神经纤维密度

**脑成像**

- fMRI: 痛觉网络激活
- PET: 受体占位, 代谢
- MEG: 时间动态

## 20.9 治疗策略

### 药物治疗

**非甾体抗炎药(NSAIDs)**

- 机制: COX抑制 → PG合成↓
- 适应: 炎症性疼痛
- 限制: 胃肠/肾毒性

**阿片类药物**

- 机制: μ受体激动
- 适应: 重度急性/癌痛
- 风险: 耐受, 依赖, 呼吸抑制

**抗抑郁药**

- TCAs: 去甲肾上腺素再摄取抑制
- SNRIs: 5-HT/NE双重抑制
- 适应: 神经病理性痛

**抗惊厥药**

- 加巴喷丁: Ca²⁺通道α2δ亚基
- 卡马西平: Na⁺通道阻断
- 适应: 神经病理性痛

**局部治疗**

- 利多卡因: Na⁺通道阻断
- 辣椒素: TRPV1脱敏
- 适应: 局部神经病理性痛

### 非药物治疗

**神经调控**

- TENS: 门控理论应用
- SCS: 脊髓刺激
- DBS: 深部脑刺激
- tDCS: 经颅直流电刺激

**心理治疗**

- CBT: 认知行为疗法
- 正念: 疼痛接纳
- 生物反馈: 自主调节

**物理治疗**

- 运动: 内啡肽释放
- 手法: 门控激活
- 热/冷: 受体调节

### 新兴靶点

**离子通道**

- Nav1.7: 先天性无痛靶点
- TRPV1: 选择性拮抗剂
- P2X3: ATP受体阻断

**神经营养因子**

- Anti-NGF抗体: tanezumab
- GDNF调节剂

**表观遗传**

- HDAC抑制剂
- miRNA调节

## 20.10 特殊疼痛状态

### 先天性无痛症

**Nav1.7缺失** (CIP)

- SCN9A突变 → 无痛觉
- 嗅觉丧失
- 认知正常
- 启示: Nav1.7特异性镇痛药

**NTRK1突变** (CIPA)

- TrkA受体缺失
- 无痛 + 无汗 + 智力障碍
- NGF信号缺失

### 慢性疼痛转化

**危险因素**

- 遗传: COMT, GCH1变异
- 心理: 灾难化, 恐惧回避
- 社会: 支持缺乏

**生物标志物**

- 脑结构: 灰质密度↓
- 功能连接: DMN改变
- 分子: microRNA谱

### 安慰剂镇痛

**神经机制**

- 预期 → PFC激活
- 内源性阿片释放
- PAG-RVM激活
- 可被纳洛酮阻断

**影响因素**

- 期望强度
- 条件化学习
- 社会因素
- 遗传变异(COMT)

## 关键洞察

1. **多维构建**: 疼痛 = 感觉 + 情感 + 认知, ¬简单的刺激-反应
2. **双向调制**: 上行传递 ↔ 下行调控, 动态平衡决定体验
3. **可塑性双刃剑**: 适应性保护 ↔ 病理性慢性化
4. **个体化机制**: 遗传 × 环境 × 心理 → 疼痛表型
5. **治疗复杂性**: 多靶点 + 多模式 + 个体化 = 最佳策略
