# 第*56*章\_决策和意识 Summary

## 🧠 决策神经科学核心

```
决策过程：
信息整合 → 价值评估 → 选择执行 → 结果反馈
```

## 🎯 决策神经回路

### 前额叶决策系统

```
关键脑区功能：
orbitofrontal cortex(OFC): 价值表征，结果预期
dorsolateral PFC(dlPFC): 工作记忆，认知控制
anterior cingulate cortex(ACC): 冲突监测，错误检测
ventromedial PFC(vmPFC): 情感整合，社会决策
```

### 基底核决策回路

```
选择机制：
纹状体: 动作价值编码
黑质致密部: 多巴胶能学习信号
苍白球: 竞争性选择门控
```

## 💰 价值评估机制

### 预期价值计算

```
价值公式：
EV = ∑(概率i × 效用i)
风险偏好调节：确定性等值
```

**神经基础**：

- OFC: 抽象价值表征
- 纹状体: 动作特异性价值
- 前岛叶: 主观价值感受

### 跨期选择

```
时间折扣：
未来奖励价值 = 即时价值 × 折扣函数
双曲线折扣 vs 指数折扣
```

**神经机制**：

- vmPFC: 延迟奖励价值
- 边缘系统: 即时奖励冲动
- dlPFC: 冲动控制

## 🤔 认知控制系统

### 执行功能网络

```
控制层次：
中央执行: 目标维持，注意控制
认知灵活性: 任务切换，规则更新
工作记忆: 信息临时存储操作
抑制控制: 冲动抑制，干扰阻断
```

### 冲突监测机制

```
ACC功能：
冲突检测 → 控制信号 → PFC调节
错误相关负性电位(ERN)
```

## 🎭 情绪与决策

### 躯体标记假说

```
Damasio理论：
情绪反应 → 躯体感觉 → 决策指导
vmPFC损伤 → 情绪缺失 → 决策异常
```

### 双系统模型

```
系统竞争：
系统1: 快速，自动，情绪驱动
系统2: 缓慢，控制，理性分析
```

## 🧭 意识神经理论

### 全局工作空间理论

```
Dehaene模型：
局部加工 → 全局广播 → 意识内容
前额-顶叶网络: 意识接入
```

**关键特征**：

- 阈值效应: 全局点火
- 可报告性: 言语表达
- 注意调节: 意识门控

### 整合信息理论

```
Tononi理论：
意识 = 整合信息量(Φ)
复杂性 + 整合性 = 意识水平
```

### 高阶思维理论

```
元认知理论：
一阶表征 + 二阶监测 = 意识体验
前额叶元认知网络
```

## 📊 意识测量方法

### 行为指标

```
意识标准：
主观报告: 内省自我评估
客观行为: 区分判断能力
元认知: 信心判断准确性
```

### 神经指标

```
意识相关负电位：
P3b: 意识接入标记
后期正电位: 可报告性
全局点火: 高频γ同步
```

## 🔄 决策学习机制

### 强化学习

```
预测误差学习：
δ = 实际奖励 - 预期奖励
多巴胶能信号编码预测误差
```

**时间差分学习**：

- 连续预测更新
- 价值函数逐步精化
- 策略梯度优化

### 模型基础vs无模型学习

```
双学习系统：
模型基础: 前瞻性，灵活，认知资源消耗大
无模型: 习惯化，自动，计算高效
```

## ⚡ 关键要点

1. **分布式处理**：决策涉及多脑区协调整合
2. **价值计算**：OFC等区域进行抽象价值表征
3. **认知控制**：前额叶系统调节冲动和执行
4. **情绪整合**：情绪信号指导理性决策
5. **意识门控**：全局工作空间理论解释意识机制
6. **学习优化**：强化学习持续优化决策策略
7. **个体差异**：决策风格反映神经回路差异

```
决策价值 = ∑(概率 × 效用 × 时间折扣 × 风险调节)
意识强度 = 信息整合度 × 全局可及性 × 元认知精度
学习率 = 预测误差 × 不确定性 × 注意权重
```
