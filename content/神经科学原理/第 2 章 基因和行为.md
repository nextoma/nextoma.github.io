# 第*3*章\_神经细胞、神经回路和行为\_Summary

## 核心概念

**P:** 神经系统基本组织原理

- `神经元` := 基本信号单元 [~860亿个]
- `胶质细胞` := 支持性细胞 [数量2-10倍于神经元]
- `行为` := `神经回路输出` → `环境响应`

**Notion:**

- `N` = 神经元
- `G` = 胶质细胞
- `S` = 信号
- `C` = 连接/回路
- `B` = 行为

## 3.1 神经系统细胞类型

### 3.1.1 神经细胞结构与功能

**P:** 神经元形态学组成

```
N := {细胞体, 树突, 轴突, 突触前末梢}
```

**结构-功能关系:**

- `细胞体` → 代谢中心 [细胞核 + 内质网]
- `树突` → 输入接收器 [树状分支]
- `轴突` → 信号传导器 [0.1mm-1m]
- `突触前末梢` → 输出释放器

**P:** 动作电位特性

- `AP` := 全有或全无脉冲 [100mV振幅]
- `传导速度` := 1-100 m/s
- `信息编码` ≠ 信号形式，而是传导通路

**重要原理:**

1. **神经元学说:** ∀N都是独立细胞单元
2. **动态极化原理:** 电信号单向流动 [树突/细胞体 → 轴突]
3. **连接特异性:** N间特定连接模式，非随机连接

### 神经元分类

**形态学分类:**

- `单极神经元` := {1个主要突起} [主要见于无脊椎动物]
- `双极神经元` := {椭圆细胞体, 1树突, 1轴突} [感觉细胞]
- `假单极神经元` := 双极融合型 [体感信息传递]
- `多极神经元` := {1轴突, 多树突} [脊椎动物主导]

**功能分类:**

- `感觉神经元` := 外周 → CNS信息传递
- `运动神经元` := CNS → 肌肉/腺体命令传递
- `中间神经元` := {中继型, 局部型}

**ex:**

- 脊髓运动神经元接受: ~10,000个接触 [1,000细胞体 + 9,000树突]
- 小脑浦肯野细胞接受: ~1,000,000个接触

### 3.1.2 胶质细胞支持功能

**P:** 胶质细胞特征

- `数量关系` := G数量 = (2-10) × N数量
- `功能` ≠ 电信号参与，而是支持与调节
- `作用` := {绝缘, 环境维持, 血管耦合, 连接指导}

## 3.2 神经细胞回路组织

### 膝跳反射模型

**P:** 简单反射回路

```
拉伸刺激 → 肌梭激活 → 感觉神经元 → 脊髓 → 运动神经元 → 肌肉收缩
```

**信息流:**

1. `感觉信息`: 肌肉 → CNS
2. `运动命令`: CNS → 目标肌肉
3. `抑制指令`: CNS → 拮抗肌
4. `协调信息`: 局部 → 高级中枢

**连接模式:**

- `发散` := 1个N → 多个目标细胞 [1感觉N → 45-50运动N]
- `汇聚` := 多个N → 1个目标细胞 [~130感觉N → 1运动N]

### 抑制回路

**前馈抑制:**

```
感觉N → {伸肌运动N(+), 抑制性中间N → 屈肌运动N(-)}
```

**反馈抑制:**

```
运动N → {肌肉(+), 抑制性中间N → 自身(-)}
```

**功能:**

- 前馈抑制 ⟹ 协调拮抗肌群
- 反馈抑制 ⟹ 自我调节机制

## 3.3 神经信号组织原理

### 3.3.1 四组件信号模型

**P:** 统一信号处理框架

```
模型神经元 := {输入组件, 整合组件, 传导组件, 输出组件}
```

**信号类型转换:**

1. `分级输入信号` [受体电位/突触电位]
2. `触发信号` [阈值判断]
3. `全有或全无传导信号` [动作电位]
4. `输出信号` [神经递质释放]

### 膜电位基础

**P:** 静息膜电位机制

- `静息电位` := -65mV [范围: -40至-80mV]
- `维持机制` := {Na+-K+泵, 离子通道选择性渗透}
- `离子浓度梯度`:
  - Na+: 细胞外/细胞内 = 10:1
  - K+: 细胞内/细胞外 = 20:1

**兴奋性变化:**

- `去极化` := 膜电位↓ → AP产生能力↑ [兴奋性]
- `超极化` := 膜电位↑ → AP产生能力↓ [抑制性]

### 3.3.2 输入组件 - 分级局部信号

**受体电位特征:**

- `振幅` ∝ `刺激强度`
- `持续时间` ∝ `刺激持续时间`
- `传播` := 被动衰减 [~1-2mm, 1mm处振幅≈1/3]
- `类型` := 多数去极化，少数超极化(视网膜)

**突触电位特征:**

- `产生机制` := 神经递质 → 受体结合 → 离子通道开启
- `振幅` ∝ 递质释放量
- `极性` := 取决于受体类型，非递质类型
- `传播` := 被动，局限性

### 3.3.3 触发区 - 阈值整合

**P:** 轴突起始段功能

- `位置` := 电压敏感Na+通道密度最高区域
- `功能` := ∑(输入信号) ≥ 阈值 → AP产生
- `整合原理` := 空间+时间总和

### 3.3.4 传导组件 - 动作电位传播

**AP特征:**

- `全有或全无` := 阈值以下无信号，阈值以上相同幅度
- `标准化` := 振幅、持续时间恒定 [~1ms持续，100mV振幅]
- `再生传导` := 沿轴突无损传播
- `髓鞘化` ⟹ 传导速度↑

**信息编码:**

- `强度` := AP频率
- `持续时间` := AP序列时间跨度
- `模式` := {规律放电, 突发放电, 静默期模式}

**P:** 标记线原理

- `信息类型` ≠ 信号本身，而是 `传导通路`
- `ex:` 视觉信号通路 ≠ 触觉信号通路

### 3.3.5 输出组件 - 神经递质释放

**释放机制:**

```
AP到达 → Ca2+内流 → 囊泡融合 → 胞吐 → 递质释放
```

**信号特征:**

- `递质量` ∝ AP数量和频率
- `效应` := 取决于突触后受体，非递质种类
- `同一递质` → 不同受体 → 不同效应

### 牵张反射信号转换示例

**完整信号链:**

```
拉伸 → 受体电位(分级) → AP(频率编码) → 递质释放 → 突触电位 →
运动神经元AP → 肌肉递质释放 → 肌肉收缩
```

**编码转换:**

- `刺激强度/持续` ↔ `受体电位振幅/时间`
- `受体电位振幅` ↔ `AP频率`
- `AP频率` ↔ `递质释放量`

## 3.4 神经细胞分子多样性

**P:** 分子水平差异性

- `形态相似` ∧ `分子差异显著`
- `变异来源` := {离子通道组合, 递质类型, 受体类型, 基因表达}

**功能后果:**

- `不同阈值` := 离子通道差异 → 兴奋性差异
- `疾病选择性` := 分子差异 → 特定神经元易感性

**ex:** 疾病特异性

- `ALS + 脊灰` → 运动神经元
- `麻风 + 晚期梅毒` → 感觉神经元
- `帕金森` → 多巴胺神经元
- `重症肌无力` → 肌肉递质受体

## 3.5 反射到复杂行为

### 反射回路基本特征保留

**P:** 复杂行为基础

1. `感觉编码神经元` := 放电率 ∝ 刺激特征强度
2. `运动准备神经元` := 放电率变化 → 运动参数控制

### 复杂行为vs反射差异

**时间特征:**

- `反射` := 刺激后立即响应
- `认知` := 可延迟响应，条件依赖

**连接灵活性:**

- `反射` := 相对固定连接
- `认知` := 灵活连接通路 [@皮层联合区]

**P:** 皮层联合区特征

- `持续放电能力` := 数秒维持分级放电
- `功能` := 感觉-运动灵活联系调节
- `意义` := 行为摆脱瞬时反射限制

## 3.6 神经回路可塑性

### 可塑性假设

**历史发展:**

- `拉蒙·卡哈尔` [20世纪初]: 首次提出
- `科诺尔斯基` [1948]: 现代版阐述

**P:** 神经系统变化类型

1. `兴奋性变化` := 神经元对传入冲动响应能力
2. `可塑性变化` := 永久性功能转变能力

### 可塑性机制

**时间尺度:**

- `短期` := 几秒-几小时，功能性变化
- `长期` := 几天-终身，解剖结构改变

**结构变化:**

- `突触修剪` := 现有连接消除
- `突触生长` := 新连接建立
- `连接强度` := 突触有效性调节

**P:** 终身可塑性

- `关键期` ∧ `终身学习` → 个体独特性
- `机制一致性` := 简单回路可塑性原理 ↔ 复杂认知学习

## 3.7 要点总结

### 核心原理

1. **信号传导统一性:**

   ```
   N := {电信号(细胞内), 化学信号(细胞间)}
   共同特征 := {输入接收, 信号转换, 阈值机制, 轴突再生, 递质释放}
   ```

2. **支持系统:**

   ```
   G功能 := {绝缘(髓鞘), 化学环境, 血管耦合}
   ```

3. **多样性基础:**

   ```
   N差异 := {形态, 连接, 位置, 分子组成}
   分子差异 ⟹ 疾病选择性
   ```

4. **回路组织:**

   ```
   简单反射 := N功能回路基础
   ∀N ∈ 特定行为回路
   ```

5. **复杂性来源:**

   ```
   复杂行为 := {条件性, 灵活性, 时间延迟}
   关键 := 联合区长时程维持能力
   ```

6. **学习基础:**
   ```
   经验 → 连接修改
   可塑性机制 := 简单↔复杂行为学习基础
   ```

**Intuition:** 神经系统通过相对简单的基本原理(信号传导、回路组织、可塑性)支撑复杂多样的行为和认知功能。理解简单反射回路为分析复杂心理过程提供基础框架。
