# 第*60*章\_精神分裂症的思想障碍和意志障碍\_Summary

## 精神疾病的社会影响

`全球影响 := 残疾主要原因 + 自杀主要风险因素(80万例/年)`

- `合并症加重 := {糖尿病, 冠心病, 中风}等疾病预后恶化`
- `社会问题 := 无家可归 + 监狱人口大比例占比`
- `治疗局限 := 50年来精神药物疗效无显著进步`

## 精神分裂症核心特征

### P1: 三联症状群分类

```
认知症状: 最早出现
- 工作记忆&执行功能障碍
- 陈述性记忆缺陷
- 语言流畅性下降
- 面部表情情绪识别障碍
- 社会认知损伤

缺陷/阴性症状:
- 情绪响应迟钝
- 社交互动退缩
- 思想&言语内容贫乏
- 动力丧失

精神病症状:
- 幻觉(主要听觉性)
- 妄想(多种形式)
- 思维紊乱(联想松动)
```

### 克雷佩林早期认识 (20世纪初)

```
原始术语: "早发性痴呆" (强调认知丧失早期发生)
核心观点: 认知能力下降 := 精神分裂症显著特征
∵ 精神病症状 ∃ 多种精神疾病
∴ 认知障碍 := 疾病核心标志
```

## 思维障碍表现

### P2: 联想松动分类

```
语言结构异常范围: 走神 → 语无伦次

具体表现:
新词: 患者发明的内部意义单词
阻塞: 突然自发中断
叮当声: 基于语音而非意义的联想

联想松动例子:
"我本该拍电影,但不知结局...耶稣基督正写一本关于我的书"
"200万只骆驼...1000万辆出租车...圣诞老人正在反弹"
```

### 言语计划假说

```
潜在机制: ¬能形成"言语计划"引导连贯语言
机制相关: 与工作记忆缺陷等认知控制受损平行
∴ 联想松动 := 认知控制系统全面障碍的表现
```

## 疾病发展过程

### P3: 特征性时间进程

```
流行病学:
全球发病率: 0.25%-0.75% (区域差异小)
性别差异: 男:女 = 3:2, 男性发病更早
发病年龄: 青少年后期-20多岁中期

发病序列:
1. 前驱期: 认知&缺陷症状(精神病症状前数月-数年)
2. 超高风险状态/前驱症状期
3. 完整精神病发作
```

### 前驱期特征

```
认知功能下降:
- 可测量的认知衰退
- 社会孤立&多疑
- 学校/工作动力下降
- 轻微短暂幻觉等精神病性症状

预后变异性:
- 一小部分恢复
- 部分发展其他严重精神疾病
- 部分进展为完整精神分裂症

干预效果:
抗精神病药物: 无益&¬延缓发作
认知修复治疗: 显示延迟精神病发作潜力
```

## 精神病症状详解

### P4: 幻觉分类

```
定义: 无适当感觉刺激下的知觉体验
最常见: 听觉幻觉

听觉幻觉内容:
- 声音对话 (常贬损或欺凌)
- 噪音&音乐
- 命令性幻听 (自伤或伤人风险)

其他感觉模态: 可发生于任何感觉方式
```

### 妄想系统

```
定义: 无现实基础的坚定信念
特征: ¬能用文化解释 + ¬能通过论证/证据改变

妄想类型:
指称妄想: 世界充满只有患者理解的隐藏标志
偏执妄想: 被监视/跟踪/迫害信念
奇怪妄想: 思想插入/提取, 亲人被外星人替换等

现实扭曲: 严重认知-现实关系破坏
```

## 药物治疗反应

### P5: 抗精神病药物效果

```
有效症状: 精神病症状 (幻觉&妄想)
无效症状: {认知障碍, 缺陷症状}
跨疾病效果: 对多种神经精神疾病精神病症状有效

治疗局限性:
- 残留症状持续存在
- 复发常态
- 高度致残性认知损伤无有效治疗
```

### 发病机制启示

```
不同症状群 → 不同神经回路&脑区受影响
抗精神病药物作用: 疾病过程"下游"方面
∴ 对早期发病的认知&缺陷症状无益

治疗方向:
认知矫正心理疗法: 小但有前景的益处
需要: 针对发病机制上游的新治疗方法
```

## 整合性总结

`Intuition: 精神分裂症作为神经发育与认知控制的复合障碍`

疾病通过多维度破坏实现：

1. `认知维度 := 工作记忆&执行控制系统受损`
2. `情感维度 := 动机&社交互动能力缺失`
3. `现实检验维度 := 感知-认知-现实关系破坏`

展现了从神经发育异常到认知功能障碍、从个体痛苦到社会负担的全方位影响，构成了精神医学最具挑战性的疾病谱系。

`∴ 精神分裂症 := 人类高级认知功能脆弱性的极端表现，需要神经科学、遗传学与社会医学的综合攻克`
