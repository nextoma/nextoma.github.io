# 第*57*章\_周围神经和运动单元疾病 Summary

## 🔌 运动单元基本构架

```
运动单元组成：
脊髓前角细胞(α运动神经元) → 轴突 → 神经肌肉接头 → 肌纤维群
```

## 🧠 运动神经元疾病分类

### 上运动神经元病变

```
皮质脊髓束损伤特征：
肌力减退 + 肌张力增高 + 腱反射亢进 + 病理反射阳性
锥体束征: Babinski征，踝阵挛
```

### 下运动神经元病变

```
前角细胞/外周神经损伤：
肌力减退 + 肌萎缩 + 肌束颤动 + 腱反射减退/消失
神经源性改变: 去神经电位，纤颤电位
```

## 🔥 运动神经元病(ALS)

### 病理机制

```
选择性神经元死亡：
皮质运动神经元 + 脊髓前角细胞
进行性退行性变
```

**分子机制**：

- 兴奋性毒性: 谷氨酸过量
- 氧化应激: 自由基损伤
- 蛋白聚集: TDP-43, SOD1异常
- 轴突转运障碍: 细胞骨架异常

### 临床特征

```
症状进展：
局灶起病 → 逐渐扩散
上下运动神经元混合征
球麻痹: 吞咽、言语困难
呼吸肌麻痹: 最终致死原因
```

### 遗传形式

```
基因突变(5-10%家族性)：
SOD1: 超氧化物歧化酶1
TDP-43: TAR DNA结合蛋白43
FUS: RNA结合蛋白
C9orf72: 六核苷酸重复扩增(最常见)
```

## 🦵 周围神经病变

### 脱髓鞘性神经病

```
Guillain-Barré综合征(GBS)：
急性炎症性脱髓鞘性多发性神经根神经病
自身免疫攻击髓鞘
```

**临床特点**：

- 急性起病: 数天-数周
- 上行性麻痹: 远端→近端
- 感觉运动混合: 感觉症状轻微
- 脑脊液: 蛋白增高，细胞正常

### 轴索性神经病

```
糖尿病神经病：
代谢性轴索损伤
远端对称性感觉运动神经病
```

## 🔬 神经肌肉接头疾病

### 重症肌无力(MG)

```
病理机制：
乙酰胆碱受体抗体 → 受体破坏/阻断
神经肌肉传递阻滞
```

**临床特征**：

- 肌无力波动: 晨轻暮重
- 易疲劳性: 重复活动后加重
- 眼肌受累: 复视，上睑下垂
- 新斯的明试验阳性

### Lambert-Eaton综合征

```
突触前病变：
电压门控钙通道抗体
乙酰胆碱释放减少
肺癌相关副肿瘤综合征
```

## 💪 肌肉疾病

### 营养不良性肌病

```
遗传性肌肉病：
dystrophin基因缺陷(Duchenne型)
肌膜完整性破坏
进行性肌无力和萎缩
```

### 炎症性肌病

```
多发性肌炎：
自身免疫性肌肉炎症
近端肌群受累为主
血清肌酶(CK)显著升高
```

## ⚡ 电生理诊断

### 肌电图(EMG)

```
诊断价值：
神经源性: 纤颤电位，正尖波，运动单元电位改变
肌源性: 多相波增多，时限缩短
神经肌肉接头: 衰减现象
```

### 神经传导速度(NCV)

```
参数意义：
脱髓鞘: 传导速度减慢，波幅保持
轴索病: 波幅下降，速度相对保持
传导阻滞: 近端刺激波幅下降>50%
```

## 🧬 分子诊断进展

### 基因检测

```
分子诊断：
全外显子测序: 罕见病基因诊断
RNA测序: 剪接异常检测
重复序列检测: C9orf72等
```

### 生物标记物

```
液体活检：
神经丝蛋白: 轴索损伤标记
micro-RNA: 疾病特异性标记
炎症因子: 免疫活化指标
```

## 💊 治疗策略

### 症状缓解

```
对症治疗：
胆碱酯酶抑制剂: 重症肌无力
免疫抑制剂: 炎症性疾病
呼吸支持: 呼吸肌麻痹
```

### 疾病修饰治疗

```
新兴疗法：
基因治疗: SMA脊肌萎缩症
反义寡核苷酸: exon跳跃
干细胞治疗: 运动神经元替代
```

## 🔬 疾病模型

### 动物模型

```
模型系统：
SOD1转基因小鼠: ALS研究
mdx小鼠: 肌营养不良
EAE: 脱髓鞘疾病模型
```

### 体外模型

```
细胞培养：
iPSC分化运动神经元
3D肌肉组织培养
器官芯片技术
```

## ⚡ 关键要点

1. **解剖定位**：上下运动神经元损伤症状截然不同
2. **病理特异性**：脱髓鞘vs轴索病变电生理特征差异
3. **自身免疫**：多种疾病涉及免疫攻击神经肌肉系统
4. **遗传异质性**：相同表型可由多种基因变异引起
5. **进展模式**：疾病有特异性解剖扩散模式
6. **诊断技术**：电生理+影像+分子诊断综合评估
7. **治疗前景**：基因治疗等新技术带来希望

## 🎯 核心公式

```
运动单元功能 = 运动神经元数量 × 单个神经元支配肌纤维数
传导速度 = 轴突直径 × 髓鞘厚度 / 细胞内阻抗
疾病进展 = 遗传易感性 × 环境因子 × 年龄因子
```
