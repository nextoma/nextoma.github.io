# 第*19*章\_触觉 Summary

## 核心原理

**触觉系统定义**

- 触觉 := 皮肤机械刺激的感知
- 功能: [物体识别, 纹理辨别, 运动控制, 社交互动]
- P: 主动触觉 > 被动触觉 (探索性运动增强感知)
- P: 触觉 = 感觉输入 + 运动控制 (感觉运动整合)

## 19.1 外周机械感受器组织

### 受体类型与功能特化

**快适应受体(FA)**

- FA-I (Meissner): 低频振动(10-200Hz), 滑动检测
- FA-II (Pacinian): 高频振动(200-300Hz), 工具振动
- 功能: 运动检测, 握力调节
- ∵ 层状结构 → 机械滤波 ∴ 瞬态响应

**慢适应受体(SA)**

- SA-I (Merkel): 边缘, 曲率, 纹理细节
- SA-II (Ruffini): 皮肤拉伸, 手指位置
- 功能: 形状识别, 稳定抓握
- 持续放电 → 静态特征编码

### 感受野组织

**空间分辨率**

- 两点辨别阈: 指尖(1-2mm) < 手掌(10mm) < 背部(40mm)
- 受体密度: 指尖(241/cm²) > 手掌(58/cm²) > 前臂(14/cm²)
- P: 分辨率 ∝ 受体密度 ∝ 皮层表征面积

**感受野特性**

- SA-I: 小(2-3mm), 边界清晰, 热点区域
- FA-I: 小(3-5mm), 多个敏感区
- SA-II: 大(>10mm), 边界模糊, 方向选择性
- FA-II: 极大(整个手指/手), 无空间分辨

## 19.2 中枢触觉通路

### 背柱-内侧丘系通路

**第一级神经元**

- DRG神经元 → 同侧背柱上行
- 拓扑组织: 内侧(下肢) + 外侧(上肢)
- 薄束核: T6以下 → 下肢触觉
- 楔束核: T6以上 → 上肢触觉

**第二级神经元**

- 背柱核神经元 → 内侧丘系交叉
- 感受野精炼: 侧抑制 → 边界锐化
- 下行调制: 皮层 → 背柱核 (注意力调节)

**第三级神经元**

- VPL/VPM丘脑核 → S1皮层
- 保持拓扑映射 (体感小人)
- 门控功能: 睡眠/觉醒状态调节

### 三叉神经通路

- 面部触觉 → 三叉神经节 → 主感觉核/脊束核
- 交叉 → 三叉丘系 → VPM → S1面部区
- 特殊: 口腔精细辨别能力

## 19.3 体感皮层处理

### S1初级体感皮层

**四个区域组织**

- 3a区: 本体感觉(肌梭输入)
- 3b区: 皮肤触觉(SA-I, FA-I为主)
- 1区: 纹理特征
- 2区: 形状, 曲率

**柱状组织**

- 垂直柱: 同一感受野, 同一亚模态
- 分离柱: SA vs FA交替分布
- 水平连接: 相邻柱整合

**感受野层级**

- 3b → 1: 感受野扩大2-3倍
- 1 → 2: 复杂特征(边缘方向, 曲率)
- 会聚: 多个3b → 单个1/2神经元

### S2次级体感皮层

**功能特化**

- 双侧感受野 (同侧 + 对侧输入)
- 纹理/形状整合
- 注意力依赖性响应
- 工作记忆: 触觉信息保持

**输出连接**

- → 顶叶后部: 空间定位
- → 运动皮层: 感觉引导运动
- → 岛叶: 情感触觉
- → 海马: 触觉记忆

## 19.4 高级触觉处理

### 特征检测层级

**简单特征** (S1)

- 边缘检测: 方向选择性神经元
- 振动频率: FA特征检测器
- 压力梯度: SA响应模式

**复杂特征** (S2/顶叶)

- 表面纹理: 粗糙度计算
- 物体形状: 3D曲率整合
- 材质属性: 硬度/黏性/温度综合

**抽象表征** (顶叶/颞叶)

- 物体识别: 形状+纹理+大小
- 类别化: 工具/食物/材料
- 跨模态整合: 触觉-视觉匹配

### 触觉物体识别

**立体认知(Stereognosis)**

- 定义 := 通过触摸识别3D物体
- 需要: 感觉完整 + 运动控制 + 认知整合
- 探索程序: [侧向运动(纹理), 压力(硬度), 包围(体积), 轮廓跟踪(形状)]

**神经基础**

- 腹侧通路: "什么" → 物体识别
- 背侧通路: "如何" → 抓握控制
- AIP区: 抓握前视觉-触觉转换
- ex: 失认症 → 感觉正常但识别障碍

## 19.5 主动触觉与感觉运动整合

### 探索性运动

**触觉探索程序**

1. 侧向运动 → 纹理
2. 压力 → 硬度
3. 静态接触 → 温度
4. 无支撑托持 → 重量
5. 包围 → 全局形状/体积
6. 轮廓跟踪 → 精确形状
7. 部分运动 → 部件运动

**神经控制**

- 运动皮层 → 探索运动计划
- 小脑 → 运动-感觉预测
- 顶叶 → 空间-时间整合

### 触觉反馈控制

**握力调节**

- 滑动检测(FA-I) → 握力增加
- 预测控制: 预期负载 → 握力调整
- 反应时间: 触觉(65ms) < 视觉(100ms)

**精细操作**

- SA-I: 稳定抓握反馈
- FA-I: 滑动/不稳定检测
- FA-II: 工具介导触觉
- 损伤后果: 触觉麻痹 → 握力过度/不足

## 19.6 触觉感知特性

### 触觉错觉

**亚里士多德错觉**

- 交叉手指 → 单一物体感知为两个
- 机制: 异常空间映射 → 解释错误

**兔子错觉**

- 快速序列刺激 → 感知跳跃
- 机制: 时空整合 → 填充效应

**橡胶手错觉**

- 视觉-触觉同步 → 身体拥有感
- 机制: 多感觉整合 → 身体图式更新

### 触觉注意力

**空间注意力**

- 增强: 注意位置神经元活动↑
- 抑制: 非注意位置响应↓
- 感受野: 向注意焦点收缩

**特征注意力**

- 振动vs压力选择性增强
- S2/顶叶: 注意力调制最强
- 行为相关性 → 响应增益

## 19.7 发育与可塑性

### 关键期发育

**触觉地图形成**

- 胚胎期: 初级投射形成
- 出生后: 感受野精炼
- 关键期: 皮层柱形成(啮齿类P0-P7)

**经验依赖精炼**

- 感觉剥夺 → 皮层萎缩
- 丰富环境 → 突触密度↑
- 触须修剪 → 桶状皮层重组

### 成年可塑性

**使用依赖扩展**

- 盲文阅读者: 指尖表征扩大
- 音乐家: 手指皮层分离度↑
- 机制: 侧抑制减弱, 突触强化

**损伤后重组**

- 截肢: 邻近区域侵占
- 神经修复: 错误投射 → 感觉异常
- 幻肢: 皮层表征持续
- @ 第50章: 损伤与修复

## 19.8 临床相关

### 感觉检查

**基本测试**

- 轻触: 棉签(FA/SA阈值)
- 两点辨别: 空间分辨率
- 振动: 音叉(FA-II功能)
- 立体认知: 物体识别

**定量测试**

- Von Frey纤维丝: 触觉阈值
- 光栅方向: 空间敏锐度
- 振动阈值: 频率特异性

### 病理状态

**周围神经病**

- 糖尿病: 长度依赖性, 手套袜套分布
- 腕管综合征: 正中神经压迫
- 表现: 阈值↑, 两点辨别↓

**中枢病变**

- 脊髓: 背柱损伤 → 位置觉/振动觉丧失
- 脑干: 内侧丘系损伤 → 对侧触觉缺失
- 皮层: S1损伤 → 触觉失认

### 治疗应用

**感觉替代**

- 触觉-视觉: 盲人导航设备
- 触觉-听觉: 人工耳蜗触觉反馈
- 原理: 跨模态可塑性

**神经修复**

- 触觉生物反馈训练
- 虚拟现实触觉康复
- 脑机接口: 触觉反馈恢复

## 19.9 比较生物学

### 特化触觉系统

**星鼻鼹鼠**

- 22个附器, 25000个Eimer器官
- 最高触觉分辨率
- 功能: 水下猎物检测

**大象鼻**

- 150000块肌肉
- 精细操作能力
- 触觉-运动高度整合

**蜘蛛腿**

- 振动感受器
- 网振动 → 猎物定位
- 频率分析能力

## 关键洞察

1. **受体特化**: 四类机械感受器 → 互补功能 → 完整触觉体验
2. **层级处理**: S1(基本特征) → S2(整合) → 顶叶(认知) 逐级抽象
3. **感觉运动整合**: 主动探索 ↔ 感觉反馈 形成闭环控制
4. **可塑性**: 使用依赖 + 损伤诱导 → 皮层重组 (终身保持)
5. **临床意义**: 精确定位 + 定量评估 → 诊断/康复策略
