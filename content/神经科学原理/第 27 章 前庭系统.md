# 第 28 章 听觉处理的中枢神经系统 Summary

## 1. 声音定位线索
**水平面定位(方位角)**
- 双耳时间差(ITD): 最大~600μs(人类), ~35μs(小鸟)
  - 低频(<3kHz): 锁相编码 → 耳间相位差
  - 分辨率: ~1°(10μs时间差)
- 双耳强度差(ILD): 高频声音头影效应
  - 频率依赖: >2kHz显著
  - 头部大小相关

**垂直面定位与单耳线索**
- 频谱滤波: 耳廓/头部/肩部相互作用
  - 频率>8kHz: 深谱凹陷
  - 位置特异性: 与头相关传递函数(HRTF)
- 前后消歧: 频谱线索差异

## 2. 耳蜗核并行处理
**解剖组织**
- 腹侧耳蜗核(VCN): 前腹侧 + 后腹侧
- 背侧耳蜗核(DCN): 分层结构
- 音调拓扑: 低频腹侧 → 高频背侧

**腹侧核主要细胞类型**
1. 毛细胞(Bushy cells)
   - 球状: 低频 → 内侧上橄榄(ITD)
   - 小球状: 高频 → 外侧上橄榄(ILD)
   - 特征: 低电压激活K+通道 → 精确时间编码

2. 星形细胞(Stellate cells)
   - 功能: 频谱峰谷编码
   - 投射: 广泛(DCN, VNLL, IC)
   - 特征: 持续放电模式

3. 章鱼细胞(Octopus cells)
   - 功能: 声音起始检测
   - 投射: 对侧橄榄旁核 → VNLL
   - 特征: 宽频调谐, 同步检测

**背侧核特征**
- 主细胞: 梭形细胞
- 双层输入:
  - 分子层: 平行纤维(多感觉)
  - 深层: 听觉输入
- 功能: 自身运动声音抑制

## 3. 上橄榄复合体双耳处理
**内侧上橄榄(MSO) - ITD检测**
- 原理: Jeffress延迟线模型
- 组织: 双侧树突 + 系统延迟线
- 输出: 对侧半空间位置图
- 机制: 同时检测器 + 相位锁定

**外侧上橄榄(LSO) - ILD检测**
- 回路: 同侧兴奋 vs 对侧抑制
  - 兴奋: VCN小球状/星形细胞
  - 抑制: VCN球状 → MNTB → LSO
- 特殊突触: 花萼突触(Calyx of Held)
- 输出: 同侧优势编码

**橄榄耳蜗反馈**
- 内侧系统: → 双侧外毛细胞(增益控制)
- 外侧系统: → 同侧内毛细胞传入(平衡)

## 4. 下丘整合中心
**汇聚特性**
- 所有上行通路终点
- 音调层状组织
- 中央核: 频率板层(1/3倍频程间隔)

**功能整合**
- 双耳线索整合: ITD + ILD + 频谱
- 优先效应: 抑制回声
- 空间选择性增强

**输出通路**
- 上丘: 空间地图 → 定向反射
- 丘脑(MGN): → 听觉皮层

## 5. 听觉皮层组织
**初级听觉皮层(A1)**
- 音调拓扑: 嘴侧(低频) → 尾侧(高频)
- 等频轮廓: 垂直于频率梯度
- 参数映射: 带宽, 延迟, 强度, 调制

**高级区域组织**
- 核心区: 3-4个音调拓扑区
- 带状区: 7-10个次级区域
- 镜像对称: 边界频率反转

**刺激选择性递增**
P: 听神经(频率) → 耳蜗核(时间/频谱) → IC(空间) → A1(复杂特征)
- 持续vs起始激活
- 偏好刺激缩小
- 多维感受野

## 6. 皮层处理流
**双流模型**
- 腹侧流("什么"): 前腹侧 → 颞叶 → 音频识别
- 背侧流("哪里"): 后背侧 → 顶叶 → 空间定位

**皮层反馈**
- 规模: 皮层→丘脑纤维 = 10×丘脑→皮层
- 目标: MGN, IC, OCN, DCN
- 功能: 自上而下调制

## 7. 复杂声音表征
**时间编码转换**
- 同步编码: 长间隔(<100Hz) → 锁相
- 异步编码: 短间隔 → 发放率
- 互补表征策略

**音调处理**
- 音调选择神经元: A1/R边界低频区
- 谐波模板神经元: 谐波结构检测
- 缺失基频感知: 谐波→音调计算

**特化系统(蝙蝠)**
- FM-FM区: 延迟→距离(0.4-18ms)
- CF-CF区: 多普勒→速度(-2到+9m/s)
- DSCF区: 声学凹(60-62kHz扩展)

## 8. 发声反馈处理
**发声诱导抑制**
- 时程: 发声前200-300ms开始
- 来源: 前运动皮层 → 听觉皮层
- 功能: 自身声音屏蔽

**反馈监测**
- 扰动敏感性: 频移→抑制减弱/兴奋
- 错误检测: 预期vs实际比较
- 语音学习: 反馈依赖可塑性

## 核心概念总结
听觉中枢 := 并行特征提取 + 双耳整合 + 递增选择性

关键创新:
1. 延迟线计算 → 空间从时间推断
2. 花萼突触 → 微秒级精度
3. 皮层下计算 → 空间地图形成
4. 双流分离 → 识别vs定位
5. 发声-听觉整合 → 反馈控制
