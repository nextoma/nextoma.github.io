# 第*58*章\_癫痫发作和癫痫 Summary

## ⚡ 癫痫核心概念

```
癫痫定义：
脑神经元异常过度同步放电 → 反复发作性症状
慢性神经系统疾病，发作间歇期大脑功能正常
```

## 🧠 癫痫发作机制

### 神经元超兴奋性

```
兴奋-抑制失衡：
兴奋性增强: 谷氨酸能传递↑
抑制性减弱: GABA能传递↓
同步化放电: 神经元群体协同激活
```

**分子机制**：

- 离子通道异常: Na⁺, K⁺, Ca²⁺通道
- 神经递质失衡: Glu/GABA比值异常
- 膜电位不稳定: 阈值降低

### 癫痫灶形成

```
致痫灶特征：
局部神经元损伤 → 异常放电起源
传播途径: 皮层-皮层，皮层-皮层下
全面化机制: 双侧大脑半球同时受累
```

## 📊 癫痫发作分类

### 部分性发作

```
局灶性发作：
简单部分性: 意识清楚，局限症状
复杂部分性: 意识障碍，自动症
继发全面化: 部分性→全面性扩散
```

**症状类型**：

- 运动性: 局部抽搐，Jackson癫痫
- 感觉性: 感觉异常，幻觉
- 精神性: 情绪体验，既视感

### 全面性发作

```
双侧同时发作：
强直阵挛(大发作): 意识丧失，全身抽搐
失神(小发作): 短暂意识中断
肌阵挛: 快速肌肉收缩
强直/阵挛: 单一运动模式
```

## 🧬 癫痫综合征

### 特发性全面性癫痫

```
遗传性癫痫：
儿童失神癫痫: 3Hz棘慢波
青少年肌阵挛癫痫: 晨起肌阵挛
全面强直阵挛癫痫: 觉醒相关
```

### 症状性局灶性癫痫

```
继发性癫痫：
颞叶癫痫: 海马硬化，复杂部分性发作
额叶癫痫: 运动症状突出，夜间发作
枕叶癫痫: 视觉先兆
```

### 癫痫性脑病

```
发育性癫痫：
West综合征: 婴儿痉挛，高幅失律
Lennox-Gastaut综合征: 多发作类型，智力障碍
Dravet综合征: 热性惊厥相关，SCN1A基因
```

## 🔬 病理生理机制

### 离子通道病

```
遗传性癫痫基础：
SCN1A: 钠通道α1亚基，Dravet综合征
KCNQ2/3: 钾通道，新生儿癫痫
CACNA1A: 钙通道，失神癫痫
```

### 神经递质异常

```
递质失衡：
GABA系统: 合成酶缺陷，受体异常
谷氨酸系统: NMDA受体过度激活
多巴胺系统: 基底核调节异常
```

### 网络异常

```
回路病理：
皮层-丘脑回路: 失神癫痫
海马-杏仁核: 颞叶癫痫
额叶-基底核: 额叶癫痫
```

## ⚡ 脑电图特征

### 发作间期异常

```
EEG异常波形：
棘波: 尖锐快波，<80ms
尖波: 中等快波，80-200ms
棘慢复合波: 棘波后慢波
多棘慢复合波: 多个棘波
```

### 发作期EEG

```
发作模式：
局灶起始: 单侧局限性快活动
继发全面化: 双侧对称性扩散
全面性起始: 双侧同步3Hz棘慢波
```

## 💊 抗癫痫药物

### 传统抗癫痫药

```
经典药物：
苯妥英: Na⁺通道阻滞
卡马西平: Na⁺通道阻滞，部分性癫痫首选
丙戊酸: 多机制，广谱抗癫痫药
苯巴比妥: GABA受体激动
```

### 新型抗癫痫药

```
新药特点：
拉莫三嗪: Na⁺通道阻滞，较少副作用
奥卡西平: 卡马西平改良版
左乙拉西坦: SV2A蛋白结合，广谱
托吡酯: 多机制，减重副作用
```

### 作用机制分类

```
药理机制：
Na⁺通道阻滞: 苯妥英，卡马西平
Ca²⁺通道阻滞: 乙琥胺，加巴喷丁
GABA增强: 苯巴比妥，氯硝西泮
谷氨酸拮抗: 托吡酯，非氨酯
```

## 🏥 外科治疗

### 手术适应症

```
药物难治性癫痫：
单药治疗失败 + 联合用药无效
明确癫痫灶定位
切除不影响重要功能
```

### 手术类型

```
手术方式：
切除性: 颞叶切除，病灶切除
离断性: 胼胝体切开，多软脑膜下横切
调控性: 迷走神经刺激，脑深部刺激
```

### 术前评估

```
定位检查：
视频脑电图: 发作症状学与EEG对应
影像学: MRI结构异常，PET代谢异常
有创监测: 颅内电极，精确定位
```

## 🔬 新兴治疗

### 神经调控

```
调控技术：
迷走神经刺激(VNS): 减少发作频率
脑深部刺激(DBS): 丘脑前核刺激
响应性神经刺激(RNS): 闭环反馈控制
```

### 基因治疗

```
精准治疗：
反义寡核苷酸: 特定基因靶向
基因替代: 病毒载体递送
CRISPR编辑: 致病基因修正
```

## 🧠 特殊人群癫痫

### 儿童癫痫

```
发育特点：
年龄依赖性: 不同年龄特征性综合征
发育影响: 认知功能损害风险
预后差异: 良性vs灾难性癫痫
```

### 女性癫痫

```
激素影响：
月经性癫痫: 雌激素/孕激素波动
妊娠期癫痫: 药代动力学改变
致畸风险: 抗癫痫药致畸性
```

## ⚡ 关键要点

1. **兴奋抑制失衡**：GABA能抑制减弱导致神经元过度兴奋
2. **发作传播模式**：局灶起源可继发全面化扩散
3. **EEG诊断价值**：棘波等异常放电是诊断金标准
4. **药物机制多样**：离子通道阻滞和神经递质调节
5. **外科治疗选择**：药物难治性癫痫可考虑手术
6. **遗传基础重要**：多种基因变异导致不同癫痫综合征
7. **个体化治疗**：根据发作类型和患者特点选择治疗

## 🎯 核心公式

```
发作阈值 = 兴奋性驱动力 / 抑制性制动力
药物疗效 = 血药浓度 × 受体亲和力 × 靶点表达量
手术成功率 = 致痫灶完整切除 × 功能保护程度
```
