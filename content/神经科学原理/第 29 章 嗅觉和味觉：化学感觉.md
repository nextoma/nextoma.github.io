# 第*30*章\_感觉运动控制原理\_Summary

## P: 感觉运动控制的核心原理

**Notion:**

- 运动系统 := {640块骨骼肌} @ CNS控制
- 感觉处理 := 外部世界 → 内部表示
- 运动处理 := 内部表示 → 运动输出
- 感觉运动循环 := 感觉输入 ↔ 运动输出

### 30.1 运动控制挑战 := 神经系统面临的6大难题

**P: 运动控制复杂性源泉**

**核心挑战:**

1. **不确定性** `U_sens ∧ U_motor`
   - 感觉不确定性: U_sens := {球轨迹, 接触点} @ 网球例子
   - 运动不确定性: U_motor := 不同回球策略的成功概率
   - 解决策略: 预测 ∧ 运动规划 → ↓不确定性

2. **高自由度问题** `DOF_problem`
   - 系统复杂度: {600肌肉, 200关节} → 控制困难
   - 解决方案: 肌肉协同作用(synergies) → ↓自由度
   - 控制简化: 肌肉群控制 >> 单肌肉控制

3. **噪声干扰** `N(σ²)`
   - 噪声分布: ∀阶段 ∈ {感觉处理, 规划, 运动输出}
   - 噪声特性: N ∝ 运动强度 (信号依赖性)
   - 速度-精度权衡: v ↔ accuracy (反比关系)
   - 运动规划目标: min(N对任务成功的负面影响)

4. **时间延迟** `Δt_total`
   - 延迟构成: Δt_total = Δt_receptor + Δt_transmission + Δt_muscle
   - 总延迟: ~100ms (视觉>本体感觉)
   - 控制困境: 基于过时信息控制 → 潜在不稳定
   - 解决策略: 预测未来状态

5. **系统变化性** `dS/dt`
   - 短期变化: {疲劳, 环境条件} @ 比赛时间尺度
   - 长期变化: {四肢长度, 体重} @ 发育时间尺度
   - 适应需求: ∀t, 系统参数校准 → 运动学习重要性

6. **复杂动力学** `F_interaction`
   - 身体部分间相互作用: ∀部位运动 → ∀其他部位扭矩
   - ex: 举拍 → 需要踝屈肌预激活 @ 维持平衡
   - 协调需求: 精细运动协调 @ 多部位系统

### 30.2 运动控制类型分类

**P: 运动控制的三重性质**

**控制类型矩阵:**

```
控制类型 := {自主运动, 有节奏运动, 反射运动}
肌肉群 := 可被∀控制类型使用
```

**运动分类:**

1. **自主运动** `M_voluntary`
   - 定义: 意识控制下的运动
   - 特征: 有意决策 → 运动执行
   - 限制: 瞬间控制 ¬需要意识过程

2. **有节奏运动** `M_rhythmic`
   - 时空组织: 脊髓/脑干回路自主控制
   - 可自主调节但基础模式自动化
   - ex: 呼吸, 步行模式

3. **反射运动** `M_reflex`
   - 定义: 脊髓/脑干回路 @ 特定刺激 → 模式化响应
   - 优势: 速度快
   - 劣势: 灵活性低
   - 延迟: < 自主响应延迟

**Intuition:** 运动看似自动发生，意识决策≠肌肉选择

### 30.3 运动命令生成的层次结构

**P: 分层控制简化复杂性**

**控制层次 := 高层(全局目标) → 低层(实现细节)**

**层次结构:**

1. **肌肉被动特性层** (最低层)
   - 肌肉 ≈ {弹簧 + 阻尼器}
   - 弹簧特性: 拉伸 → ↑张力, 缩短 → ↓张力
   - 阻尼特性: 拉伸率 → 张力变化
   - 优势: 几乎0延迟的稳定性

2. **反射控制层**
   - 感觉输入 → 运动输出 (无高层干预)
   - 单突触反射: 25ms延迟 (肌梭 → 运动神经元)
   - 多突触反射: ~70ms延迟 (涉及运动皮层)
   - 适应性: 单突触 < 多突触反射
   - 增益缩放: 反射强度 ∝ 肌肉张力

3. **自主控制层** (最高层)
   - 有意识控制 → 可无刺激产生
   - 扰动响应延迟: ~110ms
   - 灵活性最高但速度最慢

**层次权衡: 响应速度 ↔ 处理复杂性**

### 30.4 运动控制的前馈与反馈机制

**P: 控制策略的二元性**

#### 30.4.1 前馈控制 (开环控制)

**开环控制 := 运动命令生成 ¬依赖感觉反馈**

**核心组件:**

- **反向模型** `M_inverse`
  - 功能: 计算期望运动 → 所需运动命令
  - 需求: 逆动力学知识
  - 动力学关系: 运动命令 → 身体运动

**优势:**

- 无延迟问题 (不依赖感觉反馈)
- 适用于快速运动 (ex: 眼跳30ms)

**劣势:**

- 误差累积: 规划/执行误差 → 无纠正 → 累积放大
- 复杂系统难以建立精确反向模型

**ex: 前庭-眼动反射**

- 前庭迷路感知头部旋转 → 3突触回路 → 眼动
- 纯开环: 运动中不使用视觉反馈
- 成功因素: {眼动力学简单, 直接头部旋转感知, 外部干扰小}

#### 30.4.2 反馈控制 (闭环控制)

**闭环控制 := 感觉运动回路完整, 误差纠正**

**控制类型:**

1. **阈值控制** (简单但效率低)
   - 误差 > 阈值 → 固定响应
   - ex: 恒温器系统

2. **比例控制** (更优)
   - 校正运动命令 ∝ 位置误差
   - 增益 := 每单位误差的命令调整量

**优势:**

- 噪声鲁棒性: 持续纠正 → 抵抗噪声 ∧ 环境扰动
- 自适应性: 实时响应偏差

**劣势:**

- 延迟敏感性: 反馈延迟 → 系统振荡 → 潜在不稳定
- 延迟问题: 系统响应已不存在的误差 → 错误方向纠正

**稳定性条件:**

- 无延迟: ↑增益 → ↑跟踪保真度
- 有延迟: ↑增益 + ↑延迟 → 振荡 → 不稳定

## 内部模型理论 (Box 30.1)

**P: CNS的世界预测机制**

**内部模型 := CNS中实例化的外部世界模型**

**历史起源:** Kenneth Craik (1943) "解释的本质"

> "小规模模型" @ 外部现实 ∧ 自身行动 → 尝试替代方案 → 预测最佳策略

**双重功能:**

1. **预测功能:** 运动命令 → 预期感觉结果 (前向模型)
2. **控制功能:** 期望感觉结果 → 运动命令 (反向模型)

**Notion:** 预测 ∧ 控制 = 一枚硬币的两面

**基本问题:**

1. 如何生成运动命令控制身体？ (控制)
2. 如何预测运动命令后果？ (预测)

**熟练运动表现 := 控制 ∧ 预测同时进行**

## 总体架构原理

**P: 分布式神经处理**

**运动控制网络:**

```
大脑皮层区域 ∧ 皮层下区域{基底神经节, 小脑} ∧ 脑干脊髓网络
→ 网络协作 ≠ 独立功能模块
```

**损伤影响:** 一个组件损伤 → ∀其他组件功能受影响

**控制原理总结:**

- 层次化: 简化控制复杂性
- 并行处理: 多层级同时工作
- 自适应性: 运动学习 @ 系统变化
- 预测性: 内部模型补偿延迟
- 鲁棒性: 多重机制应对不确定性

**∴ 感觉运动控制 := 复杂分布式网络的涌现属性**
