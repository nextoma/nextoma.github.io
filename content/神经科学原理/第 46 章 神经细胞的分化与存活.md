# 第*47*章\_轴突的生长和引导 Summary

## 🌱 轴突发育核心机制

```
轴突生长导航系统：
生长锥感知 → 环境线索解读 → 方向决策 → 路径延伸
```

## 🔬 神经元极性建立

### 47.1 轴突-树突分化

```
早期极性建立：
对称神经元 → 不对称信号 → 轴突/树突命运分离
```

**分子开关**：

- 轴突标识: Tau蛋白, MAP1B
- 树突标识: MAP2, 树突棘蛋白
- 极性调节: Par复合物, PI3K-Akt通路

### 47.2 树突形成调节

```
树突发育控制：
内在因子: 转录程序(CREB, MEF2)
外在因子: 活动依赖性塑性, 神经营养因子
```

**关键过程**：

- 分支模式: 内在遗传程序
- 精细调节: 突触活动反馈

## 🎯 生长锥导航系统

### 47.3 生长锥结构功能

```
感知-运动整合器：
丝状伪足(filopodia) → 环境探测
片状伪足(lamellipodia) → 运动推进
中心区(central domain) → 信号整合
```

**分子机械**：

- 细胞骨架: 肌动蛋白+微管动力学
- 受体系统: 引导受体+信号转导
- 运动装置: 分子马达+聚合/解聚

### 47.4 分子引导线索

```
四类引导信号：
吸引性长距离: Netrin → DCC/UNC-5受体
排斥性长距离: Slit → Robo受体
吸引性短距离: Ephrin → Eph受体
排斥性短距离: Semaphorin → Plexin受体
```

**整合原理**：

- 多信号组合编码
- 时空动态变化
- 剂量效应关系

## 📍 空间导航策略

### 47.5 视网膜轴突模型

```
分步导航程序：
步骤1: 离开视网膜 → 视神经盘集合
步骤2: 视交叉决策 → 同侧vs对侧选择
步骤3: 靶标识别 → 上丘精确定位
```

**地形投射**：

```
视网膜拓扑 → 上丘拓扑
鼻颞轴 ↔ 后前轴 (EphA-EphrinA)
背腹轴 ↔ 内外轴 (EphB-EphrinB)
```

## 🗺️ 引导机制分类

### 长距离引导

```
扩散性梯度：
Netrin梯度: 腹侧吸引 (脊髓交叉)
Slit梯度: 中线排斥 (双侧分离)
```

### 短距离引导

```
接触依赖：
Ephrin-Eph: 边界形成+地形投射
Semaphorin-Plexin: 局部排斥+路径选择
```

### 基底引导

```
细胞外基质：
Laminin: 粘附+生长促进
Collagen IV: 基底膜结构
Fibronectin: 迁移路径标记
```

## ⚖️ 决策整合机制

### 信号整合模型

```
多信号求和：
∑(吸引力 - 排斥力) = 净向量
         ↓
阈值比较 → 转向决策
```

**动态调节**：

- 受体敏感性变化
- 信号强度时间窗
- 内在状态修饰

### 活动依赖性调节

```
电活动调制：
Ca²⁺流入 → cAMP/cGMP → PKA/PKG → 受体敏感性
```

## 🧠 回路组装原理

### 层次化组装

```
组装层次：
粗糙连接 → 精细投射 → 拓扑地图 → 功能柱状
```

**误差纠正**：

- 初期超量连接
- 活动依赖性修剪
- 竞争性精化

### 双侧协调

```
中线组织中心：

Roof plate: BMP源，背侧定向
```

## 📊 定量关系

### 空间精度

```
投射精度: 微米级 (视觉系统)
引导距离: 毫米-厘米级
时间窗口: 小时-天级
成功率: >90% (正确靶标)
```

## ⚡ 关键要点

1. **极性建立**：早期分子不对称性决定轴突身份
2. **感知导航**：生长锥实时整合多维环境信息
3. **分步引导**：复杂路径分解为简单决策序列
4. **梯度解码**：浓度-位置转换实现精确制导
5. **地形投射**：分子标签系统建立空间对应关系
6. **质量控制**：多层纠错机制确保连接准确性
7. **活动调节**：电信号实时微调引导敏感性

## 🎯 核心公式

```
引导决策 = f(∑信号强度i × 权重i × 敏感性i)
投射精度 = 分子梯度斜率 × 受体敏感范围⁻¹
连接成功率 = 引导精度 × 时间窗口 × 靶标密度
```
