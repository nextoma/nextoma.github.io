# 第*14*章\_突触传递和神经元兴奋性的调节：第二信使 摘要

## 核心概念

**代谢型vs离子型受体**

- 离子型：直接门控，快速（ms），局部作用
- 代谢型：间接门控，缓慢（100ms-min），扩散作用
- P: 离子型介导快速行为，代谢型调节行为状态和学习

**代谢型受体家族**

1. G蛋白偶联受体（GPCRs）：7次跨膜螺旋
2. 受体酪氨酸激酶（RTKs）：单跨膜，二聚化激活

## cAMP通路

**信号级联**

- 递质 → GPCR → Gs蛋白（αs激活）→ 腺苷酸环化酶
- ATP → cAMP → PKA激活
- PKA := 2调节(R) + 2催化(C)亚基
- 4cAMP结合R亚基 → C亚基释放 → 底物磷酸化

**调控机制**

- 磷酸二酯酶：cAMP → AMP（失活）
- 磷蛋白磷酸酶：去磷酸化
- 抑制剂-1：PKA磷酸化后抑制磷酸酶
- A激酶锚定蛋白（AKAPs）：定位PKA至底物附近

## G蛋白信号

**G蛋白组成**

- ~20种α亚基，5种β亚基，12种γ亚基
- Gs：刺激腺苷酸环化酶（↑cAMP）
- Gi：抑制腺苷酸环化酶（↓cAMP）
- Gq/11：激活磷脂酶C（PLC）
- Go：大脑高表达，靶标多样

**激活循环**

1. GDP结合（静息）→ GTP交换（激活）
2. α亚基与βγ复合物分离
3. GTPase活性 → GTP水解 → GDP（失活）
4. 亚基重新结合

## 磷脂信号通路

**磷脂酶C通路**

- PIP₂水解 → IP₃ + DAG
- IP₃ → 内质网Ca²⁺释放
- DAG → PKC激活（膜定位）
- Ca²⁺ + 钙调蛋白 → CaMKII激活

**磷脂酶A2通路**

- 磷脂水解 → 花生四烯酸释放
- 环氧合酶 → 前列腺素、血栓素
- 脂氧合酶 → 白三烯等
- 细胞色素P450 → 氧化产物

## 受体酪氨酸激酶

**激活机制**

- 配体结合 → 二聚化 → 自磷酸化
- 酪氨酸残基磷酸化 → 衔接蛋白结合

**下游信号**

- Ras/MAPK级联 → 基因转录
- PLCγ激活 → IP₃/DAG产生
- PI3K → PIP₃ → Akt激活（生存信号）

**配体类型**

- 生长因子：EGF、FGF、NGF、BDNF
- 胰岛素
- ex: 长期记忆形成需要MAPK → 转录调控

## 跨细胞信使

**内源性大麻素**

- 2-AG、花生四烯酸乙醇胺（anandamide）
- 突触后产生 → 逆行至突触前
- CB1受体（突触前）→ 抑制递质释放
- P: 突触后控制突触前功能

**一氧化氮（NO）**

- 自由基，高反应性，短寿命
- NO → 可溶性鸟苷酸环化酶 → cGMP
- cGMP → PKG激活 ∨ 直接开启CNG通道
- ex: 小脑平行纤维 → NO → 浦肯野细胞LTD

**花生酸类**

- 20碳脂肪酸代谢物
- 前列腺素、血栓素、白三烯
- 作用于GPCRs ∨ 直接调节通道

## 通道调节机制

**直接调节**

- G蛋白βγ亚基 → GIRK通道开放
- βγ → 电压门控Ca²⁺通道抑制
- ex: 心脏M2受体 → GIRK → 心率减慢

**间接调节**

- PKA/PKC磷酸化 → 通道特性改变
- PIP₂耗竭 → M型K⁺通道关闭
- CaMKII自磷酸化 → 持续激活

**M型K⁺通道调节**

- 毒蕈碱受体 → PLC → ↓PIP₂ → M通道关闭
- 结果：↑兴奋性，↓尖峰频率适应
- P: 小去极化 → 大兴奋性改变

## 功能比较

| 特性     | 离子型   | 代谢型      |
| -------- | -------- | ----------- |
| 速度     | 快（ms） | 慢（s-min） |
| 作用范围 | 局部     | 扩散        |
| 持续时间 | 短暂     | 延长        |
| 调节类型 | 开/关    | 增益调节    |
| 可塑性   | 有限     | 广泛        |

## 突触调节位点

**突触前调节**

- Ca²⁺通道调节 → 递质释放量
- K⁺通道调节 → AP持续时间 → Ca²⁺内流
- 释放机制直接调节

**突触后调节**

- 离子型受体磷酸化 → 电导改变
- 受体运输/定位调节
- 内在兴奋性调节

## 持续激活机制

**蛋白激酶持续活性**

- CaMKII：自磷酸化 → Ca²⁺非依赖形式
- PKA：R亚基降解 → 游离C亚基
- PKC：蛋白水解切割 → 组成性活性

**磷酸酶抑制**

- 抑制剂-1：PKA磷酸化后活性
- 结果：延长磷酸化状态

## 临床相关性

**药物靶点**

- β-肾上腺素能受体：心血管药物
- 毒蕈碱受体：自主神经药物
- PDE抑制剂：西地那非等
- 大麻素系统：成瘾、疼痛

**疾病机制**

- GPCR突变 → 内分泌疾病
- RTK异常 → 癌症、发育障碍
- 第二信使失调 → 神经精神疾病

## 记忆与可塑性

**短期调节**

- 通道磷酸化 → 突触强度改变
- 内在兴奋性调节

**长期改变**

- MAPK → CREB → 基因转录
- 新蛋白合成 → 结构改变
- P: 短期记忆 → 长期记忆转换
