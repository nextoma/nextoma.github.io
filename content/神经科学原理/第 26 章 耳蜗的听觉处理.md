# 第 27 章 前庭系统 Summary

## 1. 前庭系统概述
**功能定义**
前庭系统 := 惯性制导系统 → 检测空间运动 + 重力方向感知
- 进化保守性: 5亿年脊椎动物史
- 双重功能: 角加速度检测(半规管) + 线性加速度检测(耳石器官)
- 信号整合: 前庭 + 视觉 + 本体感受 + 运动命令 → 空间定向

**解剖基础**
- 骨迷路: 颞骨岩部内中空结构
- 膜迷路: 内含前庭/听觉传感器
- 液体系统: 内淋巴(K+富集,150mM) ↔ 外淋巴(Na+富集,150mM)
- 神经通路: 斯卡帕神经节 → 前庭神经(CN VIII) → 前庭核

## 2. 前庭受体器官
**半规管系统(3个)**
- 结构组成: 水平管 + 前管 + 后管 → 正交排列
- 功能单位: 壶腹 → 壶腹嵴 → 壶腹帽(凝胶状)
- 检测原理: 角加速度 → 内淋巴惯性滞后 → 壶腹帽偏转 → 毛细胞刺激
- 配对关系:
  - 水平管对: 左右水平管
  - 垂直管对: 左前/右后, 右前/左后

**耳石器官(2个)**
- 椭圆囊: 水平面运动检测, ~30,000毛细胞
- 球囊: 垂直面运动检测, ~16,000毛细胞
- 结构特征: 黄斑(毛细胞区) + 耳石膜 + 碳酸钙颗粒(耳石)
- 检测原理: 线性加速度/重力 → 耳石剪切力 → 毛细胞偏转

## 3. 毛细胞转导机制
**结构特征**
- 静纤毛: ~100个, 高度递增排列
- 动纤毛: 单个, 最高位置
- 尖端连接: 机械门控通道位点

**转导过程**
P: 发状纤维束偏转 → 尖端连接张力变化 → K+通道开放/关闭
- 向动纤毛偏转: 去极化 → 传入放电↑
- 远离动纤毛: 超极化 → 传入放电↓
- 静息放电率: ~100脉冲/秒 → 双向调制能力

## 4. 中枢前庭核团
**四个主要核团**
1. 内侧核: 半规管输入 → 眼动/脊髓投射
2. 外侧核(Deiters): 耳石输入 → 外侧前庭脊髓束
3. 上核: 半规管输入 → 眼动核团
4. 下行核: 混合输入 → 小脑/网状结构

**连合系统**
- 功能: "推拉"机制 → 信号增强 + 噪声降低
- 原理: 同侧兴奋 ↔ 对侧抑制(GABA能)
- 补偿: 单侧损伤后中枢重平衡

## 5. 前庭-眼动反射(VOR)
**旋转VOR**
- 三神经元弧: 半规管 → 前庭核 → 眼动核 → 眼肌
- 补偿原理: 头部旋转速度 = -眼球旋转速度
- 神经积分: 速度信号 → 位置信号(舌下前核/间位核)
- 眼球震颤: 慢相(补偿) + 快相(复位)

**平移VOR**
- 输入: 耳石器官信号
- 特点: 距离依赖性补偿
  - 近物: 大幅眼动
  - 远物: 小幅眼动
- 计算复杂性: 需整合目标距离信息

**VOR可塑性**
- 小脑调节: 绒球/副绒球 → 突触效能调整
- 适应机制:
  - 视觉误差信号 → 下橄榄核 → 攀缘纤维 → 浦肯野细胞
  - 长期可塑性: 多位点(浦肯野细胞 + 前庭核)
- 临床意义: 眼镜佩戴适应

## 6. 前庭-脊髓反射
**内侧前庭脊髓束**
- 起源: 内侧核(主) + 下行/外侧核
- 投射: 双侧颈脊髓
- 功能: 颈部肌肉控制 + 凝视-头协调

**外侧前庭脊髓束**
- 起源: 外侧核(Deiters)
- 投射: 同侧全脊髓
- 功能: 姿势控制 + 肢体/轴向肌肉调节

## 7. 高级功能整合
**倾斜-平移歧义解决**
- 问题: 耳石不能区分倾斜vs平移
- 解决: 半规管 + 耳石信号整合 → 中枢计算分离

**主动vs被动运动区分**
- 机制: 传出拷贝 + 前向模型 → 预测误差计算
- 功能: 自生运动时VOR抑制

## 8. 皮层投射与认知
**丘脑中继**
- 腹后外侧核(VPL)
- 腹外侧核(VL)
- 投射: 体感皮层 + 顶叶 + 岛叶

**皮层区域**
- 顶岛前庭皮层(PIVC): 多感觉整合中心
- 体感皮层: 2v区(顶内沟) + 3a区(中央沟)
- 运动相关: 额叶眼区 + 辅助运动区
- 空间处理: 内侧颞上区 + 腹侧顶内区

## 9. 空间导航功能
**关键细胞类型**
- 位置细胞: 海马 → 空间位置编码
- 头朝向细胞: 背侧丘脑 → 方向指南针
- 网格细胞: 内嗅皮层 → 空间度量
- 边界细胞: 环境边界
- 速度细胞: 运动速度编码

**前庭贡献**
- 半规管: 头部方向估计
- 耳石: 重力参考 + 线速度信息
- 整合: 前庭 + 本体感受 + 运动传出 → 空间地图更新

## 10. 临床应用
**诊断工具**
- 热量试验: 温水(44°C)/冷水(30°C) → 诱发眼震
  - 记忆口诀: COWS (Cold Opposite, Warm Same)
- 头脉冲试验: 快速头动 → VOR增益评估

**病理状态**
1. 单侧损伤:
   - 急性: 眩晕 + 眼震(向健侧)
   - 慢性: 中枢补偿
2. 双侧损伤:
   - 无眩晕(平衡损失)
   - 振动性视力丧失
   - 头动时视觉不稳

## 核心概念总结
前庭系统 := 多感觉整合中枢 + 运动控制器 + 空间定向器

关键创新:
1. 推拉编码 → 高信噪比
2. 多感觉整合 → 歧义解决
3. 预测误差 → 主动/被动区分
4. 分布式表征 → 无专一皮层但广泛影响
