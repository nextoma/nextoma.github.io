# 第*37*章\_小脑\_Summary

## P: 小脑的计算统一性与功能多样性

**Notion:**

- 小脑体积 := 10% 总脑体积 ∧ >50% 神经元
- 小脑皮层 := 高度规则重复单元 + 相同基本微回路
- 计算原理 := 不同区域对不同输入执行相似计算操作

**P: 小脑功能的五大基本原则**

1. **前馈控制** - 感觉反馈前提供肌肉收缩控制
2. **内部模型** - 依靠身体内部模型处理感觉输入
3. **时序功能** - 运动和感知时序的特殊作用
4. **适应学习** - 运动技能适应学习的关键
5. **非运动功能** - 与大脑皮层非运动区域广泛联系

## 37.1 小脑损伤的临床症状学

### 37.1.1 运动和姿势的特征性异常

**P: Gordon Holmes的经典观察**

> "左臂的动作是下意识的，但右臂的每一个动作我都得考虑"
> := 自动处理水平中断 → 皮层主动编程需求↑

**共济失调 := 运动不协调的综合症候群**

#### 肢体共济失调的核心特征

**1. 辨距障碍** `Dysmetria`

- 表现: 弯曲手部路径不对称
- 机制: 超过或低于预期目标 + 振荡
- 病理: 目标预测计算缺陷

**2. 运动分解** `Movement Decomposition`

- 表现: 多关节运动分解为多个部分
- 目的: 简化多关节运动控制
- 效果: 实际效率降低

**3. 相互作用力矩补偿失败**

- 正常: 肘部运动 → 肩部自动稳定
- 异常: 预测失败 → 延时反馈稳定(效果差)
- 结果: 单关节运动 → 多关节扰动

**4. 意图震颤** `Intention Tremor`

- 特点: 手接近目标时明显振荡
- 机制: 错误过度纠正尝试的连续
- 视觉依赖: 闭眼时基本消失

**5. 轮替运动障碍** `Dysdiadochokinesia`

- 表现: 重复运动速度规律性异常
- 测试: 快速交替运动困难
- 反映: 时间协调功能缺陷

#### 步态和平衡异常

**步态共济失调特征:**

- 步伐不规律(时间位置不一致)
- 重心转移困难 → 跌倒风险
- 躯干摆动(坐立行走时)
- 宽步进模式(补偿性稳定策略)

#### 其他运动症状

**构音障碍:** 说话含糊不清 + 时间不规律
**眼球震颤:** 慢相快相反复眼动
**肌张力减退:** 对被动肢体移位抵抗↓
**摆动反射:** 膝跳后腿部钟摆样多次摆动

### 37.1.2 感知能力和认知功能的影响

#### 本体感受能力的特异性缺陷

**P: 主动vs被动运动的差异**

**正常现象:** 主动运动本体感受 > 被动运动
**小脑患者:**

- 被动运动本体感受: 正常
- 主动运动本体感受: 明显缺陷
- **机制假说:** 小脑预测主动运动展开 → 运动协调 + 位置感知

#### 认知功能的区域特异性缺陷

**P: 功能成像揭示的认知参与**

**早期发现:**

- 朗读 vs 默读: 口部运动小脑区域↑
- 词汇联想 vs 朗读: 右小脑激活显著↑
- 学习缺陷: 右小脑损伤 → 词汇联想任务学习障碍

**认知缺陷的区域特异性:**

```
中线小脑(蚓体) → 情绪/情感失调 (∵边缘结构联系)
右小脑半球 → 语言功能障碍 (∵左大脑皮层联系)
左小脑半球 → 视觉空间功能障碍 (∵右大脑皮层联系)
```

**认知缺陷的变异性因素:**

- 测试时机: 损伤后早期 > 晚期
- 皮层代偿: 逐渐弥补功能丧失
- 损伤年龄: 童年损伤 → 更强烈持久缺陷
- 功能差异: 运动功能障碍 >> 认知功能障碍

**皮层代偿能力差异:**

- 认知过程: 皮层代偿能力强
- 运动控制: 皮层代偿能力弱

## 37.2 小脑的解剖结构与连接

### 37.2.1 小脑的基本解剖组织

**P: 大型皮层下脑结构的层次组织**

#### 基本结构组成

**灰质外层:** 小脑皮层
**内部白质:** 连接纤维
**深部核(3对):** 顶核, 间位核(栓状核+球状核), 齿状核

#### 表面复杂性与分区

**叶状分区(横向):**

- 原裂: 前叶 + 后叶(小脑体)
- 后外侧裂: 小脑体 + 绒球小结叶

**区域分区(纵向):**

- 中线: 小脑蚓体
- 两侧: 小脑半球{中间区域, 外侧区域}

## 小脑功能原理总结

**P: 计算统一性支撑功能多样性**

### 设计原理

1. **模块化重复** - 相同微回路处理不同输入
2. **前馈预测** - 运动后果预先计算
3. **误差学习** - 预测误差驱动适应
4. **时序处理** - 精确时间编码计算
5. **广泛投射** - 运动非运动功能整合

### 功能意义

**运动控制:**

- 精确协调多关节运动
- 平滑运动轨迹控制
- 快速适应环境变化
- 技能学习优化

**认知功能:**

- 语言处理时序
- 工作记忆维持
- 执行功能调控
- 情绪调节参与

### 临床价值

**诊断意义:**

- 共济失调综合征特征性表现
- 区域特异性功能缺陷模式
- 运动vs认知功能分离

**治疗靶点:**

- 运动康复训练
- 平衡功能训练
- 认知功能训练
- 神经调制治疗

**∴ 小脑 := 通过统一计算架构实现运动协调与认知调控的精密预测系统**

**Intuition:** 小脑体现了神经系统"预测控制"的核心智慧——不是被动响应已发生的错误，而是主动预测即将发生的需求，通过前馈控制实现精确协调，这种"预知而后控"的策略是高级运动技能和认知能力的神经基础。
