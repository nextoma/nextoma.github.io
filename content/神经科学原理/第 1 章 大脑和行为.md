# 第*2*章\_基因和行为 Summary

## 核心概念

**基本原理**: ∀行为 = f(基因 × 环境)

- 简单动物模式化行为 ← 环境影响
- 人类复杂行为 ← 基因约束
- 基因 → RNA/蛋白质 → 大脑发育/功能 → 行为

## 遗传学基础

### 遗传力证据

**双胞胎研究** (高尔顿, 1883):

- 同卵双胞胎: 100%基因共享
- 异卵双胞胎: ~50%基因共享
- 同卵一致性 > 异卵一致性 ⟹ 遗传成分存在
- Notion: 遗传力 := 群体特征中遗传因素比例

**精神疾病遗传风险**:

- 精神分裂症: 一般人群2% → 同卵双胞胎48%
- 遗传力估计值: ~50% (多数神经精神特征)

### 分子遗传学

**DNA结构**:

- 双链结构: A-T, G-C配对
- 基因 = {外显子(编码) + 内含子(非编码)}
- 人类基因组: ~20,000蛋白编码基因 + >40,000非编码RNA

**基因表达调控**:

```
DNA → 转录 → RNA → 剪接 → mRNA → 翻译 → 蛋白质
         ↑
    调控元件: {启动子, 增强子, 沉默子, 绝缘子}
```

**染色体组织**:

- 人类: 46条染色体 = 22对常染色体 + 2条性染色体
- XX(女性), XY(男性)
- 线粒体DNA: 母系遗传

### 基因型-表现型关系

**等位基因类型**:

- 纯合子: 相同等位基因
- 杂合子: 不同等位基因
- 半合子: X染色体基因(男性)

**突变效应**:

- 隐性: 需两个突变等位基因
- 显性: 一个突变等位基因即可表现
  - 单倍体不足: 50%基因产物不够
  - 显性负效应: 异常蛋白干扰正常功能

## 基因进化保守性

**跨物种基因共享**:

- 人类-黑猩猩: 99%蛋白编码基因
- 人类独有基因: <1%
- 与无脊椎动物共享: >50%基因

**保守功能类别**:

- 代谢酶: 10%
- 信号转导: 16%
- 核酸结合: 14%
- 细胞结构: 9%
- 未知功能: 42%

## 动物模型研究

### 昼夜节律 (果蝇→小鼠→人类)

**Period基因突变效应**:

- 野生型: 24小时周期
- 短日突变: 19小时
- 长日突变: 28小时
- 心律失常: 无节律

**分子时钟机制**:

```
时钟蛋白 → 激活Per/Tim转录
     ↓
Per/Tim蛋白积累
     ↓
Per/Tim二聚体 → 抑制自身转录
     ↓
蛋白降解 → 循环重启
```

**跨物种保守性**:

- 果蝇Period ↔ 小鼠Per
- 果蝇Clock ↔ 小鼠Clock
- PAS结构域: 共同特征

### 觅食行为变异

**果蝇幼虫** (Sokolowsky):

- 漫游者: 长距离移动
- 守家者: 相对静止
- 基因基础: foraging基因 → cGMP依赖性蛋白激酶

**蜜蜂年龄相关行为**:

- 年轻蜂: 护理(低foraging表达)
- 年老蜂: 觅食(高foraging表达)

### 社会行为调控

**线虫聚集行为**:

- 独居 ⊕ 群居进食模式
- NPR-1基因: 单氨基酸替换 → 行为改变

**哺乳动物配对关系**:

- 催产素(雌性) → 配对形成
- 加压素(雄性) → 配对维持
- 草原田鼠 vs 山地田鼠: V1a受体分布差异
  - 腹侧苍白球高表达 → 一夫一妻制
  - 腹侧苍白球低表达 → 非一夫一妻制

## 人类遗传综合症

### 单基因疾病

**苯丙酮尿症**:

- PAH基因突变 → 苯丙氨酸羟化酶缺失
- 苯丙氨酸积累 → 神经毒性
- 治疗: 低蛋白饮食
- ex: 基因×环境相互作用

### 神经发育综合症

**威廉综合症**:

- 7q11.23缺失(~27基因)
- 表现型: 高社交性 + 空间处理障碍
- 语言保留 ≠ 整体智力

**脆性X综合症**:

- FMR1基因突变 → FMRP蛋白缺失
- FMRP功能: mRNA翻译调控
- 影响: 突触可塑性 → 学习障碍
- 30%男性 → 自闭症谱系障碍

**Rett综合症**:

- MECP2基因突变(X连锁)
- 女性特异: 男性胚胎致死
- 进行性退化: 6-18月龄开始
- DNA甲基化调控异常

## 复杂精神疾病遗传学

### 自闭症谱系障碍(ASD)

**遗传架构**:

- 遗传力: 高(同卵双胞胎一致性~90%)
- 男女比例: 3:1

**风险变异类型**:

1. 罕见CNVs: 16p11.2, 1q21, 15q11-13
2. 新生突变: >100个基因确认
3. 功能富集:
   - 突触形成/功能基因
   - 转录调控基因
   - FMRP靶标

**基因多效性**:

- 同一CNV → {ASD, 精神分裂症, 智力障碍}
- "一对多"现象

### 精神分裂症

**遗传+环境因素**:

- 遗传力: ~80%
- 环境: 产前压力(饥荒, 感染)

**GWAS发现** (40,000病例):

- 108个风险位点
- 单个变异效应: <25%风险增加
- 多基因风险评分: 最高十分位 → 8-20倍风险

**C4基因机制**:

- MHC区域最强信号
- C4A表达 ↑ → 精神分裂症风险 ↑
- 假设: 突触修剪过度

**罕见变异**:

- 22q11缺失 → 精神病风险
- 外显子组测序: SETD1A等基因

## 遗传研究策略

### 经典遗传学

表现型 → 基因型:

1. 随机诱变
2. 筛选行为改变
3. 定位突变基因

### 反向遗传学

基因型 → 表现型:

1. 选择目标基因
2. 基因敲除/敲入
3. 分析表现型

### 现代技术

- Cre/loxP: 组织特异性敲除
- 四环素系统: 时间控制表达
- RNAi: 基因表达降低
- CRISPR: 精确基因编辑

## 关键洞察

**基因-行为关系复杂性**:

- 不完全外显率
- 可变表现度
- 遗传异质性: 多基因→同一表型
- 基因多效性: 一基因→多表型

**精神疾病共同特征**:

- 遗传风险重叠
- 常见+罕见变异共同作用
- 基因×环境相互作用关键

**进化视角**:

- 行为基因高度保守
- 调控变化 > 新基因产生
- 模式生物→人类转化价值

**未来方向**:

- 更大样本量 → 更多基因发现
- 风险基因 → 生物学机制
- 基因组学 + 神经影像 → 病理生理学
- 精准医疗: 基因型指导治疗
