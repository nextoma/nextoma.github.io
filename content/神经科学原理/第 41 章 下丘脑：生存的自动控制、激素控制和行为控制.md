# 第*42*章\_情感\_Summary

## 核心概念定义

`情感 := 神经生物学状态 → 协调行为&认知响应`

- `基本情感 := {快乐, 恐惧, 愤怒, 厌恶, 悲伤}`
- `复杂情感 := {羞耻, 内疚, 尴尬, 骄傲, 嫉妒}`
- `核心情感维度 := [效价(愉快↔不愉快), 强度(低↔高唤醒)]`

## 情感vs感觉区分

```
情感 := 中央脑状态 (可测量的生理响应)
感觉 := 有意识的情绪体验 (主观报告)
∴ 情感 ≠ 感觉 (达尔文进化保留性原理)
```

## 情感触发机制

### 刺激分类

```
非条件刺激 := 先天强化特性
ex: 疼痛电击, 恶心味道

条件刺激 := 联想学习获得情感意义
∴ ∀情感刺激: 检测 → 3系统激活
```

### 三系统响应模式

```
情感刺激 → 生理系统激活:
1. 内分泌系统 → 激素分泌 → 血液循环 → 身体组织&大脑
2. 自主神经系统 → {心血管, 内脏器官, 体腔组织}调节
3. 肌肉骨骼系统 → {僵滞, 战斗, 逃跑, 面部表情}
```

## 情感神经科学历史理论

### P1: 詹姆斯外围反馈理论 (1890)

```
情感刺激 → 感觉系统 → 运动皮层 → 身体响应
                                      ↓
有意识恐惧 ← 感觉反馈皮层处理 ← 外围反馈

Intuition: "我们因为逃跑而感到害怕"
```

### P2: 坎农-巴德中枢理论

```
情感 := 中枢神经系统内部过程

感觉信息 → 丘脑 → {下丘脑, 大脑皮层}
下丘脑: 情绪评估 → 脑干&脊髓 → 躯体响应
丘脑皮层: → 有意识感觉
```

### P3: 帕佩兹回路

```
经典情感环路:
扣带皮层 → 海马体 → 下丘脑 → 丘脑前部 → 扣带皮层
      ↑___________________|

Notion: 第一个解剖学特异性情感回路模型
```

## 情感测量方法学

唤醒测量: 皮肤电导响应
```

```
评分维度 := [效价强度, 特定情感强度]

ex测量工具:

```

```

```

```
