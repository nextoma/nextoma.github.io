# 第7章 神经系统的细胞 - 符号化摘要

## 核心概念定义

- **神经元** := 具有电兴奋性和长距离通信能力的特化细胞
- **神经胶质细胞** := 支持、调节神经元功能的细胞群体
- **细胞极化** := 神经元形态和功能的方向性不对称
- **细胞骨架** := 维持细胞形状和介导胞内转运的蛋白纤维系统

## 神经系统细胞分类

### P1: 神经系统细胞双重组成原理

神经系统 := 神经元 + 神经胶质细胞
∀ 复杂神经功能: 需要两类细胞协同作用

### 神经胶质细胞分类

**大胶质细胞** (Macroglia, ~90%):

1. **少突胶质细胞**: CNS髓鞘形成 + 轴突隔离
2. **施旺细胞**: PNS髓鞘形成 + 神经肌肉突触调节
3. **星形胶质细胞**: 营养支持 + 血脑屏障 + 信号调节

**小胶质细胞** (Microglia, ~10%):
功能: 免疫监控 + 吞噬作用 + 稳态调节

**髓鞘形成机制**:
施旺细胞内舌 → 轴突缠绕 → 多层膜结构 → 髓鞘
郎飞结间距 ≈ 1mm

## 7.1 神经元和胶质细胞的结构特征

### P2: 共同祖细胞原理

神经元 ∧ 胶质细胞 ← 神经上皮祖细胞
∴ 共享基本细胞生物学特征

### 细胞膜结构

**不对称双分子层**:

- 疏水屏障: 阻止大多数水溶性物质
- 内外膜叶组成不对称
- 含特殊蛋白: 离子通道 + 受体 + 转运蛋白

### 细胞质组成

**胞质溶胶**:

- 水相环境: 少数蛋白游离
- 功能复合物: 多蛋白非共价组装
- ex: NMDA受体复合体 > 100种支架蛋白

**膜性细胞器系统**:
液泡系统 := {粗面ER, 光滑ER, 高尔基复合体, 分泌小泡, 内体, 溶酶体}

**蛋白合成-转运途径**:
粗面ER → 高尔基复合体 → 分泌小泡 → 胞吐
内吞小泡 → 早期内体 → {循环内体 | 晚期内体 → 溶酶体}

### 核质交换

**核孔复合体**:

- 自由扩散: 分子量 < 5kDa
- 主动转运: 核定位信号 + 输入受体 + 核孔蛋白

**调节蛋白转运**:
胞质合成 → 核输入受体识别 → 核孔转运 → 基因调控

## 神经元形态极化

### P3: 神经元功能区室化原理

细胞体 ≠ 树突 ≠ 轴突
功能特化 ∝ 结构分化

**细胞体-树突连续性**:

- 细胞器分布: 完全连续
- 功能梯度: 密度随距离↓
- 树突棘: 光滑ER突起 + 多聚核糖体 → 局部蛋白合成

**轴突功能边界**:
轴突丘 := 细胞体与轴突功能分界
∵ 轴突缺乏: 核糖体 + 粗面ER + 高尔基复合体
∴ 轴突依赖: 胞体蛋白合成 + 轴突转运

### 树突棘类型

1. **细型**: 小接受面 + 突触受体集中
2. **粗短型**: 短而厚 + 成熟脑少见
3. **蘑菇型**: 大头部 + 复杂内膜系统

**棘可塑性**: 形态变化 ↔ 突触强度 ↔ 学习记忆

## 7.2 细胞骨架决定细胞形状

### P4: 细胞骨架结构-功能原理

细胞骨架 → 细胞形状 + 细胞器分布 + 胞内转运
占细胞总蛋白 ~25%

### 微管系统

**微管结构**:

- 直径: 25nm
- 组成: 13根原丝螺旋排列
- 原丝: α-微管蛋白 + β-微管蛋白交替柱
- 长度: 可达0.1mm

**微管极性**:

- 正端(+): 生长端 (朝向突起末端)
- 负端(-): 解聚端 (朝向细胞体)

**区域特异性**:

- 轴突: 单向排列 (+端→远端)
- 树突: 双向排列 (+端↔远端)

**动态特性**:
微管生长: GTP-微管蛋白 → GDP-微管蛋白 + Pi
动态不稳定: 快速聚合-解聚循环 (分裂细胞)
微管稳定: 微管相关蛋白(MAPs) → 稳定聚合物

**微管相关蛋白区域特异性**:

- 树突: MAP2
- 轴突: Tau蛋白 + MAP1b

**翻译后修饰调节**:
乙酰化 + 去酪氨酸化 + 聚谷氨酰化 → 微管稳定性

### 神经丝系统

**分级组装结构**:
单体 → 卷曲螺旋二聚体 → 四聚体(原丝) → 原纤丝 → 神经丝
最终直径: 10nm

**功能**: 细胞骨架支撑 + 轴突径向扩张

### 微丝系统

**肌动蛋白聚合物**:

- 直径: ~7nm
- 结构: 球状G-肌动蛋白双股螺旋
- 极性结构: 单体不对称性

**动态调节**:

- ~50%肌动蛋白为游离单体
- 结合蛋白调节: 聚合促进 + 加帽 + 切断 + 交联

**结构可塑性功能**:
微管 + 微丝动态 → 突起伸缩 → 突触连接变化 → 学习记忆

## 病理性蛋白积聚

### P5: 蛋白质稳态失调原理

正常蛋白 → 异常折叠/聚集 → 细胞功能障碍 → 神经退行性疾病

### Tau蛋白病变

**阿尔茨海默病**:
正常: Tau ↔ 微管结合/胞质游离
病理: 高磷酸化Tau → 成对螺旋丝 → 神经原纤维缠结

**机制**: 蛋白酶降解失败 → 不溶性积聚 → 微管蛋白聚合干扰

**其他Tau病**: 进行性核上性麻痹 + 额颞痴呆
遗传因素: Tau基因突变

### β-淀粉样蛋白病变

**淀粉样斑块形成**:
APP → β-分泌酶处理 → β-淀粉样蛋白肽 → 细胞外积聚

**遗传因素**: APP基因 + 早老素1/2基因突变

### α-突触核蛋白病变

**帕金森病路易体**:
可溶性α-突触核蛋白 → 不溶性聚集体 → 球形内含物

**泛素化**: 蛋白酶体降解标记 → 降解失败指示

## 关键生物学原理总结

### P6: 神经元特殊性原理

4大特征区别于一般细胞:

1. **形态极化**: 树突(接受) ← 胞体 → 轴突(传递)
2. **电兴奋性**: 离子通道 + 受体 → 跨膜电位变化
3. **化学传递**: 特殊分泌 → 神经递质释放
4. **长距离转运**: 细胞骨架 → 蛋白/细胞器转运

### P7: 胶质细胞多功能原理

传统观点: 纯结构支持
现代认识: 多方面神经元功能调节

- 发育引导 + 环境控制 + 突触支持 + 学习记忆参与

### P8: 基因表达决定性原理

细胞类型 := 特异基因表达模式
相同基因组 → 不同表达 → 不同蛋白组 → 不同功能

### P9: 结构-功能对应原理

解剖结构 ↔ 生理功能
细胞形态 ∝ 功能特化程度

### P10: 动态平衡维持原理

细胞骨架动态 + 膜系统循环 + 蛋白质更新 → 细胞稳态维持

- **髓鞘疾病**: 胶质细胞功能评估
