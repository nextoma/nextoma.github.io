## 语言结构层次

### P2: 分层组织原理

```
音素(Phonemes) := 有限&独特语音单元集合
→ 词素(Morphemes) := 最小结构单位
→ 单词(Words) := 意义载体
→ 句子(Sentences) := 语法规则组合的无限可能

关键特征: 有限元素 → 无限组合可能性
```

### 音位系统的语言特异性

```
ex: /r/和/l/区别
英语: 音位不同 (rock ≠ lock)
日语: 同一音位类别 (可互换)

ex: /p/和/b/区别
西班牙语: 区分pano和bano
英语: 视为相同发音

∴ 儿童学习任务: 发现本语言的音位分组规则
```

### 共振峰频率机制

```
语音区别基础: 声道形状 → 声学变化 → 共振峰频率模式
时间特征: 毫秒级快速变化
听觉要求: 跟踪快速共振峰变化 → 语义区分

技术挑战: 计算机连续语音流单词识别困难
∵ 口语中单词间无声音中断 (≠书面语空格分隔)
```

## 语言规则系统

### 音位排列规则

```
音素组合约束:
英语: /zb/组合不允许
波兰语: /zb/常见 (ex: Zbigniew)
∴ 各语言特异性音位组合规则
```

### 词素变化规则

```
前缀系统: un- (否定意义) + 形容词
后缀系统:
- 复数: {-s, -es} (pot→pots, box→boxes)
- 时态: 规则动词 + 词尾变化 (play→{plays,playing,played})
- 不规则: go→went (¬goed), break→broke (¬breaked)
```

### 句法规则

```
语序差异:
英语: 主-谓-宾 (He eats cake)
日语: 主-宾-谓 (Kare wa keeki o tabemasu)

短语结构:
英语: 形容词+名词 (a very intelligent man)
法语: 名词+形容词 (un homme tres intelligent)
```
