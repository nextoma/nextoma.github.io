# 第*61*章\_情绪障碍和焦虑 Summary

## 🧠 情绪神经回路

```
核心情绪系统：
杏仁核(恐惧) ↔ 前额叶皮层(调节) ↔ 海马体(记忆情境)
边缘系统 × 皮层控制 = 情绪表达
```

## 😰 焦虑障碍谱系

### 广泛性焦虑障碍(GAD)

```
临床特征：
持续过度担忧(>6个月)
躯体症状: 肌肉紧张，疲劳，睡眠障碍
认知症状: 注意困难，烦躁
```

**神经基础**：

- 杏仁核过度激活: 威胁检测过敏
- 前额叶调节不足: 认知控制缺陷
- GABA系统功能低下: 抑制不足

### 恐慌障碍

```
恐慌发作特征：
突发强烈恐惧感
自主神经症状: 心悸，出汗，呼吸困难
认知症状: 失控感，濒死感
预期焦虑: 对再次发作的恐惧
```

**神经机制**：

- 蓝斑-去甲肾上腺素系统过度激活
- CO₂敏感性增高: 窒息报警理论
- 海马体-前额叶回路异常

### 社交焦虑障碍

```
社交恐惧症状：
社交场合过度恐惧
回避社交行为
负性认知偏向: 过分关注他人评价
```

## 😔 抑郁障碍

### 重症抑郁障碍(MDD)

```
核心症状：
情绪症状: 持续抑郁情绪，兴趣丧失
认知症状: 注意困难，记忆问题，思维迟缓
躯体症状: 食欲改变，睡眠障碍，疲劳
```

**神经基础**：

- 前额叶-边缘系统失衡
- 海马体萎缩: 应激相关
- 默认模式网络异常: 反刍思维

### 双相障碍

```
情绪极端摆动：
躁狂发作: 情绪高涨，活动增多，判断力差
抑郁发作: 与单相抑郁类似
混合状态: 躁狂抑郁症状并存
```

**神经机制**：

- 奖励系统异常: 多巴胶能功能紊乱
- 情绪调节回路不稳定
- 昼夜节律异常: 生物钟基因变异

## 🧬 神经生物学机制

### 单胺类递质假说

```
经典理论：
5-羟色胺(5-HT)↓ → 抑郁
去甲肾上腺素(NE)↓ → 抑郁
多巴胺(DA)异常 → 双相障碍
```

**局限性**：

- 时间延迟矛盾: 药物即刻作用vs疗效延迟
- 症状异质性: 单一机制无法完全解释
- 治疗抗性: 部分患者对单胺药物无反应

### HPA轴异常

```
应激轴功能紊乱：
慢性应激 → 皮质醇分泌异常 → 海马体损伤
糖皮质激素受体敏感性改变
地塞米松抑制试验异常
```

### 神经可塑性理论

```
可塑性缺陷：
BDNF↓ → 神经元萎缩，突触减少
神经发生↓ → 海马体功能受损
抗抑郁药促进神经可塑性恢复
```

## 🧠 脑回路异常

### 恐惧回路

```
恐惧条件反射：
杏仁核 → 下丘脑/脑干 → 恐惧反应
前额叶皮层 → 杏仁核抑制(消退)
焦虑障碍: 消退学习缺陷
```

### 奖励回路

```
快感缺失机制：
腹侧被盖区-伏隔核通路功能减退
多巴胶能信号异常
奖励预期和体验分离
```

### 认知控制回路

```
执行功能缺陷：
背外侧前额叶皮层功能减退
前扣带皮层过度激活: 负性情绪关注
工作记忆和注意控制受损
```

## 🧬 遗传易感性

### 遗传度

```
双生子研究：
重症抑郁: 遗传度37%
双相障碍: 遗传度85%
焦虑障碍: 遗传度30-50%
```

### 候选基因

```
易感基因：
5-HTTLPR: 5-HT转运体基因多态性
COMT: 多巴胺代谢酶
BDNF: 脑源性神经营养因子
FKBP5: 糖皮质激素受体调节
```

### 基因-环境交互作用

```
G×E模型：
易感基因 × 环境应激 = 发病风险增加
表观遗传修饰: DNA甲基化改变
幼年创伤对基因表达的持久影响
```

## 💊 治疗策略

### 药物治疗

```
抗抑郁药分类：
SSRI: 5-HT再摄取抑制剂，一线治疗
SNRI: 5-HT/NE再摄取抑制剂
三环类: 多靶点，副作用较大
MAOI: 单胺氧化酶抑制剂，严重病例
```

**新型药物**：

- 氯胺酮: NMDA受体拮抗剂，快速抗抑郁
- 艾氯胺酮: 氯胺酮衍生物
- 致幻剂辅助治疗: 裸盖菇素，MDMA

### 心理治疗

```
循证心理治疗：
认知行为治疗(CBT): 认知重构，行为激活
人际心理治疗(IPT): 人际关系改善
正念认知治疗(MBCT): 复发预防
暴露疗法: 焦虑障碍特异性
```

### 物理治疗

```
神经调控技术：
电休克治疗(ECT): 严重抑郁，快速有效
经颅磁刺激(TMS): 左侧dlPFC刺激
深部脑刺激(DBS): 难治性病例
迷走神经刺激(VNS): 慢性刺激
```

## 🔬 生物标记物

### 神经影像标记

```
影像学发现：
结构MRI: 海马体萎缩，前额叶灰质减少
功能MRI: 默认模式网络异常
PET: 葡萄糖代谢异常
DTI: 白质完整性受损
```

### 外周标记物

```
血液标记：
炎症因子: IL-6, TNF-α升高
皮质醇: HPA轴功能异常
BDNF: 神经可塑性标记
```

## 👶 发育视角

### 儿童青少年情绪障碍

```
发育特点：
大脑成熟不同步: 边缘系统早熟，前额叶滞后
青春期激素影响: 情绪调节困难
学校适应压力: 社交焦虑高发
```

### 老年期情绪障碍

```
特殊考虑：
血管性抑郁: 脑血管病相关
痴呆相关抑郁: 认知衰退伴发
躯体疾病共病: 症状鉴别困难
```

## ⚡ 关键要点

1. **回路异常**：情绪障碍涉及多个脑回路功能失调
2. **神经递质系统**：单胺类递质假说仍是治疗基础
3. **应激脆弱性**：遗传易感性与环境应激交互作用
4. **可塑性机制**：神经可塑性缺陷及其修复是关键
5. **个体化治疗**：生物标记物指导精准治疗
6. **发育敏感期**：青春期是情绪障碍高发期
7. **治疗多样性**：药物、心理、物理治疗联合应用

## 🎯 核心公式

```
发病风险 = 遗传易感性 × 环境应激 × 发育阶段敏感性
治疗效果 = 药物疗效 × 心理治疗 × 患者依从性
康复程度 = 症状改善 + 功能恢复 + 复发预防
```
