# 第15章 递质释放 核心逻辑总结

## 主线架构

递质释放 := 化学突触信息传递核心机制

- 突触前末梢去极化 → Ca²⁺内流 → 囊泡融合 → 递质释放
- 量子释放假说 ⟹ 离散包装单位释放
- SNARE蛋白机制 := 膜融合分子基础

## 1. 去极化触发释放

### 1.1 电压依赖性释放

去极化阈值 := ~40mV (相对静息电位为正)

- 去极化 → 递质释放 [非线性关系]
- 10mV去极化增加 ⟹ 10倍释放增加
- Na⁺/K⁺通量 ≠ 释放必需条件
- 去极化本身 := 触发因素

### 1.2 实验证据

鱿鱼巨型突触实验:

- TTX阻断Na⁺通道 → 动作电位↓ → EPSP↓
- TEA阻断K⁺通道 + 直接去极化 → 正常释放
- ∴ 离子通量非必需，去极化是关键

## 2. Ca²⁺内流机制

### 2.1 Ca²⁺触发释放

Ca²⁺双重功能:

1. 去极化电荷载体
2. 胞内第二信使 → 触发胞吐

Ca²⁺浓度梯度:

- [Ca²⁺]ₒ ~2mM >> [Ca²⁺]ᵢ ~10⁻⁷M
- 巨大电化学驱动力 → 快速内流

### 2.2 Ca²⁺-释放关系

非线性依赖性:

- Ca²⁺内流2倍增加 → 释放>16倍增加
- 协同性系数 ~4-5 (多个Ca²⁺结合)
- 释放延迟 := 1-2ms (突触延迟)

### 2.3 Ca²⁺通道类型

| 类型  | 定位      | 功能     | 激活阈值 |
| ----- | --------- | -------- | -------- |
| P/Q型 | 活性区    | 快速释放 | 高电压   |
| N型   | 活性区    | 快速释放 | 高电压   |
| L型   | 胞体/树突 | 慢释放   | 高电压   |
| R型   | 胞体/树突 | 调节性   | 高电压   |
| T型   | 胞体      | 起搏器   | 低电压   |

## 3. 量子释放理论

### 3.1 量子假说

量子 := 单个囊泡递质含量

- 微型终板电位(mEPP) ~0.5mV
- 代表~5000个ACh分子
- 自发释放频率 ~1Hz

### 3.2 统计特性

释放遵循二项分布:

```
E = n × p × a
```

- n := 可释放位点数
- p := 释放概率
- a := 量子响应大小

### 3.3 释放模式

单囊泡释放 vs 多囊泡释放:

- 单囊泡: 每活性区最多1个囊泡
- 多囊泡: 受体饱和，响应定型化
- 释放概率范围: 0.1-0.9

## 4. 囊泡周期

### 4.1 胞吐步骤

1. 囊泡填充 (转运蛋白，V-ATPase)
2. 囊泡聚集 (储备池)
3. 停靠 (活性区结合)
4. 启动 (ATP依赖)
5. Ca²⁺触发融合
6. 递质释放

### 4.2 膜回收机制

| 机制         | 时间尺度 | 特点              |
| ------------ | -------- | ----------------- |
| 融合孔可逆   | 10-100ms | kiss-and-run/stay |
| 网格蛋白介导 | ~60s     | 经典途径          |
| 超快内吞     | 50ms     | 非网格蛋白        |
| 批量回收     | 高频后   | 大量内陷          |

## 5. SNARE蛋白机制

### 5.1 核心融合机制

SNARE复合体:

- v-SNARE: 突触泡蛋白(囊泡)
- t-SNAREs: 突触融合蛋白 + SNAP-25(质膜)
- 4螺旋束结构 → 膜接近

### 5.2 调节蛋白

- Munc18: SM蛋白，必需辅因子
- Complexin: 抑制自发释放，增强诱发释放
- NSF/SNAP: ATP依赖性复合体解离

## 6. Ca²⁺传感器

### 6.1 突触结合蛋白

结构特征:

- 跨膜域 + 2个C2域
- C2A结合3个Ca²⁺，C2B结合2个Ca²⁺
- 总计5个Ca²⁺结合位点

### 6.2 触发机制

Ca²⁺结合 → 磷脂相互作用 → SNARE激活:

1. 静息: 部分拉链化SNARE-complexin复合体
2. Ca²⁺流入: 突触结合蛋白结合Ca²⁺
3. 膜相互作用 → complexin解离
4. SNARE完全拉链 → 融合

### 6.3 亚型特异性

- Syt1/2/9: 快速同步释放
- Syt7: 异步释放
- 不同Ca²⁺亲和力 → 释放特性差异

## 7. 活性区结构

### 7.1 蛋白支架

核心组分:

- RIM: 中央组织蛋白
- Munc13: 囊泡启动
- RIM-BP: Ca²⁺通道定位
- Rab3/27: 囊泡停靠

### 7.2 超微结构

电镜断层扫描显示:

- 钉子(pins): 膜锚定点
- 肋骨(ribs): 囊泡连接
- 横梁(beams): 水平支架
- Ca²⁺通道定位于钉子位置

### 7.3 功能整合

活性区实现:

- 囊泡精确定位
- Ca²⁺通道聚集(<100nm距离)
- 快速耦合(μs级)
- 可塑性调节位点

## 8. 突触可塑性

### 8.1 短期可塑性

增强机制:

- 残余Ca²⁺积累
- 动作电位展宽 → Ca²⁺内流↑
- Munc13激活 → 囊泡动员

抑制机制:

- 囊泡耗竭
- Ca²⁺通道失活
- 受体脱敏

### 8.2 活动依赖性调节

高频刺激效应:

- 强直增强: 刺激期间递质↑
- 强直后增强(PTP): 持续数分钟-小时
- 残余Ca²⁺ → CaMKII激活

### 8.3 突触前调节

轴-轴突触:

- 突触前抑制: GABA_A → Cl⁻内流 → 去极化 → Na⁺通道失活
- 突触前易化: 5-HT → cAMP → K⁺通道关闭 → AP延长
- G蛋白直接调节释放机制

## 9. 调节机制

### 9.1 Ca²⁺通道调节

- G蛋白βγ亚基 → 通道关闭
- 磷酸化调节通道活性
- 空间定位调控耦合效率

### 9.2 囊泡池调节

- 突触蛋白磷酸化 → 储备池动员
- Munc13的DAG/Ca²⁺激活
- RIM磷酸化 → 释放增强

### 9.3 释放概率调控

影响因素:

- Ca²⁺传感器表达谱
- SNARE蛋白可用性
- 活性区蛋白磷酸化状态
- 代谢状态(ATP水平)

## 10. 病理意义

### 10.1 通道病

- Timothy综合征: L型通道突变 → 自闭症
- 偏瘫性偏头痛: P/Q型通道突变
- Lambert-Eaton综合征: 抗P/Q通道抗体

### 10.2 毒素靶点

- 肉毒毒素: 裂解SNARE蛋白
- 破伤风毒素: 裂解突触泡蛋白
- α-银环蛇毒素: 阻断ACh受体
- 芋螺毒素: 阻断N型Ca²⁺通道
