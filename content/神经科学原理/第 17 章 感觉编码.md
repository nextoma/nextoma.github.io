# 第*18*章\_体感系统的受体 Summary

## 核心定义与原理

**体感系统** := 检测身体物理刺激的感觉系统

- 包括: [触觉, 本体感觉, 温度觉, 疼痛, 瘙痒, 内脏感觉]
- P: 受体特异性 → 感觉模态特异性
- P: 标记线原理 → 特定受体 → 特定通路 → 特定感觉

**背根神经节(DRG)神经元**

- 结构: 伪单极神经元 (胞体 + 单轴突分支)
- 外周分支 → 感受器末梢
- 中枢分支 → 脊髓/脑干
- 胞体位置: DRG内(脊神经) ∨ 颅神经节(颅神经)

## 18.1 功能组织

### 神经纤维分类

**按直径/传导速度**

- Aα (12-20μm, 70-120m/s): 肌梭, 腱器官
- Aβ (5-12μm, 30-70m/s): 机械受体(触觉)
- Aδ (2-5μm, 12-30m/s): 冷觉, 快痛
- C (0.4-1.2μm, 0.5-2m/s): 温觉, 慢痛, 瘙痒

**功能意义**

- 传导速度 ∝ 纤维直径 ∝ 髓鞘厚度
- 快纤维 → 需要快速响应的信息(触觉, 本体感觉)
- 慢纤维 → 持续监测信息(温度, 慢性痛)

### 感受野特性

- 定义 := 引起神经元响应的皮肤区域
- 大小: 指尖(2-3mm²) < 手掌(5-10mm²) < 背部(>100mm²)
- 密度 ∝ 1/感受野大小 ∝ 空间分辨率
- P: 感受野重叠 → 刺激位置精确编码

## 18.2 机械受体系统

### 皮肤机械受体

**低阈值机械受体(LTMR)**

1. **Meissner小体** (FA-I)
   - 位置: 真皮乳头(无毛皮肤)
   - 适应: 快速
   - 功能: 低频振动(10-200Hz), 轻触
   - 感受野: 小(3-5mm)
   - ex: 纹理感知, 滑动检测

2. **Pacinian小体** (FA-II)
   - 位置: 深层真皮/皮下
   - 适应: 极快
   - 功能: 高频振动(200-300Hz)
   - 感受野: 大(>10cm²)
   - 机制: 层状结构 → 机械滤波 → 瞬态响应
   - ex: 工具使用振动反馈

3. **Merkel盘** (SA-I)
   - 位置: 表皮基底层
   - 适应: 慢
   - 功能: 形状, 边缘, 纹理
   - 感受野: 小(2-3mm), 边界清晰
   - 特殊: Merkel细胞 + 神经末梢复合体
   - ex: 盲文阅读, 精细辨别

4. **Ruffini末梢** (SA-II)
   - 位置: 真皮深层
   - 适应: 慢
   - 功能: 皮肤拉伸, 关节位置
   - 感受野: 大, 边界模糊
   - 方向选择性: 对特定拉伸方向敏感
   - ex: 把握物体, 手指位置感

**毛囊受体**

- 类型: [警戒毛(Guard), 曲折毛(Zigzag), 敏锐毛(Awl)]
- 纵向披针形末梢 → 毛发偏转检测
- 功能: 轻触, 空气流动
- C-LTMR: 情感性触觉(愉悦触摸)

### 本体感受器

**肌梭**

- 结构: 梭内肌纤维 + Ia/II类传入
- 功能: 肌肉长度 ∧ 长度变化率
- γ运动神经元 → 敏感性调节
- @ 第32章: 牵张反射

**高尔基腱器**

- 位置: 肌腱-肌肉连接处
- 功能: 肌肉张力检测
- Ib类传入 → 张力反馈
- 保护机制: 过度张力 → 抑制性反射

**关节受体**

- Ruffini样末梢: 关节极限位置
- Pacinian样小体: 关节运动
- 韧带受体: 关节稳定性监测

## 18.3 温度受体系统

### TRP通道家族

**冷受体**

- TRPM8: 15-25°C激活, 薄荷醇敏感
- TRPA1: <17°C极冷, 芥末油敏感
- 传入: Aδ纤维(冷) + C纤维(极冷)

**温受体**

- TRPV3: 33-39°C温暖
- TRPV4: 27-35°C
- TRPV1: >43°C热痛阈值
- 传入: C纤维为主

**温度编码**

- 静息放电 → 温度偏离 → 频率变化
- 动态响应: 温度变化率编码
- 静态响应: 绝对温度编码
- 适应: 持续刺激 → 响应衰减

## 18.4 伤害性受体系统

### 伤害性受体类型

**机械伤害性受体**

- 高阈值机械受体(HTM)
- Aδ纤维: 快速刺痛
- Piezo2通道参与?

**热伤害性受体**

- TRPV1: >43°C激活, 辣椒素受体
- TRPV2: >52°C极热
- 致敏: 炎症 → 阈值降低

**化学伤害性受体**

- 酸敏感: ASIC通道(pH<6)
- ATP受体: P2X3(组织损伤)
- 炎症介质: [缓激肽, PGE2, 5-HT, 组胺]

**多模式伤害性受体**

- C纤维多觉型(C-PMN)
- 响应: 机械 + 热 + 化学
- 沉默受体: 炎症后激活

### 痛觉双通路

**快痛通路**

- Aδ纤维 → 脊髓浅层 → 外侧脊髓丘脑束
- 特征: 定位准确, 刺痛, 快速
- 功能: 撤退反射, 警告

**慢痛通路**

- C纤维 → 脊髓浅层 → 前脊髓丘脑束
- 特征: 定位模糊, 灼烧感, 持续
- 功能: 促进愈合行为

### 痛觉调制

**外周敏化**

- 炎症汤: [K⁺, H⁺, ATP, 缓激肽, PGE2]
- NGF → TrkA → TRPV1磷酸化 → 阈值↓
- 轴突反射: 神经肽释放 → 血管扩张

**中枢敏化**

- 脊髓背角: NMDA受体激活 → LTP
- 下行调制: PAG → RVM → 脊髓
- @ 第20章: 痛觉详细机制

## 18.5 瘙痒受体系统

**特异性瘙痒受体**

- MrgprA3: 氯喹诱导痒
- MrgprC11: BAM8-22肽
- IL-31受体: 慢性瘙痒

**瘙痒-疼痛相互作用**

- 标记线分离: 痒 ≠ 弱痛
- 抑制: 搔抓 → 痛觉 → 痒觉抑制
- 脊髓: 胃泌素释放肽(GRP)系统

## 18.6 内脏感觉系统

**内脏受体类型**

- 机械受体: 膨胀, 收缩
- 化学受体: pH, 渗透压, 代谢产物
- 温度受体: 核心体温监测
- 多模式受体: 内脏痛

**传入通路**

- 迷走神经: 80%内脏传入
- 脊神经: 盆腔脏器
- 特点: 感受野大, 定位差

**内脏痛特征**

- 牵涉痛: 内脏 → 体表投射
- ex: 心绞痛 → 左臂/胸部
- 机制: 脊髓会聚投射

## 18.7 信息编码机制

### 刺激强度编码

**频率编码**

- 刺激强度↑ → 放电频率↑
- 动态范围: 0.1-1000Hz
- Weber-Fechner定律: R = k·log(S/S₀)

**群体编码**

- 募集原则: 强度↑ → 激活受体数↑
- 空间总和: 多受体 → 信号汇聚
- 阈值分布: 确保宽动态范围

### 适应机制

**快适应(FA)**

- 机制: [机械滤波, 通道失活, Ca²⁺依赖]
- 功能: 变化检测, 振动感知
- ex: Pacinian小体层状结构

**慢适应(SA)**

- 机制: 持续通道开放
- 功能: 静态信息, 持续压力
- ex: Merkel盘持续放电

## 18.8 解剖组织

### 皮节分布

**脊髓节段** → **体表区域**

- 颈节(C1-8): 头颈上肢
- 胸节(T1-12): 躯干
- 腰节(L1-5): 下肢前部
- 骶节(S1-5): 下肢后部, 会阴

**临床意义**

- 带状疱疹: 单皮节分布
- 脊髓损伤: 感觉平面定位
- 神经根病: 皮节感觉异常

### 中枢投射

**脊髓组织**

- Rexed分层(I-X层)
- I-II层: 伤害性/温度/痒
- III-IV层: 低阈值机械
- V-VI层: 广动态范围神经元

**上行通路**

- 背柱-内侧丘系: 精细触觉, 本体感觉
- 脊髓丘脑束: 痛温觉, 粗触觉
- 脊髓小脑束: 无意识本体感觉
- @ 第19章: 触觉通路详述

## 18.9 临床意义

### 感觉检查

**基本测试**

- 轻触: 棉签(Aβ纤维)
- 痛觉: 针刺(Aδ/C纤维)
- 温度: 冷热试管
- 振动: 音叉(Pacinian)
- 位置觉: 关节被动运动

### 病理状态

**感觉缺失**

- 周围神经病: 长度依赖(手套袜套样)
- 脊髓病: 分离性感觉障碍
- ex: 糖尿病神经病变, 脊髓空洞症

**异常感觉**

- 痛觉过敏: 阈值降低
- 痛觉超敏: 反应增强
- 异常性疼痛: 非痛刺激→痛
- 机制: 外周/中枢敏化

**幻肢感觉**

- 截肢后持续感觉
- 机制: 皮层重组, 中枢表征持续
- @ 第50章: 神经可塑性

## 关键洞察

1. **受体特异性**: 每种受体类型 → 特定感觉模态 (标记线原理)
2. **多层次编码**: 分子(离子通道) → 细胞(受体类型) → 系统(通路分离)
3. **动态调节**: 适应 ∧ 敏化机制 → 灵活响应环境变化
4. **并行处理**: 多种受体类型 ∧ 传导速度 → 丰富感觉体验
5. **临床相关**: 受体/通路特异性 → 诊断定位 ∧ 治疗靶点
