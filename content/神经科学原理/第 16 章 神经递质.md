# 第*17*章\_感觉编码 Summary

## 核心哲学基础

**柏拉图洞穴隐喻**

- 囚犯 := 感知者，仅见阴影
- 阴影 := 短暂/不连贯感觉
- 意义赋予 := 感知构建过程
- ∴ 感知 = 大脑主动构建，¬ 被动接收

**感知本质**

- P: "山就是我所定义的山" (格伦德尔)
- 感知 := 自我强加的结构
- 模糊感觉 → 意义构建 → 丰富体验
- 目标: 理解构建机制 ∧ 目的

## 17.1 心理物理学原理

### 感觉系统功能

- 基本问题: [∃物体?, 何物?, 有变化?]
- 核心功能: 检测 ∧ 识别
- 亚里士多德五感: [视,听,触,味,嗅]
- 现代扩展: +[疼痛,温度,本体感觉,前庭,内脏]

### 检测阈值

- **绝对阈值** := 50%试验中检测到的最小刺激
- **差别阈值(JND)** := 可区分的最小差异
- 韦伯定律: ΔS/S = K (恒定比例)
- ex: 重量K=1/40, 亮度K=1/60

### 刺激-感觉关系

- **韦伯-费希纳定律**: I = K·log(S/S₀)
  - I := 感觉强度
  - S := 刺激强度
  - 对数关系 → 宽动态范围压缩

- **史蒂文斯幂定律**: I = K(S - S₀)ⁿ
  - n<1: 压缩(亮度n=0.5)
  - n=1: 线性(长度)
  - n>1: 扩展(电击n=3.5)

### 信号检测理论

- 感知 = 信号检测 + 决策
- 噪声分布 ↔ 信号+噪声分布
- d' := 敏感度指标 = (μ*信号 - μ*噪声)/σ
- β := 决策标准(可调节)
- ROC曲线 → 区分敏感度vs偏差

## 17.2 神经编码机制

### 受体类型与转导

- **机械受体**: 压力/振动 → 离子通道开放
- **化学受体**: 分子结合 → GPCR/离子通道
- **光受体**: 光子 → 视紫红质构象变化
- **温度受体**: 温度 → TRP通道激活
- ∀受体: 刺激能量 → 感受器电位 → 动作电位

### 标记线编码

- P: 特定受体 → 特定通路 → 特定感觉
- 调制方式 ≠ 改变感觉性质
- ex: 电刺激视神经 → 光感(¬声音)
- @ 第22章: 视网膜特异性通路

### 群体编码

- 单个神经元 := 模糊编码器
- 群体活动 → 精确表征
- 贝叶斯推断: P(刺激|响应) ∝ P(响应|刺激)·P(刺激)
- 向量编码: 运动方向 = Σ(权重\_i × 方向\_i)

### 时间编码

- **相位型受体**: 快适应 → 变化检测
- **紧张型受体**: 慢适应 → 持续监测
- 适应机制: [受体脱敏, Ca²⁺依赖, 通道失活]
- 功能: 突出新刺激 ∧ 防止饱和

### 感受野组织

- 定义 := 引起响应的刺激空间
- 中心-周边拮抗 → 边缘增强
- ON中心: 中心+ 周边-
- OFF中心: 中心- 周边+
- 侧抑制 → 对比度增强 ∧ 空间分辨率↑

## 17.3 中枢回路精炼

### 拓扑映射

- P: 受体表面 → 脑图谱保持邻近关系
- 体感: 躯体拓扑图(小人图)
- 视觉: 视网膜拓扑图
- 听觉: 音调拓扑图(频率)
- 放大因子 ∝ 行为重要性
  - ex: 手指/唇 > 背部(体感)
  - ex: 中央凹 > 周边(视觉)

### 平行处理流

- ∀模态: 多条并行通路
- 视觉: [M通路(运动), P通路(形状/颜色), K通路]
- 体感: [背柱(精细触觉), 脊髓丘脑(粗触/温度)]
- 分流原则: 特征 → 专门通路 → 整合

### 层级处理

- 感受野复杂度: 初级 < 次级 < 联合皮层
- 视觉: V1(边缘) → V2(轮廓) → V4(形状) → IT(物体)
- 特征检测器层级:
  - 简单细胞: 方向选择性
  - 复杂细胞: 方向+位置不变性
  - 超复杂细胞: 端点/角检测
- @ 第23-24章: 视觉层级详述

### 自上而下调节

- 注意 → 增益调节 → 响应增强
- 预期 → 预测编码 → 误差最小化
- 语境 → 感受野调节
- 机制: [皮层反馈, 丘脑门控, 神经调质]

## 17.4 核心原理总结

### 四种基本响应特征

1. **模态**: 刺激类型(光/声/触等)
2. **位置**: 空间定位(感受野/拓扑图)
3. **强度**: 幅度编码(频率/群体)
4. **时程**: 时间动态(起始/持续/适应)

### 编码-解码框架

- 编码: 刺激 → 神经活动模式
- 解码: 神经活动 → 感知重建
- 优化目标: [信息最大化, 能量最小化, 预测误差最小]
- 限制因素: [噪声, 带宽, 能量]

### 可塑性与学习

- 短期: 适应/习惯化/敏感化
- 长期: 感受野重组/皮层重映射
- 关键期: 发育窗口期可塑性↑
- 成年可塑性: 经验依赖调节
- @ 第49章: 经验与突触精炼
- @ 第54章: 学习与记忆机制

### 多感觉整合

- 汇聚区: [上丘, 顶叶, 颞叶]
- 整合原则:
  - 空间一致性 → 增强
  - 时间同步性 → 结合
  - 逆有效性: 弱信号时整合↑
- 贝叶斯整合: 最优权重 ∝ 可靠性
- ex: McGurk效应(视听冲突)

### 感知恒常性

- P: 变化输入 → 稳定感知
- 机制:
  - 预测性编码
  - 先验知识整合
  - 语境归一化
  - 基准线适应
- ex: 颜色恒常(光照变化)
- ex: 大小恒常(距离变化)
