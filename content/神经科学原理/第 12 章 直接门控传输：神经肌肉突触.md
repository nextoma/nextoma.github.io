# 第*13*章\_中枢神经系统的突触整合 摘要

## 中枢vs外周突触复杂性

**关键区别**

- 肌纤维：1个运动神经元输入 → 仅兴奋（ACh → 烟碱受体）
- 中枢神经元：1000s输入 → 兴奋+抑制（多种递质/受体）
- 神经肌肉接头：1:1传递（高安全系数）
- 中枢突触：需50-100输入产生AP（汇聚需求）
- P: CNS中的信号整合 ⟹ 复杂计算能力

## 兴奋性传递：谷氨酸能系统

**离子型谷氨酸受体**

_AMPA受体_

- 结构：四聚体（4亚基：GluA1-4）
- 离子选择性：Na⁺/K⁺（gNa/gK ≈ 1）
- Ca²⁺通透性：被GluA2阻断（Q→R RNA编辑 @M2）
- 动力学：快速上升/衰减（~ms）
- 辅助亚基：TARPs（stargazin家族）→ 表面运输+门控调节

_NMDA受体_

- 结构：四聚体（2×GluN1 + 2×GluN2A-D）
- 离子选择性：Ca²⁺ >> Na⁺/K⁺
- 电压依赖性：Mg²⁺阻断 @静息(-65mV) → 去极化解除
- 双重需求：谷氨酸+去极化 → "符合检测器"
- 动力学：缓慢上升/衰减（10s-100s ms）
- 药理学：APV（拮抗剂），PCP/氯胺酮（通道阻断剂）

_红藻氨酸受体_

- 结构：四聚体（GluK1-5亚基）
- 功能：类似AMPA，较少

**受体组织**

- 突触后密度（PSD）：~350nm直径
- NMDA：~20受体（中心位置）
- AMPA：10-50受体（外周）
- 支架：PSD-95 PDZ域 → 受体聚集
- 蛋白网络：PSD-95 → GKAP → Shank → Homer → 细胞骨架

**结构域**

- 氨基末端域（ATD）：亚基组装
- 配体结合域（LBD）：蛤壳结构 → 谷氨酸结合
- 跨膜域：M1,M3,M4螺旋 + M2孔环
- M2选择性滤器：Q/R编辑位点决定Ca²⁺通透性

## 抑制性传递：GABA能/甘氨酸能系统

**GABA_A受体**

- 结构：五聚体（2α + 2β + γ/δ）
- 离子选择性：Cl⁻通道
- 亚基多样性：6α、3β、3γ、1δ型 → 功能多样性
- 结合位点：GABA @α-β界面，苯二氮䓬类 @α-γ界面
- 药理学：苯二氮䓬类、巴比妥类、乙醇、麻醉剂 → ↑通道开放

**甘氨酸受体**

- 结构：五聚体（3α + 2β）
- 离子选择性：Cl⁻通道
- 分布：主要脊髓/脑干
- 病理：α亚基突变 → 惊吓病
- 拮抗剂：马钱子碱 → 惊厥

**抑制机制**

1. 超极化：ECl (-70mV) < Vrest (-65mV) → Cl⁻内流 → 超极化
2. 分流：↑gCl → ↓输入电阻 → EPSP振幅降低
3. 塑造：自发活跃神经元的模式控制

## 突触可塑性

**长时程增强（LTP）**

- 诱导：高频刺激 → NMDA激活 → Ca²⁺内流
- 表达：↑AMPA受体 + ↑递质释放
- Ca²⁺信号：CaMKII、PKC激活 → 受体磷酸化
- 逆行信使：NO、内源性大麻素 → 突触前增强
- P: 输入特异性强化（赫布规则）

**脉冲时序依赖可塑性**

- 突触前在突触后之前 → LTP
- 突触后在突触前之前 → LTD
- 生理相关性 > 强直刺激

## 临床相关性

**兴奋性毒性**

- 机制：过量谷氨酸 → Ca²⁺超载 → 蛋白酶/脂酶激活 → 细胞死亡
- 疾病：中风、癫痫持续状态、亨廷顿病
- 悖论：生理性NMDA激活 → 神经保护

**神经精神疾病**

- 精神分裂症：NMDA功能低下（PCP/氯胺酮模型）
- 抗NMDA脑炎：自身抗体 → 精神病+癫痫发作
- 抑郁症：低剂量氯胺酮 → 快速抗抑郁效果
- 自闭症：neuroligin突变 → 突触发育缺陷
- 癫痫：GABA_A突变、GluA3自身抗体

## 突触形态学

**I型（兴奋性）**

- 圆形囊泡
- 不对称密度
- 位置：树突棘
- 递质：谷氨酸

**II型（抑制性）**

- 扁平囊泡
- 对称密度
- 位置：胞体、树突轴、AIS
- 递质：GABA、甘氨酸

## 整合原理

**空间总和**

- 距离规则：近端 > 远端有效性
- 补偿：↑远端受体密度
- 电缆特性：电流随距离衰减

**时间总和**

- AMPA：短暂EPSCs（~ms）
- NMDA：延长EPSCs（10s-100s ms）
- 整合窗口由τ决定

**计算操作**

- 兴奋：去极化向阈值
- 减法抑制：超极化离开阈值
- 除法抑制：分流降低增益

## 受体进化

**基因家族**

1. Cys环/pLGIC：nAChR、GABA_A、GlyR（五聚体）
2. 谷氨酸受体：AMPA、NMDA、红藻氨酸（四聚体）
3. P2X受体：ATP门控（三聚体）

**进化关系**

- 家族内同源性 >> 家族间相似性
- 亚基基因：复制 → 分化 → 特化
- 选择性剪接 → 进一步多样性
