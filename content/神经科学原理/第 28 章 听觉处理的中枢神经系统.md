# 第 29 章 嗅觉和味觉：化学感觉 Summary

## 1. 嗅觉系统概述
**嗅觉感觉神经元(OSN)**
- 位置: 嗅上皮(~5cm²人类)
- 结构: 双极神经元 + 纤毛(含受体)
- 寿命: 30-60天 → 基底干细胞持续更新
- 数量: 人类~350种受体, 小鼠~1000种

**气味受体特征**
- 家族: GPCR超家族成员
- 结构: 7次跨膜 + 可变结合口袋
- 多样性: 氨基酸序列高度可变
- 其他家族: TAAR(微量胺相关), MS4R

## 2. 气味编码原理
**组合编码策略**
- 一个神经元-一个受体: 每个OSN表达单一受体类型
- 多对多关系:
  - 一个受体 → 多种气味
  - 一种气味 → 多个受体
- 组合特异性: 每种气味 := 独特受体组合

**信号转导机制**
P: 气味结合 → Gαolf激活 → AC III → cAMP↑ → CNG通道开放 → Ca²⁺/Na⁺内流 → 去极化

**浓度编码**
- 低浓度: 高亲和力受体激活
- 高浓度: 低亲和力受体募集 → 组合改变 → 感知变化
- ex: 硫萜品醇: 热带水果(低) → 葡萄柚(中) → 腐臭(高)

## 3. 嗅觉信息处理通路
**嗅上皮组织**
- 空间区域: 不同区域表达不同受体组
- 分散分布: 同受体神经元随机散布
- 优势: 部分损伤后仍能检测

**嗅球转化**
- 小球汇聚: 同受体OSN → 2个小球(双侧各1)
- 数量压缩: 100:1 (OSN:僧帽/簇状细胞)
- 精确映射: 受体特异性小球位置固定
- 个体间相似性: 小球激活模式保守

**嗅觉皮层**
- 主要区域: 梨状皮层(最大)
- 其他: 前嗅核, 杏仁核皮层核, 嗅结节, 内嗅皮层
- 组织: 分散表征(无拓扑映射)
- 整合: 多模态输入(嗅/味/视)

## 4. 信息素检测系统
**犁鼻器(VNO)**
- 受体: V1R + V2R家族(各~100种)
- V1R: 挥发性化学物质
- V2R: 蛋白质信息素(含大胞外域)
- 通路: VNO → 副嗅球 → 内侧杏仁核 → 下丘脑

**功能差异**
- 主嗅觉: 气味识别 + 学习
- 犁鼻器: 先天行为 + 生理响应
- 物种特异: 人类无VNO

## 5. 味觉系统
**五种基本味觉**
1. 甜味: 能量(碳水化合物)
2. 鲜味: 蛋白质
3. 苦味: 毒素警告
4. 咸味: 电解质平衡
5. 酸味: 腐败/发酵

**味蕾组织**
- 位置: 舌(主要) + 上颚/咽/会厌
- 乳突类型:
  - 菌状: 前2/3舌(1-5味蕾/乳突)
  - 叶状: 舌侧缘(数百味蕾)
  - 环状: 舌后部(数千味蕾)
- 细胞组成: ~100味觉细胞/味蕾

## 6. 味觉受体机制
**GPCR类受体**
- 甜味: T1R2 + T1R3 (异二聚体)
- 鲜味: T1R1 + T1R3 (异二聚体)
- 苦味: T2R家族(~30种)
  - 单细胞表达多种T2R
  - 高亲和力(μM级)

**离子通道受体**
- 咸味: ENaC (上皮Na⁺通道)
  - 低盐: ENaC → 食欲
  - 高盐: 苦/酸细胞激活 → 厌恶
- 酸味: Otop1 (质子选择性通道)

## 7. 味觉编码原理
**标记线原理**
- 一个细胞-一种味觉: 甜/鲜/苦/咸/酸细胞分离
- 专用通路: 特定细胞 → 特定神经元 → 特定行为
- 证据: 受体错配表达改变行为偏好

**物种差异**
- 大熊猫: 缺失鲜味受体(竹食性)
- 猫科: 缺失甜味受体(肉食性)
- 吸血蝙蝠: 缺失甜味+鲜味(血食性)

## 8. 味觉信息通路
**外周通路**
- 颅神经: VII(鼓索), IX(舌咽), X(迷走)
- 神经节: 膝状, 岩骨, 结状

**中枢通路**
P: 味蕾 → 孤束核 → (臂旁核) → 丘脑VPM → 味觉皮层(岛叶/额岛盖)

**皮层组织**
- 热点映射: 甜味/苦味分离区域
- 直接激活: 引发相应行为(吸引/厌恶)
- 先天性: 无需学习即有响应

## 9. 比较化学感觉
**果蝇嗅觉**
- 受体: ~60个OR基因(独立进化)
- 组织: OSN → 嗅叶小球 → 蘑菇体/侧原脑
- 相似性: 一神经元一受体, 小球汇聚

**线虫化学感觉**
- 神经元: 32个化学感觉神经元
- 受体: ~1700个GPCR(最多)
- 特点: 一神经元多受体
- 行为: 标记线硬连线

**进化驱动力**
- 快速进化: 受体家族变化快
- 独立起源: 不同门类独立进化
- 生态适应: 反映饮食/生活方式

## 核心概念总结
化学感觉 := 组合编码(嗅觉) + 标记线(味觉)

关键创新:
1. 组合编码 → 巨大辨别能力
2. 小球汇聚 → 稳定空间映射
3. 标记线 → 先天行为连接
4. 受体多样化 → 生态位适应
5. 持续更新 → 终生功能维持
