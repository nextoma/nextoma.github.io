# 第16章 神经递质 核心逻辑总结

## 主线架构

神经递质 := 化学突触传递信息载体

- 小分子递质 ↔ 神经肽 [两大类别]
- 合成 → 储存 → 释放 → 作用 → 清除 [生命周期]
- 囊泡转运蛋白 ⟹ 递质浓缩机制

## 1. 递质鉴定标准

### 1.1 经典四条标准

递质定义:

1. 突触前合成
2. 囊泡储存 + 胞吐释放
3. 外源应用 ~> 内源效应
4. 清除机制存在

### 1.2 现代扩展

非经典递质:

- 气体递质(NO) := 跨膜扩散
- 内源性大麻素 := 逆行信使
- 腺苷/ATP := 嘌呤能传递

## 2. 小分子递质

### 2.1 乙酰胆碱

合成通路:

```
胆碱 + 乙酰辅酶A → [胆碱乙酰转移酶] → 乙酰胆碱
```

- 限速酶 := 胆碱乙酰转移酶(ChAT)
- 分布: 神经肌肉接头 + 自主神经节 + 基底核
- 降解: 乙酰胆碱酯酶(AChE) → 胆碱 + 乙酸

### 2.2 儿茶酚胺递质

生物合成级联:

```
酪氨酸 → [TH] → L-多巴 → [AADC] → 多巴胺
         ↓[DBH]
      去甲肾上腺素
         ↓[PNMT]
       肾上腺素
```

关键酶特性:
|酶|缩写|定位|调节|
|---|---|---|---|
|酪氨酸羟化酶|TH|胞质|限速，反馈抑制|
|芳香族氨基酸脱羧酶|AADC|胞质|广泛分布|
|多巴胺β-羟化酶|DBH|囊泡内|膜结合|
|苯乙醇胺N-甲基转移酶|PNMT|胞质|肾上腺特异|

### 2.3 血清素(5-HT)

合成:

```
色氨酸 → [TPH] → 5-羟色氨酸 → [AADC] → 5-HT
```

- 限速酶 := 色氨酸羟化酶(TPH)
- 分布: 中缝核 → 全脑投射
- 功能: 情绪、睡眠、食欲调节

### 2.4 氨基酸递质

| 递质   | 前体       | 合成酶             | 功能   |
| ------ | ---------- | ------------------ | ------ |
| 谷氨酸 | α-酮戊二酸 | 谷氨酰胺酶         | 兴奋性 |
| GABA   | 谷氨酸     | GAD                | 抑制性 |
| 甘氨酸 | 丝氨酸     | 丝氨酸羟甲基转移酶 | 抑制性 |

### 2.5 ATP/腺苷

嘌呤能传递:

- ATP → ADP → AMP → 腺苷 [降解级联]
- P2X受体 := 离子型(ATP)
- P2Y受体 := 代谢型(ATP)
- P1受体 := 代谢型(腺苷)

## 3. 囊泡转运机制

### 3.1 转运驱动力

质子梯度建立:

```
V-ATPase + ATP → ADP + Pi + H⁺(囊泡内)
ΔpH ~2.5单位 (pH 5.0 vs 7.4)
```

### 3.2 囊泡转运蛋白家族

| 转运蛋白 | 底物        | 特异性 | 浓缩倍数  |
| -------- | ----------- | ------ | --------- |
| VAChT    | 乙酰胆碱    | 高     | ~500x     |
| VMAT1/2  | 所有生物胺  | 低     | ~100,000x |
| VGLUT1-3 | 谷氨酸      | 高     | ~50x      |
| VGAT     | GABA/甘氨酸 | 中     | ~10x      |

### 3.3 转运机制

反向转运模式:

```
递质(胞质) + 2H⁺(囊泡) ⇌ 递质(囊泡) + 2H⁺(胞质)
```

- 12次跨膜结构
- 与细菌耐药转运蛋白同源

## 4. 神经肽特征

### 4.1 生物合成

肽处理通路:

```
基因 → mRNA → 前前肽 → [信号肽酶] → 前肽
      → [内切酶] → 活性肽 + 修饰
```

### 4.2 主要肽家族

| 家族       | 成员                   | 功能             |
| ---------- | ---------------------- | ---------------- |
| 阿片肽     | 脑啡肽、内啡肽、强啡肽 | 镇痛、奖赏       |
| 速激肽     | P物质、神经激肽        | 疼痛传递         |
| 神经垂体肽 | 催产素、加压素         | 社会行为、水平衡 |
| 胃肠肽     | VIP、CCK、生长抑素     | 消化、饱腹感     |

### 4.3 多聚蛋白处理

前阿黑皮素(POMC)实例:

```
POMC → ACTH + β-脂蛋白
     ↓        ↓
   CLIP    β-内啡肽 + γ-脂蛋白
     +         ↓
  α-MSH    α-内啡肽
```

- 组织特异性处理
- 不同产物协同/拮抗作用

## 5. 囊泡类型对比

### 5.1 小透明囊泡

- 直径: ~40nm
- 内容: 小分子递质
- 合成: 局部回收
- 释放: 活性区，快速
- Ca²⁺需求: 局部微域

### 5.2 大致密核心囊泡

- 直径: 70-250nm
- 内容: 神经肽 + 小分子
- 合成: 胞体TGN
- 释放: 非活性区，慢速
- Ca²⁺需求: 全面升高

## 6. 递质清除机制

### 6.1 清除方式

| 机制   | 递质           | 特点     | 时间尺度 |
| ------ | -------------- | -------- | -------- |
| 酶降解 | ACh            | AChE水解 | <1ms     |
| 再摄取 | 生物胺、氨基酸 | 转运蛋白 | 1-10ms   |
| 扩散   | 神经肽、气体   | 被动     | 100ms-s  |

### 6.2 质膜转运蛋白

神经递质钠共转运体(NSS)家族:

```
递质 + Na⁺ + Cl⁻ (胞外) → 递质 + Na⁺ + Cl⁻ (胞内)
```

- DAT: 多巴胺转运体
- NET: 去甲肾上腺素转运体
- SERT: 5-HT转运体
- GAT1-3: GABA转运体

谷氨酸转运体(EAAT)家族:

```
Glu⁻ + 3Na⁺ + H⁺ ⇌ Glu⁻(胞内) + K⁺
净电荷: +2 → 内向电流
```

## 7. 共释放现象

### 7.1 共存模式

常见组合:

- ACh + VIP (副交感)
- ACh + CGRP (运动神经元)
- 谷氨酸 + 多巴胺 (中脑)
- 谷氨酸 + 强啡肽 (海马)
- ATP + 任何递质

### 7.2 功能意义

共传递优势:

- 时间编码: 快速(小分子) + 持续(肽)
- 空间编码: 突触(私有) + 旁分泌(社交)
- 功能互补: 兴奋 + 调节

## 8. 药理学靶点

### 8.1 合成调节

- L-DOPA: TH旁路，帕金森病治疗
- 卡比多巴: 外周AADC抑制剂

### 8.2 囊泡转运干预

- 利血平: VMAT阻断 → 囊泡耗竭
- 丁苯那嗪: VMAT2选择性抑制

### 8.3 再摄取抑制

| 药物     | 靶点         | 临床应用  |
| -------- | ------------ | --------- |
| SSRIs    | SERT         | 抑郁症    |
| 可卡因   | DAT/NET/SERT | 滥用药物  |
| 安非他明 | 逆转DAT      | ADHD/滥用 |

### 8.4 降解酶抑制

- 毒扁豆碱: AChE抑制 → 肌无力治疗
- MAO抑制剂: 生物胺降解↓ → 抗抑郁

## 9. 活动依赖性调节

### 9.1 短期调节

磷酸化机制:

```
神经活动↑ → cAMP↑ → PKA激活
         → TH磷酸化 → 活性↑
```

- 时间尺度: 秒-分钟
- 可逆性: 磷酸酶去磷酸化

### 9.2 长期调节

转录调控:

```
持续活动 → CREB磷酸化 → CRE结合
        → TH基因转录↑ → 酶蛋白↑
```

- 时间尺度: 小时-天
- 协调调节: TH、DBH、PNMT共调

## 10. 检测技术

### 10.1 组织化学

- Falck-Hillarp: 儿茶酚胺荧光
- 免疫组化: 抗体标记
- 原位杂交: mRNA定位
- 电镜金标: 超微定位

### 10.2 功能检测

- 安培法: 电化学检测(DA、5-HT)
- 荧光假递质(FFN): 实时释放监测
- GFP融合: 活体标记
- 光遗传报告基因: 递质释放可视化

## 关键整合

**递质多样性基础**:
合成酶特异性 × 囊泡转运选择性 × 受体分布 → 化学编码

**能量耦合**:
ATP → H⁺梯度 → 递质浓缩 (囊泡)
ATP → Na⁺/K⁺梯度 → 递质再摄取 (质膜)

**时空特异性**:

- 小分子: 快速/点对点/活性区
- 神经肽: 缓慢/扩散性/突触外
- 共释放: 多维信息编码

**可塑性接口**:
递质合成 ⟷ 神经活动 ⟷ 基因表达
↓
突触强度调节 → 学习记忆基础

**临床相关性**:
递质系统异常 ⟹ 神经精神疾病

- 多巴胺↓ → 帕金森病
- 5-HT/NE失调 → 抑郁症
- 谷氨酸过度 → 兴奋毒性
- GABA不足 → 癫痫/焦虑
