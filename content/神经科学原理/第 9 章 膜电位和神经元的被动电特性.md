# 第10章 传播信号：动作电位 - 符号化摘要

## 核心概念定义

- **动作电位** := 自传播的全或无电信号，实现远距离信号传递
- **阈值** := 触发动作电位产生的最小去极化水平 (~-50mV)
- **不应期** := 动作电位后神经元暂时失去再次兴奋能力的时期
- **电压门控** := 离子通道响应膜电位变化而开闭的机制

## 动作电位的四大特性

### P1: 动作电位独特性原理

区别于一般生物过程的分级响应，动作电位具有数字化特性

**特性1: 阈值启动**
膜电位 < 阈值 → 被动电紧张响应
膜电位 ≥ 阈值 → 主动全或无动作电位

**特性2: 全或无特性**
∀ 超阈刺激: 动作电位幅度与形状恒定
刺激强度 ≠ 响应强度 (区别于欧姆定律)

**特性3: 无衰减传导**
动作电位沿轴突传播：恒定幅度 + 自我再生
区别于被动电传导的指数衰减

**特性4: 不应期限制**
绝对不应期 + 相对不应期 → 传导频率上限

## 10.1 动作电位由离子流过电压门控通道产生

### 历史性发现 (Cole & Curtis, 1939)

**关键观察**: 动作电位期间膜电导急剧增加
膜电导 ↑ → 离子渗透性 ↑ → 动作电位机制线索

### Hodgkin-Katz洞察

**Na⁺依赖性**: 降低[Na⁺]ₒ → 动作电位幅度↓
**双相机制**:

- 上升相: Na⁺内流主导
- 下降相: K⁺外流主导

### 10.1.1 电压钳技术

### P2: 电压钳原理

**目标**: 打破膜电位与离子通道门控的正反馈循环
**机制**: 负反馈控制 → 膜电位钳制在指令值

**正反馈问题**:
去极化 → Na⁺通道开放 → Na⁺内流 → 进一步去极化 → ...

**电压钳解决方案**:
测量膜电位 → 与指令电位比较 → 反馈放大器 → 补偿电流注入

**数学表达**:
I_clamp = -I_membrane
∴ V_m = V_command (恒定)

### 膜片钳扩展

**全细胞模式**: 小神经元电压钳记录
**优势**:

- 细胞内环境控制
- 单细胞分析
- 高时间分辨率

### 电流成分分离

**总膜电流**: I_m = I_c + I_l + I_Na + I_K

**各成分**:

- **电容电流** (I_c): 瞬时充放电，仅在电压跃变时存在
- **漏电流** (I_l): 恒定，通过静息通道
- **Na⁺电流** (I_Na): 瞬时内向，早期激活
- **K⁺电流** (I_K): 持续外向，延迟激活

### 药理学分离

**河豚毒素** (TTX): 特异性阻断电压门控Na⁺通道
**四乙胺** (TEA): 特异性阻断部分电压门控K⁺通道

TTX存在 → I_m = I_c + I_l + I_K
TEA存在 → I_m = I_c + I_l + I_Na

### 10.1.2 离子电流的电压依赖性

### P3: 离子电流双因子模型

**电流方程**:
I_Na = g_Na × (V_m - E_Na) (公式10.1)
I_K = g_K × (V_m - E_K) (公式10.2)

**双因子决定**:

1. **电导因子**: g_Na, g_K (开放通道数量)
2. **驱动力因子**: (V_m - E_ion) (电化学梯度)

### Na⁺电流特性

**电压依赖性**:

- 小去极化 → I_Na内向↑ (g_Na↑ + 驱动力大)
- V_m → E_Na → I_Na → 0 (驱动力→0)
- V_m > E_Na → I_Na外向 (驱动力反向)

**时间依赖性**:
激活快速 → 失活 → 电流下降

### K⁺电流特性

**电压依赖性**:
去极化↑ → g_K↑ ∧ (V_m - E_K)↑ → I_K外向↑

**时间依赖性**:
激活缓慢 → 持续开放 (无失活)

## 通道门控机制

### P4: 电压门控分子机制

**电压传感器**: 带电氨基酸残基感受膜电场变化
**门控过程**: 构象变化 → 通道孔开闭

### Na⁺通道三状态模型

**状态转换**:
关闭态(C) ⟷ 开放态(O) → 失活态(I)
↑******\_\_\_******|

**状态特征**:

- **关闭态**: 静息状态，可激活
- **开放态**: 传导离子
- **失活态**: 不传导，需复活才能再激活

### 失活机制

**N型失活**: 快速失活，"球-链"模型
**分子机制**: 胞内球状结构阻断通道孔

**生理意义**:

- 限制Na⁺内流持续时间
- 确保动作电位单向传播
- 产生不应期

## 动作电位的分子基础

### P5: Hodgkin-Huxley模型

**数学描述**:
C_m(dV_m/dt) = I_stim - g_Na×m³×h×(V_m-E_Na) - g_K×n⁴×(V_m-E_K) - g_l×(V_m-E_l)

**门控变量**:

- **m**: Na⁺通道激活 (m³ → 协同性)
- **h**: Na⁺通道失活
- **n**: K⁺通道激活 (n⁴ → 协同性)

### 动作电位时程分析

**Phase 0 - 去极化**:
阈值达到 → Na⁺通道快速开放 → Na⁺内流 → 正反馈
dV/dt > 0, I_Na >> I_K

**Phase 1 - 早期复极化**:
Na⁺通道开始失活 → Na⁺电流↓
部分K⁺通道开放 → K⁺外流

**Phase 2 - 平台期** (某些细胞):
Ca²⁺通道激活 → Ca²⁺内流平衡K⁺外流

**Phase 3 - 复极化**:
Na⁺通道完全失活 → K⁺通道完全激活 → K⁺外流主导
dV/dt < 0, I_K >> I_Na

**Phase 4 - 后超极化**:
K⁺通道缓慢关闭 → 膜电位逐渐恢复静息值

## 传导机制

### P6: 动作电位传导原理

**局部电路理论**: 动作电位区域 → 邻近膜区域去极化 → 传播

**传导方程**:
v = √(R_m/(R_a×C_m)) × f(通道动力学)

其中:

- v: 传导速度
- R_m: 膜电阻
- R_a: 轴质电阻
- C_m: 膜电容

### 传导速度影响因素

**轴突直径**:
直径↑ → R_a↓ → 传导速度↑
∵ R_a ∝ 1/截面积 ∝ 1/d²

**髓鞘化**:
髓鞘包裹 → R_m↑ ∧ C_m↓ → 传导速度↑↑
跳跃式传导: 郎飞结间快速传导

**温度效应**:
温度↑ → 通道动力学↑ → 传导速度↑
Q₁₀ ≈ 2-3 (每10°C变化的速度因子)

## 不应期机制

### P7: 不应期生理意义

**绝对不应期**: Na⁺通道失活期，无法激发动作电位
**相对不应期**: Na⁺通道部分恢复，需更强刺激

**分子基础**:
Na⁺通道失活 → 可用Na⁺通道数↓ → 兴奋性↓

**功能重要性**:

1. 单向传导保证
2. 最高放电频率限制
3. 编码动态范围确定

## 动作电位的生理变异

### 不同神经元类型

**快速Na⁺通道**: 大多数神经元，快速传导
**慢速Ca²⁺通道**: 某些神经元，慢传导 + 长持续时间

### 调节机制

**通道磷酸化**: 蛋白激酶调节 → 门控特性改变
**神经调质**: 间接调节通道功能

## 病理意义

### 通道病

**Nav通道突变**:

- 癫痫: 失活缺陷 → 过度兴奋
- 肌强直: 持续激活 → 肌肉痉挛

**获得性通道功能障碍**:

- 脱髓鞘疾病 → 传导阻滞
- 缺氧缺血 → 通道功能异常

### 药物作用点

**局部麻醉药**: Na⁺通道阻断 → 感觉阻滞
**抗癫痫药**: Na⁺通道调节 → 异常放电抑制

## 关键原理总结

### P8: 数字信号原理

动作电位 = 生物数字信号
模拟输入(膜电位变化) → 数字输出(全或无动作电位)

### P9: 分子机器原理

电压门控离子通道 = 分子电压传感器 + 选择性滤器 + 门控机制
蛋白构象变化 ↔ 离子传导状态

### P10: 能量效率原理

被动离子扩散驱动 → 无需ATP → 高效信号传递
代价: 需要离子泵恢复梯度

### P11: 时空编码原理

动作电位频率 ↔ 信号强度 (频率编码)
动作电位时间 ↔ 信号时序 (时间编码)
