# 第 26 章 耳蜗的听觉处理 Summary

## 1. 耳朵的三个功能部分
**基本结构与功能划分**
- 外耳: 耳廓(反射器) + 外耳道 → 捕获声音能量 → 聚焦到鼓膜
- 中耳: 听小骨链(锤骨-砧骨-镫骨) → 振动传递 + 阻抗匹配
- 内耳: 耳蜗(螺旋结构) → 机电转导 → 神经信号生成

**声音传递路径**
P: 声压波(340m/s) → 鼓膜振动 → 听小骨运动 → 镫骨活塞作用 → 卵圆窗位移 ⟹ 前庭阶液体压力变化

**中耳放大机制**
- 面积比: 鼓膜/镫骨足板 ≈ 20:1
- 杠杆比: 锤骨/砧骨长度 ≈ 1.4:1
- 总压力增益: ~30倍 ∵ 阻抗匹配需求

## 2. 耳蜗的结构与功能
**三个充液隔室(scala)**
- 前庭阶(上): 外淋巴 ← 卵圆窗连接
- 中阶(中): 内淋巴 ← 柯蒂氏器所在
- 鼓阶(下): 外淋巴 ← 圆窗连接
- 顶端连通: 螺旋孔 → 压力平衡

**基底膜特性梯度**
- 形态学梯度: 基部(窄,50μm,硬) → 顶端(宽,500μm,软)
- 频率映射: 基部(20kHz) → 顶端(20Hz), 对数分布
- 行波特性: 频率 → 特征位置 → 最大振幅点

**柯蒂氏器组成**
- 内毛细胞: 1排, ~3500个, 主要感觉功能
- 外毛细胞: 3排, ~12000个, 主动放大功能
- 盖膜: 胶状架, 与外毛细胞束接触
- 支持细胞: 柱细胞 + 戴特氏细胞

## 3. 毛细胞的机电转导
**发状纤维束结构**
- 静纤毛: 肌动蛋白核心, 高度递增排列
- 尖端连接: 钙粘蛋白23 + 原钙粘蛋白15 → 机械门控
- 形态差异: 内毛细胞(线性) vs 外毛细胞(V/W形)

**转导机制**
P: 束偏转(±100nm) → 尖端连接张力变化 → 通道开放概率调节 → K+内流(主要) + Ca2+内流 → 受体电位(±25mV)

**转导通道特性**
- 分子组成: TMC1/2(孔道) + TMIE + LHFPL5 := 通道复合体
- 门控速度: <80μs @室温 ∵ 直接机械门控
- 通道数量: ~2个/静纤毛
- 选择性: 非选择性阳离子通道, 孔径~1.3nm

**适应机制**
- 慢适应(~20ms): 肌球蛋白马达 → 尖端连接插入点移动
- 快适应(<1ms): Ca2+结合 → 通道关闭偏好
- 功能: 维持工作点 + 高通滤波

## 4. 主动放大机制
**外毛细胞电动性(Electromotility)**
- 分子基础: Prestin蛋白 → 电压依赖构象变化
- 响应特性: 去极化→收缩, 超极化→伸长, >80kHz
- 能量来源: 跨膜电场 (非ATP依赖)

**发状纤维束主动运动**
- 自发振荡: 临界振荡器特性
- 频率调谐: 特征频率处最大放大
- 能量输出: 可对外做功

**放大效果**
- 灵敏度提升: 40-50dB @ 低声压级
- 频率选择性: 特征频率处锐化
- 压缩非线性: 10dB SPL敏感性 > 80dB SPL的100倍

## 5. 霍普夫分岔原理
**临界振荡器特性**
- 系统状态: 振荡不稳定性边缘 (C = Cc)
- 数学描述: dZ/dt = -(C-Cc-2iπfc)Z - B|Z|²Z + F

**功能特性**
1. 频率选择性放大: fc处最大增益
2. 压缩非线性: 响应 ∝ 输入^(1/3)
3. 自发振荡: C > Cc时产生
4. 耳声发射: 诱发性 + 自发性

## 6. 神经编码
**神经支配模式**
- 内毛细胞: 90%传入纤维, 1:10(细胞:纤维)
- 外毛细胞: 主要传出支配, 直接突触
- 螺旋神经节: ~30000双极神经元

**带状突触特性**
- 形态: 突触带 + 突触囊泡储备池
- 递质: 谷氨酸(主要)
- Ca2+通道: 静息时部分激活 → 持续释放
- 特殊蛋白: Otoferlin替代synaptotagmin

**频率编码策略**
1. 位置编码: 音调拓扑映射 @ 基底膜→神经
2. 时间编码: 锁相放电 (<3kHz)
3. 强度编码: 放电率 ∝ log(声压)

## 7. 听力损失与治疗
**感音神经性聋**
- 病因: 毛细胞退化(主要) > 神经损伤
- 遗传因素: USH基因群 → 尖端连接缺陷
- 环境因素: 噪声, 耳毒性药物(氨基糖苷类)

**治疗方法**
1. 助听器: 频率特异性放大 @ 残余听力
2. 人工耳蜗:
   - 原理: 电极阵列 → 直接刺激神经纤维
   - 利用: 音调拓扑映射
   - 效果: ~350,000植入者, 个体差异大
3. 未来方向: 干细胞分化, 光遗传学刺激

## 8. 进化与比较
**趋同进化特征**
- 共同特性: 主动放大, 频率选择性, 耳声发射
- 机制多样性:
  - 哺乳动物: 电动性 + 束运动
  - 非哺乳动物: 束运动 + 电共振
- 频率范围差异: 哺乳动物(>100kHz) vs 其他(<14kHz)

## 核心概念总结
耳蜗 := 机械频率分析器 + 主动放大器 + 神经编码器

关键创新:
1. 直接机械门控 → 微秒级响应
2. 双重放大机制 → 百万倍动态范围
3. 霍普夫分岔 → 最优检测灵敏度
4. 音调拓扑映射 → 频率位置编码
