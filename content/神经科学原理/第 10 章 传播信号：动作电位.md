# 第*11*章\_突触传递概述 摘要

## 核心概念

**突触传递类型**

- 电突触：间隙连接 → 直接离子电流流动 → 快速传递（~0ms延迟）
- 化学突触：神经递质释放 → 受体结合 → 离子通道调节（0.3-5ms延迟）
- P: 单个神经元平均形成~1000个突触连接（浦肯野细胞：~100k输入，颗粒细胞：~4输入）

## 电突触

**结构与功能**

- 间隙连接通道 := 成对半通道（连接子），每个6个连接蛋白亚基
- 通道孔径：~1.5nm直径 ⟹ 对离子+小分子(<1kDa)非选择性
- 连接蛋白36 := 哺乳动物最常见的神经元连接蛋白
- 4nm突触间隙 vs 化学突触20-40nm

**信号特性**

- 双向传输（通常）∨ 整流突触（单向）
- 分级传输：阈下信号直接传递
- 欧姆定律：ΔV = I × Rin ⟹ 较小的突触后细胞显示更大电压变化
- ex: 小龙虾巨大运动突触，金鱼毛特讷氏细胞逃避反应

**功能作用**

- 同步化：电耦合细胞 → 更高触发阈值 但同步放电
- 代谢耦合：Ca²⁺、IP₃、cAMP在细胞间传递
- 胶质网络：星形胶质细胞Ca²⁺波（~1-20μm/s传播）
- 疾病相关：连接蛋白32突变 → Charcot-Marie-Tooth病；连接蛋白26 → 先天性耳聋

## 化学突触

**传递过程**

1. 动作电位 → 电压门控Ca²⁺通道开放 @活动区
2. Ca²⁺内流 → 囊泡融合（胞吐作用）→ 神经递质释放
3. 跨突触间隙扩散（20-40nm）
4. 受体结合 → 离子通道调节 → 突触后电位

**放大原理**

- 1个囊泡 → 数千递质分子 → 数千离子通道开放
- 小突触前末梢 ⟹ 大突触后去极化可能

**受体类型**

_离子型受体_

- 结构：4-5亚基复合物形成通道孔
- 直接门控：配体结合 → 构象改变 → 通道开放
- 快速传递：毫秒级响应
- 功能：快速行为（ex: 牵张反射）

_代谢型受体_

- 结构：7跨膜α螺旋域，与通道分离
- 间接门控：G蛋白激活 → 第二信使 → 蛋白激酶 → 通道磷酸化
- 缓慢传递：100ms至分钟持续时间
- 功能：行为调节，学习强化通路

**关键原理**

- P: 递质作用取决于受体特性，而非递质身份
- 相同递质 → 不同效应取决于受体类型
- ex: 乙酰胆碱 → 神经肌肉接头兴奋 但心脏抑制

## 整合与可塑性

**突触共存**

- 电突触+化学突触可共存于同一神经元
- 发育：电突触先形成 → 引导化学突触形成 → 电传输下调
- 成熟回路：两类型相互作用（ex: 视网膜双极细胞）

**调节机制**

- 间隙连接：蛋白激酶磷酸化 ↔ 偶联强度
- pH/Ca²⁺敏感性 → 受损细胞隔离
- 邻近突触的神经递质 → 间隙连接代谢调节

**突触可塑性**

- 同突触：同一突触活动依赖性改变
- 异突触：外部因素调节递质释放
- 时间尺度：秒 → 终生
- 功能意义：记忆存储基质（@Ch53-54）

## 比较特征

| 特性     | 电突触       | 化学突触 |
| -------- | ------------ | -------- |
| 距离     | 4nm          | 20-40nm  |
| 传导物质 | 离子电流     | 神经递质 |
| 延迟     | ~0ms         | 0.3-5ms+ |
| 方向     | 双向（通常） | 单向     |
| 放大     | 无           | 有       |
| 调节     | 有限         | 广泛     |

## 临床相关性

- 连接蛋白突变 → 周围神经病、耳聋
- 突触传递缺陷 → 神经/精神疾病（@Ch57）
- 间隙连接功能障碍 → 脱髓鞘疾病
