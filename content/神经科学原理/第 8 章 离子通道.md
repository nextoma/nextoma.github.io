# 第9章 膜电位和神经元的被动电特性 - 符号化摘要

## 核心概念定义

- **膜电位** := 跨膜电位差 (Vm = Vin - Vout)
- **静息膜电位** := 无信号时膜的稳定电位 (Vr ≈ -60~-70mV)
- **平衡电位** := 单一离子电化学平衡时的膜电位
- **去极化** := 膜电位变得不那么负 (向0mV方向)
- **超极化** := 膜电位变得更负 (远离0mV方向)

## 9.1 跨细胞膜的电荷分离产生静息膜电位

### P1: 电荷分离原理

**静息状态下**:
膜外侧: 正电荷过量
膜内侧: 负电荷过量
电荷分离 → 跨膜电位差

**膜电位定义**:
Vm = Vin - Vout (公式9.1)
其中: Vout = 0 (惯例) → Vm = Vin

### 电信号基础

**电流载体**: 阳离子(+) + 阴离子(-)
**电流方向**: 正电荷净移动方向
∴ 阳离子沿电流方向，阴离子沿反方向

**膜电位变化**:

- **去极化**: 电荷分离↓ → Vm向0接近
- **超极化**: 电荷分离↑ → Vm更负

### P2: 被动vs主动响应原理

**被动响应** (电紧张电位):
小电位变化 → 无门控通道开放 → 线性响应

**主动响应**:
去极化达阈值 → 电压门控通道开放 → 动作电位

**超极化**: 几乎总是被动响应
**小去极化**: 通常被动响应
**大去极化**: 可能触发主动响应

## 9.2 静息膜电位由非门控离子通道和门控离子通道决定

### 神经元离子分布 (鱿鱼巨轴突)

| 离子 | 胞内浓度(mM) | 胞外浓度(mM) | 平衡电位(mV) |
| ---- | ------------ | ------------ | ------------ |
| K⁺   | 400          | 20           | -75          |
| Na⁺  | 50           | 440          | +55          |
| Cl⁻  | 52           | 560          | -60          |
| A⁻   | 385          | 无           | 无           |

### 9.2.1 神经胶质细胞: 仅K⁺通透

**P3: K⁺平衡电位原理**
仅K⁺通透膜 → 静息电位 = K⁺平衡电位

**机制**:

1. K⁺浓度梯度 → K⁺外流
2. K⁺外流 → 膜外正电荷积累，膜内负电荷积累
3. 电位差阻碍进一步K⁺外流
4. 平衡: 化学驱动力 = 电驱动力

**Nernst方程**:
Ex = (RT/zF) × ln([X]o/[X]i) (公式9.2)

**简化形式**(25°C):
Ex = (58mV/z) × log([X]o/[X]i) (公式9.3)

**K⁺平衡电位计算**:
EK = 58 × log(20/400) = -75mV

### 9.2.2 神经细胞: 三种离子通透

**多离子通透性**: K⁺ + Na⁺ + Cl⁻ 同时通透
(A⁻大分子无法透过膜)

### 膜电位决定机制

**P4: 复合膜电位原理**
多离子通透膜电位 = 各离子按电导率加权平均

**两离子情况 (K⁺ + Na⁺)**:

- 纯K⁺膜: Vm = EK = -75mV
- 加入Na⁺通道 → Na⁺内流 → 轻微去极化
- 新平衡: Na⁺内流 = K⁺外流

**稳态条件**:
∑I_ion = 0 (各离子电流代数和为零)

**相对电导重要性**:
gK >> gNa → Vm接近EK
∵ K⁺通道数量 >> Na⁺通道数量
∴ 小的K⁺净驱动力 = 大的Na⁺净驱动力

### Goldman-Hodgkin-Katz方程

**多离子膜电位**:
Vm = (RT/F) × ln[(PK[K]o + PNa[Na]o + PCl[Cl]i)/(PK[K]i + PNa[Na]i + PCl[Cl]o)]

其中: P = 通透性系数

### 9.2.3 离子泵维持浓度梯度

### P5: 稳态维持原理

静息状态 ≠ 平衡状态
静息状态 = 稳态 (持续能量消耗维持)

**Na⁺-K⁺泵** (Na⁺-K⁺ ATPase):

- 功能: 3Na⁺ out + 2K⁺ in / 1ATP
- 结果: 电荷不平衡 → 生电泵
- 作用: 维持Na⁺/K⁺梯度

**泵与通道区别**:
| 特征 | 离子通道 | 离子泵 |
|------|----------|---------|
| 转运机制 | 被动扩散 | 主动转运 |
| 能量需求 | 无需ATP | 需要ATP |
| 转运速率 | 10⁷-10⁸ 离子/秒 | 10²-10³ 离子/秒 |
| 方向 | 顺梯度 | 逆梯度 |

### Ca²⁺稳态维持

**胞质Ca²⁺浓度**: 50-100nM (比胞外低10⁴倍)

**Ca²⁺清除机制**:

1. **Ca²⁺-ATPase**: 2Ca²⁺ out + 2H⁺ in / ATP
2. **Na⁺-Ca²⁺交换器**: 3-4Na⁺ in + 1Ca²⁺ out

**次级主动转运**:
Na⁺梯度能量 → Ca²⁺逆梯度转运
最终能量来源: Na⁺-K⁺泵的ATP水解

### P型ATPase家族

**结构特征**:

- α亚基: 110kDa, 10个跨膜螺旋
- β亚基: 辅助组装和膜表达(Na⁺-K⁺泵)

**人类Na⁺-K⁺泵亚型**:

- ATP1A1: 广泛分布
- ATP1A2: 突变→家族性偏瘫性偏头痛
- ATP1A3: 神经元特异, 突变→运动障碍
- ATP1A4: 特殊分布

## 神经元被动电特性

### P6: 电路模型原理

神经元 ≈ RC电路 (电阻-电容组合)

**膜电阻 (Rm)**:

- 决定因素: 开放离子通道密度
- 影响: 电流-电压关系

**膜电容 (Cm)**:

- 成因: 脂质双分子层介电特性
- 作用: 电荷存储, 时间延迟

### 时间常数和空间常数

**膜时间常数**: τ = Rm × Cm

- 意义: 膜电位变化的时间尺度
- 典型值: 1-20ms

**空间长度常数**: λ = √(Rm/Ra)

- 意义: 电信号空间衰减的特征长度
- Ra: 轴质电阻

### 电缆理论

**P7: 电缆传播原理**
神经元突起 ≈ 有损耗传输线

**电位衰减**: V(x) = V₀ × e^(-x/λ)
其中: x=距离, λ=长度常数

**时间响应**: V(t) = V∞ × (1 - e^(-t/τ))
其中: τ=时间常数

## 生理意义

### P8: 信号整合原理

树突被动特性 → 突触信号整合能力

**时间求和**: 连续输入 → 电容积分效应
**空间求和**: 多点输入 → 电阻网络效应

### 能量代谢

**ATP消耗分配**:

- Na⁺-K⁺泵: ~50-70%
- Ca²⁺泵和其他: ~20-30%
- 蛋白合成等: ~10-20%

## 病理意义

### 泵功能异常疾病

- **家族性偏瘫性偏头痛**: ATP1A2突变
- **快速发作肌张力障碍**: ATP1A3突变
- **儿童交替性偏瘫**: ATP1A3突变
- **强心苷中毒**: Na⁺-K⁺泵抑制

### 膜电位异常

- **去极化阻滞**: K⁺外流受阻
- **动作电位衰竭**: Na⁺/K⁺梯度丧失
- **Ca²⁺超载**: Ca²⁺清除障碍

## 关键原理总结

### P9: 膜电位多因素决定原理

静息膜电位 := f(离子浓度梯度, 膜通透性, 泵活性)
Vm = 各离子平衡电位的加权平均

### P10: 稳态能量依赖原理

静息膜电位维持 ⟹ 持续ATP消耗
∵ 离子泄漏 + 泵逆转运 = 稳态平衡

### P11: 被动特性决定信号传播

Rm, Cm, Ra → τ, λ → 信号时空特性
