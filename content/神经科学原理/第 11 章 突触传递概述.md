# 第*12*章\_直接门控传输：神经肌肉突触 摘要

## 神经肌肉接头结构

**解剖组织**

- 运动轴突 → 分支 → 突触扣结 → 终板接触
- 突触间隙：100nm宽，含基底层 + 乙酰胆碱酯酶
- 初级褶皱 → 次级连接褶皱（ACh受体集中于顶部：>10,000/μm²）
- 活动区：囊泡聚集位点，含电压门控Ca²⁺通道

**功能特化**

- 终板大小：20-100μm（因物种而异）
- 安全系数 := 3-4×阈值（75mV EPSP vs ~20mV阈值）
- P: 高安全系数对运动、呼吸、逃避反应至关重要

## 信号传递过程

**ACh释放与结合**

1. 动作电位 → Ca²⁺内流 @活动区
2. 囊泡融合（胞吐作用）→ ACh释放
3. ACh跨间隙扩散（~微秒）
4. 受体结合 → 通道开放 → 阳离子内流

**终板电位（EPP）**

- 局部去极化：振幅高达75mV
- 时间进程：上升1-2ms，衰减较慢（被动电缆特性）
- 振幅 ∝ 1/距离（被动扩散）
- P: EPP振幅通常超过阈值3-4×

## ACh受体通道特性

**结构组织**

- 五聚体：2α + β + γ + δ亚基（肌肉型）
- 神经元变体：2α + 3β ∨ 5α（α7型）
- 每个亚基：4个跨膜螺旋（M1-M4）
- ACh结合位点：2个位点 @α-γ和α-δ界面
- 孔径：~0.6nm（电生理）vs 0.3-0.4nm（晶体学）

**离子选择性**

- 非选择性阳离子通道：Na⁺、K⁺、Ca²⁺通透
- gNa/gK ≈ 1.8（轻微Na⁺偏好）
- 反转电位：0mV（ENa + EK的加权平均）
- 阴离子排斥：3个负电荷环

**单通道行为**

- 基本电流：-2.7pA @-90mV
- 电导：γ = 30pS（欧姆性，电压无关）
- 开放时间：~1ms平均（随机）
- 离子流量：~17,000离子/次开放

## 门控机制

**状态转换**

```
关闭(R) + 2ACh ↔ 结合(A₂R) ↔ 开放(A₂R*) ↔ 脱敏(A₂D*)
```

**结构改变**

- 配体结合 → 细胞外域收缩（"花瓣闭合"）
- 扭转运动 → M2螺旋旋转/倾斜
- 亮氨酸门（M2位置9）在孔中部开放
- 脱敏：孔在细胞内侧收缩

**关键结构元件**

- Cys环：15aa特征序列（所有pLGIC家族）
- 选择性滤器：Thr/Ser + Glu环近细胞质侧
- 疏水门：Leu残基 @M2中心

## 量子释放

**微终板电位（mEPPs）**

- 自发事件：0.5-1.0mV，~1/秒
- 单量子 := ~5000-10000 ACh分子
- P: EPP = ~200,000量子释放的总和

**量子假说**

- Del Castillo & Katz：ACh以离散包释放
- 单囊泡 → 单量子 → 单mEPP
- 诱发EPP := 量子的整数倍

## 药理学与临床相关性

**拮抗剂与阻断剂**

- 箭毒/d-筒箭毒碱：竞争性拮抗剂 @ACh位点
- α-银环蛇毒素：不可逆结合 @ACh位点
- 琥珀胆碱：通过脱敏去极化阻断

**疾病状态**

- 重症肌无力：自身抗体 → ↓ACh受体
- 先天性肌无力综合征：受体突变
- 慢通道综合征：M2突变 → 延长开放
- ex: 施旺细胞连接蛋白突变 → 脱髓鞘

## 等效电路模型

**EPP组成**

- 突触电导：gEPSP ≈ 5×10⁻⁶S（激活时）
- 驱动力：(Vm - EEPSP)，其中EEPSP = 0mV
- 电流方程：IEPSP = gEPSP × (Vm - EEPSP)
- 总EPP = 200,000通道 × 2.7pA/通道 ≈ 500nA
