# 第*62*章\_影响社会认知的障碍：孤独症谱系障碍 Summary

## 🧠 孤独症谱系障碍(ASD)核心

```
ASD三大核心症状：
1. 社会交流和互动缺陷
2. 重复刻板行为和兴趣
3. 感觉处理异常
```

## 👥 社会认知缺陷

### 心理理论(ToM)缺陷

```
他人心理状态理解：
一级ToM: 理解他人信念(4-5岁正常发展)
二级ToM: 理解他人对第三者的信念
ASD患者: ToM发展延迟或缺陷
```

**神经基础**：

- 内侧前额叶皮层: 心理状态表征
- 颞顶联合区: 注意和心理理论整合
- 上颞沟: 社会知觉加工

### 情绪识别困难

```
面部表情识别：
基本情绪识别相对保留
复杂情绪识别困难
眼部区域注意不足
面孔加工异常
```

### 社会动机缺陷

```
社会奖励敏感性↓：
社会刺激内在奖励价值降低
非社会刺激偏好增强
社会回避行为
```

## 🔄 刻板重复行为

### 行为类型分类

```
重复行为谱系：
低级运动刻板: 摇摆，转圈，手部动作
高级认知刻板: 仪式化行为，强迫性排列
感觉寻求: 特定感觉刺激偏好
局限兴趣: 狭窄专注特定主题
```

**神经机制**：

- 皮层-基底核-丘脑回路异常
- 纹状体功能异常: 习惯性行为过强
- 前额叶认知灵活性受损

## 🎭 感觉处理异常

### 感觉调节障碍

```
感觉反应模式：
感觉过敏: 对刺激过度反应
感觉迟钝: 对刺激反应不足
感觉寻求: 主动寻求感觉刺激
感觉回避: 避免特定感觉输入
```

### 感觉整合困难

```
多感觉整合缺陷：
视听整合异常
本体感受异常
触觉防御
前庭觉异常
```

## 🧬 神经生物学机制

### 大脑解剖异常

```
结构异常：
大脑体积: 早期过度生长，后期归一化
小脑: 浦肯野细胞减少
杏仁核: 体积异常，功能连接异常
胼胝体: 连接度减少
```

### 神经连接异常

```
连接理论：
局部过连接: 短距离连接增强
长距离欠连接: 脑区间协调困难
白质完整性异常: 胼胝体，弓状束
默认模式网络异常
```

### 兴奋-抑制失衡

```
E/I失衡理论：
GABA抑制功能不足
谷氨酸兴奋过度
γ振荡异常: 神经同步化缺陷
```

## 🧬 遗传学基础

### 遗传异质性

```
遗传复杂性：
遗传度: 双生子研究~90%
多基因贡献: 数百个风险基因
罕见大效应变异: CNVs, 单基因突变
常见小效应变异: GWAS发现
```

### 关键易感基因

```
ASD风险基因：
CHD8: 染色质重塑，转录调节
SHANK家族: 突触支架蛋白
NLGN/NRXN: 突触粘附分子
MECP2: 表观遗传调节(Rett综合征)
FMR1: 脆性X综合征
```

### 从头突变

```
新发突变：
父源年龄效应: 新发突变率增加
功能缺失突变: 蛋白功能破坏
剂量敏感性: 基因表达量异常
```

## 👶 发育轨迹

### 早期发育标志

```
早期预警信号：
12个月: 缺乏社会性微笑
18个月: 缺乏共同注意
24个月: 言语发育迟缓
眼神接触回避
对名字呼唤无反应
```

### 发育回归

```
技能丧失：
15-30%患儿经历回归
语言技能丢失
社交技能退化
可能机制: 免疫异常，突触修剪过度
```

## 🗣️ 语言和交流障碍

### 语言发育异常

```
交流困难：
语言发育迟缓: 50%患者
回音样语言: 重复他人话语
代词使用混乱: "你我"颠倒
语调异常: 单调或夸张
```

### 非言语交流缺陷

```
社会交流技能：
手势使用减少
面部表情缺乏
眼神接触回避
共同注意缺陷
```

## 🧠 神经发育假说

### 突触发育异常

```
突触理论：
突触形成过多: 早期过度连接
突触修剪不足: 幼稚连接保留
突触成熟延迟: 功能发育滞后
```

### 小脑发育异常

```
小脑理论：
早期小脑损伤影响社会认知发育
小脑-大脑皮层连接异常
预测编码缺陷: 感觉预期异常
```


## 🌈 预后和适应

### 功能预后

```
影响因素：
早期语言能力: 预后最重要指标
认知水平: IQ>70预后较好
早期干预: 2-3岁前开始最佳
家庭支持: 社会环境重要性
```

### 终生支持需求

```
支持程度分级：
需要支持: 轻度，相对独立
需要大量支持: 中度，部分独立
需要极大量支持: 重度，需要全面照顾
```

## ⚡ 关键要点

1. **神经发育障碍**：ASD是大脑发育异常导致的终生性障碍
2. **遗传复杂性**：多基因贡献，遗传异质性极高
3. **社会脑异常**：社会认知网络发育和功能异常
4. **感觉整合障碍**：多感觉处理和整合困难
5. **早期识别重要**：早期诊断和干预改善预后
6. **个体差异显著**：症状严重程度和表现高度变异
7. **终生管理**：需要个体化的终生支持和服务

## 🎯 核心公式

```
ASD表型 = 遗传风险 × 发育时机 × 环境因素
干预效果 = 开始年龄⁻¹ × 强度 × 持续时间 × 个体适应性
功能预后 = 早期语言能力 × 认知水平 × 干预质量 × 家庭支持
```
