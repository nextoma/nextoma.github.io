# 第5章 行为调节神经回路的计算基础 - 符号化摘要

## 核心概念定义

- **神经编码** := 刺激特征→神经活动模式的转换过程
- **神经解码** := 神经活动→信息提取的逆过程
- **连接组学** := 神经元互连模式的研究
- **突触可塑性** := 突触强度随经验变化的能力

## 5.1 神经放电模式提供信息编码

### P1: 编码-解码原理

**编码过程**: 感觉刺激 → 电信号 → 神经活动模式
**解码过程**: 神经活动模式 → 信息提取 → 行为响应

∀ 感觉信息(S): S ⟹ 电信号模式(E) ⟹ 动作电位序列(AP)
∵ 神经系统需要处理外部信息 ∴ 必须转换为内部表征

### ex: 海马体空间认知地图

**位置细胞**: 仅在动物经过特定空间位置时激活的细胞
位置细胞(i) := {激活 | 动物位置 ∈ 位置场(i)}

**群体编码优势**:

- 单细胞: 局部编码 + 噪声干扰
- 细胞群: 完整空间覆盖 + 冗余可靠性
  精度 ∝ 1/√(细胞数量) @ 群体解码

**尖波涟漪现象**:

- 探索期间: 位置编码模式
- 静止期间: 尖波涟漪(50-500ms)
  重放速度 = 探索速度 × (10-20) @ 时间压缩
  方向: 正向重放 ∨ 反向重放

**功能**: 记忆巩固的心理排练
重放事件 ⟹ 记忆强化 ⟹ 长期存储

## 5.2 神经回路基序为信息处理提供了基本逻辑

### 连接模式分类

1. **前馈连接**: 区域A → 区域B (单向)
2. **反馈连接**: 区域B → 区域A (双向)
3. **循环连接**: 神经元群 ↔ 自身

### P2: 兴奋-抑制平衡原理

**平衡条件**: 兴奋输入 ≈ 抑制输入
∵ 兴奋输入过强 ⟹ 不稳定活动 ⟹ 癫痫样发作
∴ 需要抑制调节维持稳定性

### 5.2.1 视觉处理层级结构

**层级组织**: 视网膜 → 外侧膝状体 → V1 → V2 → V4 → 下颞皮层

**特征选择性递增**:

- 低层: 简单特征(边缘、方向)
- 高层: 复杂对象(面孔、物体)

**感受野特性变化**:
感受野大小: 视网膜 < V1 < V2 < V4 < 下颞皮层
位置不变性: 低层 < 高层
选择性: 低层 < 高层

**与机器学习对比**:
生物视觉网络 ↔ 人工神经网络
层级响应模式 ≈ 深度学习网络激活

### 5.2.2 小脑的发散-汇聚结构

**发散连接**: 2×10^8 苔藓纤维 → 5×10^10 颗粒细胞
发散比率 = 250:1

**混合表征**: 颗粒细胞(i) := ∧(输入1, 输入2, ..., 输入4)
ex: 颗粒细胞 激活条件 := 视觉刺激(网球) ∧ 运动(手腕弯曲)

**组合数计算**:

- 2个输入组合: C(100,2) = 4,950
- 3个输入组合: C(100,3) > 150,000
- n个组合: 指数级增长

**汇聚连接**: 10^5 颗粒细胞 → 1 浦肯野细胞
**功能**: 从丰富表征中提取有用信息

### 5.2.3 循环回路与持续活动

**问题**: 神经元膜时间常数 ≈ 20ms → 天然健忘性
**需求**: 认知操作需要秒-分钟级持续性

**整合回路**:

- 瞬态输入 → 持续活动 → 记忆维持
- 应用: 眼位维持、决策制定、工作记忆

### ex: 动眼神经整合器

**观察**:

- 动眼神经元: 瞬态放电(眼动) + 持续放电(眼位)
- 运动前神经元: 仅瞬态放电

**解决方案**: 循环兴奋网络
速度信号 → 整合回路 → 位置信号

**网络动力学**:

- 弱耦合: 快速衰减
- 适度耦合: 缓慢衰减
- 临界耦合: 无限维持

**调节条件**: 循环兴奋强度 = 自然衰减强度
时间常数扩展: 20ms → 20s (×1000)

## 5.3 学习和记忆取决于突触可塑性

### 学习类型分类

1. **监督学习**: 明确指导信号
2. **强化学习**: 奖励/惩罚信号
3. **无监督学习**: 无外部指导

### 5.3.1 赫布可塑性

**赫布法则** (1949):
"同时激活的细胞连接加强"
突触强度变化 ∝ 突触前活动 × 突触后活动

**数学表达**:
dw_ij/dt = η × x_i × y_j
其中: w_ij=突触权重, x_i=输入, y_j=输出, η=学习率

**稳态调节**: 防止突触无限增强
赫布可塑性 + 稳态可塑性 → 信号提取

**功能**: 无监督提取最显著输入组合
ex: 从噪声中提取正弦信号模式

### 5.3.2 小脑运动学习

### ex: 延迟性眨眼条件反射

**实验范式**:
条件刺激(CS, 光/声) + 非条件刺激(US, 吹气) → 条件反应(眨眼)

**回路映射**:

- CS信息: 苔藓纤维 → 颗粒细胞 → 浦肯野细胞
- US信息: 攀缘纤维 → 浦肯野细胞

**可塑性机制**:
颗粒细胞活动 + 攀缘纤维输入 → 长时程抑制(LTD)
突触强度: w_ij → w_ij - Δw

**学习过程**:

1. US前激活的颗粒-浦肯野突触减弱
2. 浦肯野细胞出现学习性激活暂停
3. 抑制解除 → 下游激活 → 眨眼响应

**时间精确性**:
暂停时机 = US预期时间 - ε
由颗粒细胞时间模式多样性实现

## 关键原理总结

### P3: 群体编码原理

单细胞局限性 → 群体冗余性 → 可靠信息传递
信息容量 ∝ log(神经元数量)

### P4: 层级处理原理

简单特征 → 复杂特征 → 抽象概念
感受野扩大 ∧ 选择性增强 ∧ 不变性增加

### P5: 循环维持原理

瞬态输入 + 循环激发 ⟹ 持续活动 ⟹ 短期记忆
时间常数 = 1/(1 - 循环强度)

### P6: 可塑性学习原理

- 赫布学习: 相关性检测 → 无监督特征提取
- 强化学习: 误差信号 → 行为优化
