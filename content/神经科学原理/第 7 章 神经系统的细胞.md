# 第8章 离子通道 - 符号化摘要

## 核心概念定义

- **离子通道** := 跨膜整合蛋白，形成离子选择性通道
- **离子选择性** := 通道对特定离子的优先渗透能力
- **通道门控** := 通道开放-关闭状态的转换过程
- **电导** := 电阻倒数，反映离子通透能力 (γ = 1/R)

## 8.1 离子通道是跨越细胞膜的蛋白质

### P1: 膜不透性-通道必要性原理

**膜脂双分子层**:
疏水性 → 离子排斥 → 几乎完全不透离子
∵ 离子亲水性强 + 水合鞘大 ∴ 进入脂质相能量不利

**离子-水相互作用**:
离子电荷 → 静电吸引水分子 → 水合鞘形成

- Na+: 小离子 → 强电场 → 大水合鞘 → 表观直径大
- K+: 大离子 → 弱电场 → 小水合鞘 → 表观直径小

**通道解决方案**:
跨膜蛋白通道 → 水性孔道 → 离子渗透路径

### 离子选择性机制

**P2: 双重选择机制**
选择性 := 分子筛效应 + 化学特异性

**分子筛模型**:
K+通道: 孔径适合K+水合离子，排除Na+水合离子
∵ Na+\_hydrated > K+\_hydrated ∴ K+选择性

**化学选择性(Eisenman理论)**:

- 高场强结合位点: 偏好小离子(Na+)
- 低场强结合位点: 偏好大离子(K+)

**静电相互作用**: F ∝ 1/r² (库伦定律)
∵ Na+晶体半径 < K+晶体半径
∴ Na+与负电荷结合位点相互作用更强

**选择性过滤器**:
离子脱水 + 与通道壁极性基团结合 → 选择性渗透
结合能量补偿 ≥ 脱水能量损失 → 离子通过

## 8.2 所有细胞中的离子通道都有几个功能特征

### 8.2.1 单通道电流记录

**膜片钳技术** (Neher & Sakmann, 1976):
玻璃微电极 + 膜片 → 单通道电流测量

**全或无开放**:
通道状态 := {完全关闭 | 完全开放}
电流脉冲: 恒定幅度 → 非渐进开放

**欧姆定律应用**:
i = γ × V = V/R
其中: i=电流, γ=电导, V=电压, R=电阻

**例**: ACh受体通道
i = 2pA @ -80mV → γ = 25pS

### 8.2.2 离子通透特性

**P3: 饱和动力学原理**
通道渗透 ≠ 自由扩散

**浓度-电流关系**:

- 低浓度: i ∝ [离子] (线性)
- 高浓度: i → i_max (饱和)

**米氏动力学方程**:
i = i_max × [S] / (K_d + [S])
其中: K_d = 解离常数 (~100mM)

**饱和机制**:
离子与通道内结合位点结合 → 载量有限 → 饱和现象

**弱结合特性**:
K_d高 (~100mM) vs 酶-底物 K_d (~1μM)
∴ 快速解离 → 高传导率 → 快速信号

### 电流-电压关系

**欧姆性通道**: i-V线性关系
**整流通道**: i-V非线性关系

- 外向整流: 外向电流 > 内向电流
- 内向整流: 内向电流 > 外向电流

### 通道阻断

**阻断剂结合**:
膜内外离子/分子 → 孔口或孔内结合 → 通道阻断

**电压依赖性阻断**:
带电阻断剂 + 膜电场 → 阻断程度依赖电压

**生理性阻断剂**: Mg²⁺, Ca²⁺, Na⁺, 多胺(精胺)

## 通道门控机制

### P4: 构象变化门控原理

通道门控 := 蛋白构象状态转换
稳定状态: {关闭态1, 关闭态2, ..., 开放态}

### 门控的物理模型

1. **局部构象变化**: 通道一个区域改变
2. **整体结构变化**: 全长度协同运动
3. **颗粒阻断**: 阻断颗粒摆入摆出孔口

### 门控转导机制

**A. 配体门控**:
化学配体结合 → 构象变化 → 通道开放

- 胞外配体: 神经递质
- 胞内配体: Ca²⁺, cAMP, GTP结合蛋白

**B. 磷酸化门控**:
蛋白激酶 → 磷酸化 → 构象调节
高能磷酸键 → 门控能量

**C. 电压门控**:
膜电位变化 → 带电区域受力 → 构象变化
电压传感器 + 门控机制

**D. 机械门控**:
膜拉伸/压力 → 物理力传导 → 通道开放
力传导: 膜脂 | 细胞骨架连接

### 通道失活

**P5: 失活-恢复循环**
关闭态 → 开放态 → 失活态 → 恢复 → 关闭态

**失活类型**:

1. **N型失活**: 快速失活，分子机制依赖
2. **C型失活**: 慢失活，Ca²⁺依赖

**Ca²⁺通道失活**:
Ca²⁺内流 → [Ca²⁺]\_i ↑ → 钙调蛋白结合 → 通道失活

### 药理调节

**竞争性拮抗剂**:
拮抗剂竞争激动剂结合位点 → 阻断激活
例: 筒箭毒 vs ACh, α-银环蛇毒 (不可逆)

**非竞争性调节**:
变构结合位点调节 → 门控概率改变
例: 地西泮(安定) + GABA_A受体 → 开放频率↑

## 离子通道结构

### P6: 结构-功能关系原理

通道结构 ↔ 功能特性
跨膜排列 + 孔道形成 + 门控域 + 调节域

### 结构研究方法

- **生物物理学**: 电生理特性
- **生物化学**: 蛋白纯化+功能重建
- **分子生物学**: 基因克隆+表达+突变

## 通道 vs 转运蛋白

### P7: 转运机制区别原理

**离子通道**:

- 连续水性通道
- 被动转运 (顺梯度)
- 高流速 (~10⁸ 离子/秒)
- 不需ATP

**离子泵/转运蛋白**:

- 构象循环转运
- 主动转运 (逆梯度)
- 低流速 (~10² 离子/秒)
- 消耗ATP

**速度差异**: 通道 vs 泵 = 10⁶倍差异

## 通道分类

### 按门控机制

1. **静息通道**: 维持静息电位
2. **电压门控**: 动作电位产生
3. **配体门控**: 突触传递
4. **机械门控**: 感觉转导

### 按离子选择性

- **钠通道**: Na⁺选择性 (PNa:PK ≈ 10-20:1)
- **钾通道**: K⁺选择性 (PK:PNa ≈ 100:1)
- **钙通道**: Ca²⁺选择性
- **氯通道**: Cl⁻选择性
- **非选择性**: 多种阳离子通道

## 生理意义

### P8: 电信号基础原理

离子通道 → 膜电位变化 → 电信号传递

**信号传递链**:
刺激 → 通道开放 → 离子流 → 膜电位变化 → 生理响应

### 感觉转导

- **视觉**: 单光子 → 通道响应
- **嗅觉**: 单分子 → 通道激活
- **听觉**: 原子级振动 → 毛细胞通道

### 病理意义

通道功能异常 → 神经系统疾病

- 囊性纤维化: Cl⁻通道缺陷
- 肌肉疾病: 肌肉通道异常
- 心律失常: 心肌通道异常
- 癫痫: 兴奋性通道异常

## 关键原理总结

### P9: 选择性-渗透性权衡

选择性 ↔ 渗透速率
高选择性 ⟹ 精确调节 ∧ 相对低渗透
低选择性 ⟹ 高渗透 ∧ 调节精度↓

### P10: 门控多样性原理

∃ 多种门控机制 → 复杂调节网络
配体 + 电压 + 机械 + 磷酸化 → 精细调控

### P11: 速度-能量效率原理

被动转运 → 高速率 + 低能耗 → 快速信号
∵ 不消耗ATP ∴ 能量效率高
